insert_tags

Description:

protected virtual void insert_tags (string attributes)

Inserts both opening and closing tags into the buffer at the same time. This ensures the buffer never has an invalid state with only one tag. Sets up marks for tag positions.

Parameters:

attributes

The attributes string for the tag
