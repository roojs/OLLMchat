add_widget_frame

Description:

[ CCode ( return_value_type = "GtkTextChildAnchor*" , transfer = "full" ) ]
public TextChildAnchor add_widget_frame (Frame frame)

Adds a widget frame to the end of the chat view.

The widget will be automatically resized when the chat view is resized. The widget must be a Gtk.Frame.

Parameters:

frame

The frame widget to add

Returns:

The TextChildAnchor that can be used to remove the widget later

1.0
