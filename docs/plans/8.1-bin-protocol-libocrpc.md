# 8.1 — Bin wire protocol (`libocrpc`)

> **Do not update `docs/plans/1.0-summary.md` for this plan.**

**Status:** proposed

**Spec version:** 2.9 (wide integers: width byte `1`/`8` + short path when value fits)

- **🔷** Proposed Vala uses **early return** and **inverted guards** — **🚫** no **`else`** where the happy path can return first ([`reducing-nesting`](../coding-standards.md#reducing-nesting)).

**Related:**

- [`2.10.4.27-ACTIVE-rpc-client-cleanup-and-wire-registration.md`](2.10.4.27-ACTIVE-rpc-client-cleanup-and-wire-registration.md) — wire type registration pattern
- [`8.0-libollmchat-local-gguf-backend.md`](8.0-libollmchat-local-gguf-backend.md) — orthogonal **8.x** workstream

## Purpose

- **🔷** Binary payload codec in **`libocrpc`**, namespace **`OLLMrpc.Bin`**.
- **🔷** Types: **`Stream`**, **`Serializable`** only — **🚫** no separate **`Registry`** class.
- **🔷** **`Stream`** merges codec + per-connection state — alias catalog, JIT token maps, **`register`**, **`write`**, **`parse`** all live on the **`bin`** instance.
- **🔷** **`Connection`** / **`Client`** own one **`Bin.Stream`** (`bin`) for the channel lifetime — **🚫** do not create a new stream per message; **🚫** no separate registry object beside it.
- **🔷** Primary **`Stream`** API: **`write`** (outbound GObject) and **`parse`** (inbound GObject) — RPC moves **objects**, not caller-supplied **`GLib.Type`**.
- **🔷** Short / long encodings for strings, blobs, lists, **and integers** — inline in **`bin_write_prop`** / **`bin_read_prop`** overrides; **🚫** no extra **`Stream`** helper methods.
- **🔷** **`Serializable`** methods are **`bin_`-prefixed** — four override points only: **`bin_write`**, **`bin_read`**, **`bin_write_prop`**, **`bin_read_prop`**.
- **🔷** `⏳` Phase 1: **`Stream`**, **`Serializable`**, attach **`bin`** to **`Connection`** / **`Client`** (codec only — JSON-RPC envelope unchanged).
- **🔷** `⏳` Phase 2: wire **`Request.param`** / **`Response.result`** through **`write`** / **`parse`** instead of **`Json.gobject_*`**.
- **🔷** `⏳` Phase 3 (deferred): **`Bin.Debugger`** — human dump only.
- **🚫** Domain types (**`FileNode`**, **`DirectoryNode`**, …) in **`libocrpc`** — any sample types live only in test sources (see **§5**).
- **🚫** **`Stream.to_output`** / **`Stream.from_input`** — rejected; streams come from the RPC channel.
- **🚫** **`write_short_list_count`** and similar helpers — rejected (naming + no new helpers per design).

## RPC layer today (JSON)

**ℹ️** Server **`OLLMrpc.Transport.Connection`:**

- Owns **`GLib.SocketConnection`** + **`GLib.IOChannel`** read loop.
- **`write(GLib.Object)`** — **`Json.gobject_serialize`** + NDJSON line.
- **`on_input_ready`** — parse line → **`Request.dispatch(params_node)`** → **`Json.gobject_deserialize`** into **`CallParam`**.

**ℹ️** Client **`OLLMrpc.Client`:**

- Owns **`DataInputStream`** / **`DataOutputStream`** on the socket.
- **`call(Request)`** — serialize request as JSON line; **`dispatch_line`** deserializes **`Response.result`** via process-wide **`OLLMrpc.types`**.

**ℹ️** Registration today ([`2.10.4.27`](2.10.4.27-ACTIVE-rpc-client-cleanup-and-wire-registration.md)):

- Process-wide **`OLLMrpc.register(name, gtype)`** before **`Client`** exists.
- Per-handler **`Request.register`** on server for params types.

## Bin integration model

### Ownership

```text
Connection (server) / Client (client)
  └── Bin.Stream bin          ← in/out streams + register() + JIT maps + write/parse
```

- **🔷** **`Connection`** / **`Client`** create **`bin`** once when the channel opens; hold it until disconnect.
- **🔷** **`Stream`** is built from the channel's **`in_stream`** + **`out_stream`** only.
- **🔷** **`bin.register(alias, gtype)`** — instance method; types are registered **on that stream**, not static / process-wide.
- **🔷** **Send (explicit):** caller invokes **`bin.write(obj)`** when outbound RPC carries a GObject body.
- **🔷** **Receive (automatic):** existing RPC read loop — when inbound data arrives, RPC calls **`bin.parse()`** and gets the GObject; caller does not invoke **`parse`** manually on the hot path.

### Send vs receive

**Send**

- **`Client.call`** (Phase 2) — before flushing the request line/frame, **`client.bin.write(request.param)`** (or equivalent) onto **`out_stream`**.
- Server **`Connection.reply`** (Phase 2) — **`connection.bin.write(response.result)`** for typed result bodies.
- Today JSON path stays; bin **`write`** is additive until cutover.

**Receive (hooked into read loop)**

- **Server** — **`Connection.on_input_ready`**: when a complete bin payload is available on the channel, RPC runs **`var obj = this.bin.parse()`** and passes **`obj`** into **`Request.dispatch`** (replacing **`Json.gobject_deserialize`** on params).
- **Client** — **`Client.read_loop`** / **`dispatch_line`**: when a response body is bin-framed, RPC runs **`var obj = this.bin.parse()`** and assigns **`response.result`** (replacing **`Json.gobject_deserialize`**).
- **`parse()`** returns **`Serializable`**; root **`GLib.Type`** is read from the wire inside **`parse`** — **no** type argument from caller.

### Type registration (on `bin`)

- **🔷** **`register(alias, gtype)`** is an **instance** method on **`Stream`** — populates that connection's alias catalog.
- **🔷** Call **`bin.register(...)`** after **`bin`** exists (post-**`connect()`** / post-**`start()`**), before **`write`** / **`parse`** on that channel.
- **🔷** Client: e.g. **`V2/ProjectManager`** after **`yield rpc.connect(hello)`** — register wire types on **`rpc.bin`** (parallel to JSON **`OLLMrpc.register`** today).
- **🔷** Server: register on each **`Connection.bin`** when the client channel is accepted (same alias set both ends must agree).
- **🚫** **`static Stream.register`** — rejected.
- **ℹ️** JSON **`OLLMrpc.register`** stays for Phase 1 NDJSON path; bin **`register`** is separate until Phase 2 cutover.

### Phase 1 vs Phase 2

- **Phase 1** — add **`Stream`**, **`Serializable`**; **`Connection`** / **`Client`** construct **`bin`** on open; **JSON-RPC envelope stays**
- **Phase 2** — **`Request.param`**, **`Response.result`**, server **`dispatch`** use **`write`** / **`parse`** for object bodies
- **Phase 3** — optional **`Debugger`**

**🚫** Phase 1 does **not** remove JSON-RPC framing.

## Core wire axioms

- **🔷** Big-endian on fixed multi-byte fields (token ids, uint16/uint32 length prefixes).
- **🔷** **Short string** (`string`): uint16 length (≤ 64 KiB).
- **🔷** **Large blob** (`uint8[]`): uint32 length.
- **🔷** **Short list**: 1-byte element count (≤ 255) — written inline in list **`bin_write_prop`** overrides.
- **🔷** **Long list**: uint32 element count — inline in overrides when needed.
- **🔷** **Narrow integer** (`char`, `int8`, `uchar`, `uint8`): **1 byte** payload always (no width prefix).
- **🔷** **Wide integer** (`int`…`int64`, `enum`, `uint`…`uint64`, `flags`): **width byte** `1` or `8`, then payload — **short** (1 B) when value fits **`int8`** / **`uint8`** range, else **long** (8 B BE).
- **🔷** **bool**: 1 byte (`0x00` / `0x01`).
- **🔷** Wire tokens: **`TOKEN_REG_KEY`**, **`TOKEN_REG_TYPE`**, **`TOKEN_END`** (unchanged).

---

## Data type layout

### Short string (`string`)

- uint16 BE length + UTF-8 bytes; max **65535**; overflow → **`GLib.error`**.

### Large blob (`uint8[]`)

- uint32 BE length + raw bytes; binary-safe file bodies.

### Primitives (numeric + bool)

- **`bool`:** 1 byte (`0x00` / `0x01`).
- **Narrow integer (1 B):** `char`, `int8`, `uchar`, `uint8` — one raw byte after tag (no width prefix).
- **Wide integer:** after tag → **width byte** `1` or `8`, then:
  - **`1`:** one payload byte (`int8` range for signed, `0…255` for unsigned).
  - **`8`:** eight payload bytes BE (`int64` for signed / `enum`; `uint64` for unsigned / `flags`).
- **🔷** On **write**, wide props **check magnitude** — e.g. `int64` = `5` emits `1` + one byte, not eight.
- **💩** **`float`** / **`double`** / **`date`** — deferred.
- **🚫** Varint / LEB128 / zigzag.

### Narrow integer (1 B)

- GObject types **`char`**, **`int8`**, **`uchar`**, **`uint8`** — always one byte after the property tag.

### Wide integer (width + payload)

- Width byte **`1`** or **`8`** only (invalid width → **`GLib.error`**).
- **Signed** (`int`, `int32`, `long`, `int64`, `enum`): short when **−128…127**; else 8-byte **`int64`**.
- **Unsigned** (`uint`, `uint32`, `ulong`, `uint64`, `flags`): short when **0…255**; else 8-byte **`uint64`**.

### Polymorphic objects

- type token (2 B) or `TOKEN_REG_TYPE` block + property stream + `TOKEN_END`.

### Short list (default)

- **1-byte** element count (uint8, max **255**).
- each element: type token + object stream + `TOKEN_END`.
- **🔷** Count written **inline** in **`bin_write_prop`** override: **`ctx.out_stream.write_byte ((uint8) list.size)`**.

### Long list

- **uint32 BE** element count.
- same per-element layout as short list.
- **🔷** Count written **inline**: **`ctx.out_stream.write_uint32 ((uint32) list.size)`**.

## `Stream` public surface (minimal)

- **`register(alias, gtype)`** — instance; wire alias catalog on **this** **`bin`**
- **`write(Serializable obj)`** — outbound root object
- **`parse()`** — read one root object from **`in_stream`**; returns **`Serializable`**; root type from wire only

**Internal / used by `Serializable` only** (same class, not new helper types):

- **`write_tag`** / **`read_tag`**
- **`write_type`** / **`read_type`**

**🚫** No **`to_output`** / **`from_input`** named constructors.
**🚫** No **`write_*_list_*`** on **`Stream`**.
**🚫** No private **`bin_*`** helpers on **`Serializable`**.
**🚫** No nested **`switch`** in prop encode/decode.
**🚫** Varint / LEB128 / zigzag integer encoding.

---

## `Serializable` naming (`bin_` prefix)

**Serializable** exposes only four override points — all **`bin_`-prefixed** so they do not collide with GObject or app code:

| Method | Words | Role |
| ------ | ----- | ---- |
| **`bin_write`** | 2 | Walk properties, write **`TOKEN_END`** |
| **`bin_read`** | 2 | Read tags until **`TOKEN_END`** |
| **`bin_write_prop`** | 3 | Per-prop encode (override for lists/objects) |
| **`bin_read_prop`** | 3 | Per-prop decode (override for lists/objects) |

- **🚫** No private **`bin_*`** helper methods on the interface.
- **🚫** No nested **`switch`** in **`bin_write_prop`** / **`bin_read_prop`** — case groups only when getter/setter is identical.
- **🚫** **4+** underscore words (e.g. **`bin_read_varint_int64`**).
- **🚫** Varint / LEB128 / zigzag for integers.
- **🔷** Prop walk uses GObject **`get_property`** / **`set_property`** / **`find_property`** — not ad-hoc field access.

#### Add — Full new file `libocrpc/Bin/Stream.vala`.

```vala
/*
 * Copyright (C) 2026 Alan Knowles <alan@roojs.com>
 *
 * This library is free software; you can redistribute it and/or
 * modify it under the terms of the GNU Lesser General Public
 * License as published by the Free Software Foundation; either
 * version 3 of the License, or (at your option) any later version.
 *
 * You should have received a copy of the GNU Lesser General Public License
 * along with this library; if not, write to the Free Software Foundation,
 * Inc., 51 Franklin Street, Fifth Floor, Boston, MA  02110-1301  USA
 */

namespace OLLMrpc.Bin
{
	/**
	 * Per-connection bin codec: I/O streams, JIT key/type maps, {{{write}}} / {{{parse}}}.
	 *
	 * Owned by {@link OLLMrpc.Transport.Connection} and {@link OLLMrpc.Client} as {{{bin}}}.
	 */
	public class Stream : GLib.Object
	{
		public GLib.DataOutputStream out_stream { get; construct; }
		public GLib.DataInputStream in_stream { get; construct; }

		private Gee.HashMap<string, uint16> key_to_token =
			new Gee.HashMap<string, uint16> ();
		private Gee.HashMap<uint16, string> token_to_key =
			new Gee.HashMap<uint16, string> ();

		private Gee.HashMap<string, uint16> alias_to_token =
			new Gee.HashMap<string, uint16> ();
		private Gee.HashMap<uint16, string> token_to_alias =
			new Gee.HashMap<uint16, string> ();

		private uint16 next_key_id = 0;
		private uint16 next_type_id = 0;

		private Gee.HashMap<string, GLib.Type> alias_to_gtype =
			new Gee.HashMap<string, GLib.Type> ();
		private Gee.HashMap<GLib.Type, string> gtype_to_alias =
			new Gee.HashMap<GLib.Type, string> ();

		public const uint16 TOKEN_REG_KEY = 0xFFFF;
		public const uint16 TOKEN_REG_TYPE = 0xFFFE;
		public const uint16 TOKEN_END = 0xFFFD;

		/** Register a wire alias on this connection's stream. */
		public void register (string alias, GLib.Type gtype)
		{
			this.alias_to_gtype.set (alias, gtype);
			this.gtype_to_alias.set (gtype, alias);
		}

		public Stream (
			GLib.DataInputStream in_stream,
			GLib.DataOutputStream out_stream
		) {
			Object (in_stream: in_stream, out_stream: out_stream);
		}

		public void write (Serializable obj) throws GLib.Error
		{
			this.write_type (obj.get_type ());
			obj.bin_write (this);
		}

		public Serializable parse () throws GLib.Error
		{
			GLib.Type root_type = this.read_type ();
			var obj = (Serializable) GLib.Object.new (root_type);
			obj.bin_read (this);
			return obj;
		}

		internal void write_tag (string prop_name) throws GLib.Error
		{
			if (this.key_to_token.has_key (prop_name)) {
				this.out_stream.write_uint16 (this.key_to_token.get (prop_name));
				return;
			}

			uint16 id = this.next_key_id++;
			this.out_stream.write_uint16 (TOKEN_REG_KEY);
			this.out_stream.write_uint16 (id);

			uint8 len = (uint8) uint.min (prop_name.length, 255);
			this.out_stream.write_byte (len);
			size_t written;
			this.out_stream.write_all (((uint8[]) prop_name)[0:len], out written);

			this.key_to_token.set (prop_name, id);
			this.token_to_key.set (id, prop_name);
			this.out_stream.write_uint16 (id);
		}

		internal uint16 read_tag (out string prop_name) throws GLib.Error
		{
			uint16 t = this.in_stream.read_uint16 ();

			if (t == TOKEN_END) {
				prop_name = "";
				return t;
			}

			if (t != TOKEN_REG_KEY) {
				prop_name = this.token_to_key.get (t);
				return t;
			}

			uint16 assigned_id = this.in_stream.read_uint16 ();
			uint8 len = this.in_stream.read_byte ();

			uint8[] buffer = new uint8[len + 1];
			size_t read_bytes;
			this.in_stream.read_all (buffer[0:len], out read_bytes);
			buffer[len] = 0;
			string name = (string) buffer;

			this.key_to_token.set (name, assigned_id);
			this.token_to_key.set (assigned_id, name);
			return this.read_tag (out prop_name);
		}

		internal void write_type (GLib.Type object_type) throws GLib.Error
		{
			string alias = this.gtype_to_alias.get (object_type);
			if (alias == null) {
				GLib.error (
					"Unregistered class type schema: %s",
					object_type.name ()
				);
			}

			if (this.alias_to_token.has_key (alias)) {
				this.out_stream.write_uint16 (this.alias_to_token.get (alias));
				return;
			}

			uint16 id = this.next_type_id++;
			this.out_stream.write_uint16 (TOKEN_REG_TYPE);
			this.out_stream.write_uint16 (id);

			uint8 len = (uint8) uint.min (alias.length, 255);
			this.out_stream.write_byte (len);
			size_t written;
			this.out_stream.write_all (((uint8[]) alias)[0:len], out written);

			this.alias_to_token.set (alias, id);
			this.token_to_alias.set (id, alias);
			this.out_stream.write_uint16 (id);
		}

		internal GLib.Type read_type () throws GLib.Error
		{
			uint16 t = this.in_stream.read_uint16 ();

			if (t != TOKEN_REG_TYPE) {
				string active_alias = this.token_to_alias.get (t);
				return this.alias_to_gtype.get (active_alias);
			}

			uint16 assigned_id = this.in_stream.read_uint16 ();
			uint8 len = this.in_stream.read_byte ();

			uint8[] buffer = new uint8[len + 1];
			size_t read_bytes;
			this.in_stream.read_all (buffer[0:len], out read_bytes);
			buffer[len] = 0;
			string alias = (string) buffer;

			GLib.Type gtype = this.alias_to_gtype.get (alias);
			if (gtype == 0) {
				GLib.error ("Unrecognized type alias: %s", alias);
			}

			this.alias_to_token.set (alias, assigned_id);
			this.token_to_alias.set (assigned_id, alias);
			return this.read_type ();
		}
	}
}
```

#### Add — Full new file.

```vala
/*
 * Copyright (C) 2026 Alan Knowles <alan@roojs.com>
 *
 * This library is free software; you can redistribute it and/or
 * modify it under the terms of the GNU Lesser General Public
 * License as published by the Free Software Foundation; either
 * version 3 of the License, or (at your option) any later version.
 *
 * You should have received a copy of the GNU Lesser General Public License
 * along with this library; if not, write to the Free Software Foundation,
 * Inc., 51 Franklin Street, Fifth Floor, Boston, MA  02110-1301  USA
 */

namespace OLLMrpc.Bin
{
	/**
	 * GObject types that read/write on a {@link Stream} session.
	 *
	 * Override {@link bin_write_prop} / {@link bin_read_prop} for short/long
	 * lists, nested objects, and transient fields.
	 */
	public interface Serializable : GLib.Object
	{
		public virtual void bin_write (Stream ctx) throws GLib.Error
		{
			GLib.ObjectClass obj_class = this.get_class ();
			GLib.ParamSpec[] properties = obj_class.list_properties ();

			foreach (var prop in properties) {
				if (prop.name == "g-type-instance" || prop.name == "ref-count") {
					continue;
				}
				if (!this.bin_write_prop (ctx, prop)) {
					continue;
				}
			}
			ctx.out_stream.write_uint16 (Stream.TOKEN_END);
		}

		public virtual bool bin_write_prop (
			Stream ctx,
			GLib.ParamSpec prop
		) throws GLib.Error
		{
			GLib.Value val = GLib.Value (prop.value_type);
			this.get_property (prop.name, ref val);

			if (prop.value_type == typeof (uint8[])) {
				ctx.write_tag (prop.name);
				unowned uint8[] bytes = (uint8[]) val.get_boxed ();
				if (bytes == null) {
					bytes = new uint8[0];
				}
				ctx.out_stream.write_uint32 ((uint32) bytes.length);
				size_t written;
				ctx.out_stream.write_all (bytes, out written);
				return true;
			}

			switch (prop.value_type) {
				case GLib.Type.STRING:
					ctx.write_tag (prop.name);
					string s = val.get_string () ?? "";
					if (s.length > 65535) {
						GLib.error (
							"Short string prop '%s' is %u bytes — use uint8[] for large payloads",
							prop.name,
							s.length
						);
					}
					ctx.out_stream.write_uint16 ((uint16) s.length);
					size_t written;
					ctx.out_stream.write_all (((uint8[]) s)[0:s.length], out written);
					return true;

				case GLib.Type.BOOLEAN:
					ctx.write_tag (prop.name);
					ctx.out_stream.write_byte (val.get_boolean () ? 1 : 0);
					return true;

				case GLib.Type.CHAR:
					ctx.write_tag (prop.name);
					ctx.out_stream.write_byte ((uint8) val.get_char ());
					return true;

				case GLib.Type.INT8:
					ctx.write_tag (prop.name);
					ctx.out_stream.write_byte ((uint8) val.get_schar ());
					return true;

				case GLib.Type.INT:
				case GLib.Type.INT32:
				case GLib.Type.LONG:
				case GLib.Type.INT64:
					ctx.write_tag (prop.name);
					int64 iv = val.get_int64 ();
					if (iv >= -128 && iv <= 127) {
						ctx.out_stream.write_byte (1);
						ctx.out_stream.write_byte ((uint8) (int8) iv);
						return true;
					}
					ctx.out_stream.write_byte (8);
					ctx.out_stream.write_int64 (iv);
					return true;

				case GLib.Type.ENUM:
					ctx.write_tag (prop.name);
					int64 enum_iv = (int64) val.get_int ();
					if (enum_iv >= -128 && enum_iv <= 127) {
						ctx.out_stream.write_byte (1);
						ctx.out_stream.write_byte ((uint8) (int8) enum_iv);
						return true;
					}
					ctx.out_stream.write_byte (8);
					ctx.out_stream.write_int64 (enum_iv);
					return true;

				case GLib.Type.UCHAR:
				case GLib.Type.UINT8:
					ctx.write_tag (prop.name);
					ctx.out_stream.write_byte ((uint8) val.get_uchar ());
					return true;

				case GLib.Type.UINT:
				case GLib.Type.UINT32:
				case GLib.Type.ULONG:
				case GLib.Type.UINT64:
					ctx.write_tag (prop.name);
					uint64 uv = val.get_uint64 ();
					if (uv <= 255) {
						ctx.out_stream.write_byte (1);
						ctx.out_stream.write_byte ((uint8) uv);
						return true;
					}
					ctx.out_stream.write_byte (8);
					ctx.out_stream.write_uint64 (uv);
					return true;

				case GLib.Type.FLAGS:
					ctx.write_tag (prop.name);
					uint64 flags_uv = (uint64) val.get_flags ();
					if (flags_uv <= 255) {
						ctx.out_stream.write_byte (1);
						ctx.out_stream.write_byte ((uint8) flags_uv);
						return true;
					}
					ctx.out_stream.write_byte (8);
					ctx.out_stream.write_uint64 (flags_uv);
					return true;

				default:
					return false;
			}
		}

		public virtual void bin_read (Stream ctx) throws GLib.Error
		{
			string prop_name;
			uint16 t;

			while ((t = ctx.read_tag (out prop_name)) != Stream.TOKEN_END) {
				GLib.ParamSpec? prop = this.get_class ().find_property (prop_name);
				if (prop == null) {
					continue;
				}
				this.bin_read_prop (ctx, prop);
			}
		}

		public virtual bool bin_read_prop (
			Stream ctx,
			GLib.ParamSpec prop
		) throws GLib.Error
		{
			GLib.Value val = GLib.Value (prop.value_type);

			if (prop.value_type == typeof (uint8[])) {
				uint32 len = ctx.in_stream.read_uint32 ();
				uint8[] buf = new uint8[len];
				size_t read_bytes;
				ctx.in_stream.read_all (buf, out read_bytes);
				val.set_boxed (typeof (uint8[]), buf);
				this.set_property (prop.name, val);
				return true;
			}

			switch (prop.value_type) {
				case GLib.Type.STRING:
					uint16 len = ctx.in_stream.read_uint16 ();
					uint8[] buf = new uint8[len + 1];
					size_t read_bytes;
					ctx.in_stream.read_all (buf[0:len], out read_bytes);
					buf[len] = 0;
					val.set_string ((string) buf);
					this.set_property (prop.name, val);
					return true;

				case GLib.Type.BOOLEAN:
					val.set_boolean (ctx.in_stream.read_byte () == 1);
					this.set_property (prop.name, val);
					return true;

				case GLib.Type.CHAR:
					val.set_char ((char) (int8) ctx.in_stream.read_byte ());
					this.set_property (prop.name, val);
					return true;

				case GLib.Type.INT8:
					val.set_schar ((int8) ctx.in_stream.read_byte ());
					this.set_property (prop.name, val);
					return true;

				case GLib.Type.INT:
				case GLib.Type.INT32:
				case GLib.Type.LONG:
				case GLib.Type.INT64:
					uint8 signed_width = ctx.in_stream.read_byte ();
					if (signed_width == 1) {
						val.set_int64 ((int64) (int8) ctx.in_stream.read_byte ());
						this.set_property (prop.name, val);
						return true;
					}
					if (signed_width != 8) {
						GLib.error (
							"invalid signed integer width %u on prop '%s'",
							signed_width,
							prop.name
						);
					}
					val.set_int64 (ctx.in_stream.read_int64 ());
					this.set_property (prop.name, val);
					return true;

				case GLib.Type.ENUM:
					uint8 enum_width = ctx.in_stream.read_byte ();
					if (enum_width == 1) {
						val.set_enum ((int) (int8) ctx.in_stream.read_byte ());
						this.set_property (prop.name, val);
						return true;
					}
					if (enum_width != 8) {
						GLib.error (
							"invalid enum integer width %u on prop '%s'",
							enum_width,
							prop.name
						);
					}
					val.set_enum ((int) ctx.in_stream.read_int64 ());
					this.set_property (prop.name, val);
					return true;

				case GLib.Type.UCHAR:
				case GLib.Type.UINT8:
					val.set_uchar ((uchar) ctx.in_stream.read_byte ());
					this.set_property (prop.name, val);
					return true;

				case GLib.Type.UINT:
				case GLib.Type.UINT32:
				case GLib.Type.ULONG:
				case GLib.Type.UINT64:
					uint8 unsigned_width = ctx.in_stream.read_byte ();
					if (unsigned_width == 1) {
						val.set_uint64 ((uint64) ctx.in_stream.read_byte ());
						this.set_property (prop.name, val);
						return true;
					}
					if (unsigned_width != 8) {
						GLib.error (
							"invalid unsigned integer width %u on prop '%s'",
							unsigned_width,
							prop.name
						);
					}
					val.set_uint64 (ctx.in_stream.read_uint64 ());
					this.set_property (prop.name, val);
					return true;

				case GLib.Type.FLAGS:
					uint8 flags_width = ctx.in_stream.read_byte ();
					if (flags_width == 1) {
						val.set_flags ((uint) ctx.in_stream.read_byte ());
						this.set_property (prop.name, val);
						return true;
					}
					if (flags_width != 8) {
						GLib.error (
							"invalid flags integer width %u on prop '%s'",
							flags_width,
							prop.name
						);
					}
					val.set_flags ((uint) ctx.in_stream.read_uint64 ());
					this.set_property (prop.name, val);
					return true;

				default:
					return false;
			}
		}
	}
}
```

```vala
namespace OLLMrpcTests
{
	public class TestPair : GLib.Object, OLLMrpc.Bin.Serializable
	{
		public string name { get; set; default = ""; }
		public int count { get; set; default = 0; }
	}

	public class TestBundle : GLib.Object, OLLMrpc.Bin.Serializable
	{
		public string label { get; set; default = ""; }
		public Gee.ArrayList<OLLMrpc.Bin.Serializable> items {
			get;
			set;
			default = new Gee.ArrayList<OLLMrpc.Bin.Serializable> ();
		}

		public override bool bin_write_prop (
			OLLMrpc.Bin.Stream ctx,
			GLib.ParamSpec prop
		) throws GLib.Error {
			if (prop.name != "items") {
				return base.bin_write_prop (ctx, prop);
			}

			ctx.write_tag (prop.name);
			if (this.items.size > 255) {
				GLib.error ("items list too long for short list encoding");
			}
			ctx.out_stream.write_byte ((uint8) this.items.size);
			foreach (var item in this.items) {
				ctx.write_type (item.get_type ());
				item.bin_write (ctx);
			}
			return true;
		}

		public override bool bin_read_prop (
			OLLMrpc.Bin.Stream ctx,
			GLib.ParamSpec prop
		) throws GLib.Error {
			if (prop.name != "items") {
				return base.bin_read_prop (ctx, prop);
			}

			this.items.clear ();
			uint n = ctx.in_stream.read_byte ();
			for (uint i = 0; i < n; i++) {
				GLib.Type child_type = ctx.read_type ();
				var child = (OLLMrpc.Bin.Serializable) GLib.Object.new (child_type);
				child.bin_read (ctx);
				this.items.add (child);
			}
			return true;
		}
	}

	public static int main (string[] args)
	{
		var mem = new GLib.MemoryOutputStream.resizable ();
		var out_stream = new GLib.DataOutputStream (mem);
		out_stream.set_byte_order (GLib.DataStreamByteOrder.BIG_ENDIAN);

		var write_bin = new OLLMrpc.Bin.Stream (null, out_stream);
		write_bin.register ("TestPair", typeof (TestPair));
		write_bin.register ("TestBundle", typeof (TestBundle));

		var original = new TestBundle () {
			label = "smoke",
		};
		original.items.add (new TestPair () { name = "alpha", count = 1 });
		original.items.add (new TestPair () { name = "beta", count = 42 });

		try {
			write_bin.write (original);
			out_stream.close ();

			var bytes = mem.steal_as_bytes ();
			var in_base = new GLib.MemoryInputStream.from_bytes (bytes);
			var in_stream = new GLib.DataInputStream (in_base);
			in_stream.set_byte_order (GLib.DataStreamByteOrder.BIG_ENDIAN);

			var read_bin = new OLLMrpc.Bin.Stream (in_stream, null);
			read_bin.register ("TestPair", typeof (TestPair));
			read_bin.register ("TestBundle", typeof (TestBundle));

			var parsed = read_bin.parse () as TestBundle;
			if (parsed == null) {
				GLib.printerr ("parse returned null\n");
				return 1;
			}
			if (parsed.label != "smoke" || parsed.items.size != 2) {
				GLib.printerr ("round-trip mismatch\n");
				return 1;
			}
			var first = parsed.items[0] as TestPair;
			if (first == null || first.name != "alpha" || first.count != 1) {
				GLib.printerr ("first item mismatch\n");
				return 1;
			}
		} catch (GLib.Error e) {
			GLib.printerr ("bin test failed: %s\n", e.message);
			return 1;
		}

		return 0;
	}
}
```
