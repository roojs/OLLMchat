## Purpose

- **🔷** Point every shipping caller at **`Folder.fetch_*`** / **`insert_file`** — not PM project-index methods.
- **🔷** **🚫** No `get_file_from_active_project`, `convert_fake_file_to_real`, `get_folder_at_path` in updated call sites.

---

## Caller rewrites (🔷)

### File dropdown — `liboccoder/FileDropdown.vala`

- **🚫** `set_item_model(project.project_files)`
- **🔷** `yield project.fetch_file_list()` → own `ListStore`; selection → `yield project.fetch_file(path)`

### Approvals — `liboccoder/Approvals.vala`

- **🚫** `project_files.review_files`
- **🔷** `yield project.fetch_pending_approvals()`; refresh after `File` RPC approve/revert

### Tools — `liboctools/*`, `liboccoder/Task/*`

- **🚫** `project_manager.get_file_from_active_project(...)` → **`yield active_project.fetch_file(...)`**
- **🚫** `convert_fake_file_to_real` → **`yield active_project.insert_file(...)`**
- **🚫** `project_files.folder_map` → **`yield active_project.contains_folder(dir_path)`**

### Codebase search

- **🔷** `vector.search` RPC — **🚫** `project_files.get_ids`

### Agent factory

- **🔷** `file_cache` / open tabs — **🚫** `get_recent_list`

### Source view

- **🔷** `manager.active_file` — **🚫** `project_files.get_active_file()`

### ProjectList — `libocfiles/ProjectList.vala`

- **🚫** `get_folder_in_any_project` walking maps → **`yield manager.fetch_folder(path)`**
