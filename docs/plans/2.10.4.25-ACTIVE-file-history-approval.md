# 2.10.4.25 — Pending change approval (FileHistory chunks)

> **Do not update `docs/plans/1.0-summary.md` for this plan.**

**Status:** **ACTIVE** — daemon rules **✅** confirmed; concrete proposals below; **⏳** no Vala until user approves implementation

**Parent:** [`2.10.4.0-summary.md`](2.10.4.0-summary.md)

**Pointer:** `docs/guide-to-writing-plans.md` · proposed Vala follows [`docs/coding-standards.md`](coding-standards.md) **Method names and new methods**

---

## Purpose

**🔷** The user reviews **change chunks** (`file_history` rows), not files in the abstract.

**🔷** Approve means: accept pending history through the head row (that row and every pending row behind it on the same file).

**🔷** Reject means: roll the file back to a chosen history row. The RPC must return **restored file contents** so the editor buffer updates without a separate read race.

**🔷** v1 / early draft: one popover row per file, **`approve_id`** / **`reject_id`** on each row. Richer per-chunk UI **⏳** later.

**🔷** **List fetch:** one SQL query on **`filebase`** with correlated subqueries for **`approve_id`** / **`reject_id`**. Project scope: **`Folder.roots()`** on the daemon (same distinct paths as **`Folder.roots` RPC** / client **`yield project.roots()`**) → **`instr(file_history.path, root + '/')`** per root, OR-combined inside the **`filebase.id IN (…)`** subquery (same path-condition pattern as **`Folder.load_children`**). **Not** **`project.path`** alone — alias targets can live outside the project directory prefix. Returns **`FileWithHistory[]`**. **No** in-memory **`ReviewFiles`** scan.

---

## Naming

**ℹ️** [`docs/coding-standards.md`](coding-standards.md): one word ideal, two okay, three risky, four you messed up.

- **`FileWithHistory`** — approvals list row: **`filebase`** display fields + **`approve_id`** / **`reject_id`** (three words; name is explicit — do not overload **`File`** or **`FileHistory`**)
- **`FileHistory`** — unchanged: one **`file_history`** chunk / approve-revert RPC target
- **`FileHistory.approve`** / **`FileHistory.revert`** — RPC; **`FileParams.id`** = `file_history` row id ( **`path`** = file)
- Reject content on wire: same as **`File.read`** — **`response.msg`** + **`response.msg_encode`**

---

## Daemon rules (several pending changes on one file)

**🔷 Approve:** **`approve_id`** = newest pending row. Approve marks that row and every older pending row for the file approved; file leaves the list when nothing pending remains.

**🔷 Reject:** **`reject_id`** = newest history row with a backup on disk (shipping **`File.revert()`** rule). Restore from backup; return restored text in **`response.msg`**; mark pending rejected.

**✅** User confirmed (2026-07-01).

---

## User interface

**Keep:** task-due button, popover, Approve / Reject, **`file_selected(File)`** for the editor.

**Change:** popover model is **`FileWithHistory[]`** ( **`FileBase.display_approval_*`** bindings). Selection stores **`approve_id`** / **`reject_id`**. Opening a row resolves **`File`** from **`ProjectManager.file_cache`** by **`path`** (same as today).

**🚫** No **`approve()`** / **`revert()`** on V2 **`File`**.

## Concrete code proposals

**ℹ️** Proposed Vala follows [`docs/coding-standards.md`](coding-standards.md) (router slugs: `temporary-variables`, `this-prefix`, `reducing-nesting`, `try-catch-scope`, `defensive-code-null-checks`, `gee-hashmap-access`, `arraylist-for-strings`, `signal-handlers-construct`, `method-names-new-methods`, `switch-case`, `glib-namespace-prefix`).

| Rule | This plan |
|------|-----------|
| **`var` on locals** | No `string x =` / `int n =` locals; `string[]` only as `string[] ids = {}` |
| **`this.`** | Instance members always prefixed |
| **`Gee.HashMap`** | `.set()` / `.get()` / `.has_key()` — not `[]` |
| **List fetch SQL** | DB computes **`approve_id`** / **`reject_id`** — not Vala loops over history rows |
| **RPC dispatch** | Handlers wired **inside `FileHistory.for_rpc` only** — **🚫** no shared **`construct`** guard; row ctor **`FileHistory(db, …)`** has no RPC wiring |

| **No new handler helpers** | Approvals **`on_*_clicked`** call **`hist.approve.begin`** / **`revert.begin`** inline — no **`approve_selected`** |
| **No defensive client guards** | V2 **`FileHistory.revert`** trusts RPC; no **`row == null`** / empty **`msg`** check |

### 2. `ollmfilesd/FileWithHistory.vala` — **Add** (new file)

**Add** — Extends **`FileBase`**. Carries file display fields from **`filebase`** plus history ids from the list query. **`FileHistory`** remains one history chunk; this type is the approvals popover row.

```vala
namespace OLLMfilesd
{
	/**
	 * File pending approval with newest pending / backup history ids.
	 */
	public class FileWithHistory : FileBase
	{
		/** Newest pending {@code file_history.id}. */
		public int64 approve_id { get; set; default = 0; }

		/** Newest backup {@code file_history.id} for reject. */
		public int64 reject_id { get; set; default = 0; }

		/**
		 * {@code Folder.fetch_pending_approvals} — one {@code selectQuery}.
		 * Project scope from {@link Folder.roots} (not {@code project.path} alone).
		 */
		public static Gee.ArrayList<GLib.Object> pending(
			ProjectManager manager,
			Folder project
		) throws Error {
			var list = new Gee.ArrayList<GLib.Object>();
			var root_folders = project.roots();
			string[] path_conds = {};
			foreach (var root in root_folders) {
				var escaped_path = root.path.replace("'", "''");
				path_conds += "(instr(file_history.path, '"
					+ escaped_path + "/') = 1 OR file_history.path = '"
					+ escaped_path + "')";
			}
			var root_scope = " AND (" + string.joinv(" OR ", path_conds) + ")";
			var q = """
SELECT
	filebase.id,
	filebase.path,
	filebase.last_change_type,
	(
		SELECT
			file_history.id
		FROM
			file_history
		WHERE
				file_history.filebase_id = filebase.id
			AND
				file_history.status = 0
		ORDER BY
			file_history.timestamp DESC
		LIMIT 1
	) AS approve_id,
	(
		SELECT
			file_history.id
		FROM
			file_history
		WHERE
				file_history.filebase_id = filebase.id
			AND
				file_history.backup_path != ''
		ORDER BY
			file_history.timestamp DESC
		LIMIT 1
	) AS reject_id
FROM
	filebase
WHERE
		filebase.is_need_approval = 1
	AND
		filebase.delete_id = 0
	AND
		filebase.base_type = 'f'
	AND
		filebase.id IN (
			SELECT
				file_history.filebase_id
			FROM
				file_history
			WHERE
				file_history.status = 0""" + root_scope + """
		)
""";
			var rows = new Gee.ArrayList<FileWithHistory>();
			var query = new SQ.Query<FileWithHistory>.with_properties(
				manager.db,
				"filebase",
				new string[] { "manager" },
				new Value[] { manager }
			);
			query.selectQuery(q, rows);
			foreach (var row in rows) {
				list.add(row);
			}
			return list;
		}
	}
}
```

**ℹ️** **`SELECT`** lists **`id`**, **`path`**, **`last_change_type`** only. Project scope: **`project.roots()`** → OR **`instr(file_history.path, root + '/')`** conditions inside the **`IN`** subquery ( **`Folder.load_children`** pattern). **🚫** Not **`project.path`** / **`LIKE`** alone. Display uses inherited **`FileBase.display_approval_*`**. **`File`** wire type unchanged.

```vala
		private ProjectManager rpc_manager;

		public signal void rpc_approve(OLLMrpc.Request request);
		public signal void rpc_revert(OLLMrpc.Request request);

		/**
		 * Wire dispatch singleton ({@code FileHistory.approve} / {@code revert}).
		 */
		public FileHistory.for_rpc(ProjectManager manager)
		{
			Object();
			this.rpc_manager = manager;
			this.rpc_approve.connect((request) => {
				var p = (FileParams) request.param;
				var file = this.rpc_manager.get_file_from_active_project(
					p.path
				);
				if (file == null) {
					request.reply(new OLLMrpc.Response() {
						id = request.id,
						error = new OLLMrpc.Error(
							OLLMrpc.RpcErrorCode.INTERNAL_ERROR,
							"file not found"
						)
					});
					return;
				}
				var rows = new Gee.ArrayList<FileHistory>();
				FileHistory.query(this.rpc_manager.db).select(
					"WHERE id = %lld".printf(p.id),
					rows
				);
				rows.get(0).approve(this.rpc_manager.db, file);
				var row = new File(this.rpc_manager);
				row.copy_from(file, {
					"manager",
					"buffer",
					"parent"
				});
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					result = row,
					result_type = "File",
					msg = "ok"
				});
			});
			this.rpc_revert.connect((request) => {
				var p = (FileParams) request.param;
				var rows = new Gee.ArrayList<FileHistory>();
				FileHistory.query(this.rpc_manager.db).select(
					"WHERE id = %lld".printf(p.id),
					rows
				);
				rows.get(0).revert.begin(
					request,
					this.rpc_manager,
					(obj, res) => {
						rows.get(0).revert.end(res);
					}
				);
			});
		}
```

**Add** — Instance approve / revert (history row methods):

```vala
		/**
		 * Approve pending history through this row (head) for @file.
		 */
		public void approve(SQ.Database db, File file)
		{
			var pending = new Gee.ArrayList<FileHistory>();
			FileHistory.query(db).select(
				"WHERE filebase_id = %lld AND status = 0 "
				+ "AND timestamp <= %lld".printf(
					file.id,
					this.timestamp
				),
				pending
			);
			foreach (var row in pending) {
				row.status = 1;
				FileHistory.query(db).updateById(row);
			}
			file.is_need_approval = false;
			file.last_change_type = "";
			file.saveToDB(db, null, false);
		}

		/**
		 * Revert @file from this row's backup; reply with restored content.
		 * Never throws — every failure path calls {@code request.reply} and returns.
		 */
		public async void revert(
			OLLMrpc.Request request,
			ProjectManager manager
		) {
			var p = (FileParams) request.param;
			var file = manager.get_file_from_active_project(p.path);
			if (file == null) {
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					error = new OLLMrpc.Error(
						OLLMrpc.RpcErrorCode.INTERNAL_ERROR,
						"file not found"
					)
				});
				return;
			}
			if (this.change_type == "added") {
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					error = new OLLMrpc.Error(
						OLLMrpc.RpcErrorCode.INTERNAL_ERROR,
						"Cannot revert added files"
					)
				});
				return;
			}
			if (this.backup_path == ""
				|| !GLib.FileUtils.test(
					this.backup_path,
					GLib.FileTest.EXISTS
				)) {
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					error = new OLLMrpc.Error(
						OLLMrpc.RpcErrorCode.INTERNAL_ERROR,
						"backup file does not exist"
					)
				});
				return;
			}
			var now = new GLib.DateTime.now_local();
			var revert_history = new FileHistory(
				manager.db,
				file,
				"revert",
				now
			);
			revert_history.status = 1;
			try {
				yield revert_history.commit();
			} catch (GLib.Error e) {
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					error = new OLLMrpc.Error(
						OLLMrpc.RpcErrorCode.INTERNAL_ERROR,
						e.message
					)
				});
				return;
			}
			var backup_file = GLib.File.new_for_path(this.backup_path);
			var target_file = GLib.File.new_for_path(this.path);
			var parent_dir = target_file.get_parent();
			try {
				if (parent_dir != null && !parent_dir.query_exists()) {
					parent_dir.make_directory_with_parents(null);
				}
				backup_file.copy(
					target_file,
					GLib.FileCopyFlags.OVERWRITE,
					null,
					null
				);
			} catch (GLib.Error e) {
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					error = new OLLMrpc.Error(
						OLLMrpc.RpcErrorCode.INTERNAL_ERROR,
						e.message
					)
				});
				return;
			}
			file.last_modified = now.to_unix();
			file.saveToDB(manager.db, null, false);
			this.status = -1;
			FileHistory.query(manager.db).updateById(this);
			file.last_change_type = "";
			file.is_need_approval = false;
			file.saveToDB(manager.db, null, false);
			uint8[] data;
			string etag;
			try {
				yield target_file.load_contents_async(
					null,
					out data,
					out etag
				);
			} catch (GLib.Error e) {
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					error = new OLLMrpc.Error(
						OLLMrpc.RpcErrorCode.INTERNAL_ERROR,
						e.message
					)
				});
				return;
			}
			var row = new File(manager);
			row.copy_from(file, {
				"manager",
				"buffer",
				"parent"
			});
			request.reply(new OLLMrpc.Response() {
				id = request.id,
				result = row,
				result_type = "File",
				msg = row.is_text
					? (string) data
					: GLib.Base64.encode(
						data[
							0:data.length > 0
								? data.length - 1
								: 0
						]
					),
				msg_encode = row.is_text ? 0 : 1
			});
		}
```

**Note:** **`revert` never throws.** Known bad cases (missing file, added row, missing backup) → **`request.reply` + `return`**. **`try`/`catch`** only on **`commit`**, filesystem restore (**mkdir** + **copy**), and **`load_contents_async`** — each block replies and returns; nothing re-throws. Daemon does not touch **`file.buffer`**.

### 8. `libocfiles/V2/FileHistory.vala` — **Add** (new file, V2 RPC client)

**Add** — Instance **`approve()`** / **`revert()`** only (no DB).

```vala
namespace OLLMfiles
{
	/**
	 * File change history row (V2 wire + approve/revert RPC).
	 */
	public class FileHistory : Object
	{
		public static void rpc_register()
		{
			OLLMrpc.register("FileHistory", typeof(FileHistory));
		}

		public weak ProjectManager manager { get; set; }
		public int64 id { get; set; default = 0; }
		public string path { get; set; default = ""; }

		/**
		 * Approve through {@link id} ({@code FileHistory.approve} RPC).
		 */
		public async void approve()
		{
			yield this.manager.rpc.call(new OLLMrpc.Request() {
				method = "FileHistory.approve",
				param = new OLLMfilesd.FileParams() {
					path = this.path,
					id = this.id
				}
			});
		}

		/**
		 * Revert to {@link id}; apply content from {@code response.msg}.
		 */
		public async void revert()
		{
			var response = yield this.manager.rpc.call(
				new OLLMrpc.Request() {
					method = "FileHistory.revert",
					param = new OLLMfilesd.FileParams() {
						path = this.path,
						id = this.id
					}
				}
			);
			var row = (File) response.result;
			row.manager = this.manager;
			var text = response.msg;
			if (response.msg_encode == 1) {
				text = (string) GLib.Base64.decode(response.msg);
			}
			if (row.buffer != null) {
				row.buffer.set_text(text);
			}
			this.manager.file_cache.set(row.path, row);
		}
	}
}
```
