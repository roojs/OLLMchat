## Purpose

**üî∑** The user reviews **change chunks** (`file_history` rows), not files in the abstract.

**üî∑** Approve means: accept pending history through the head row (that row and every pending row behind it on the same file).

**üî∑** Reject means: roll the file back to a chosen history row. Daemon restores on disk and updates DB; **client** refreshes the editor buffer via **`File.read`** after **`FileHistory.revert`** (same pattern as any other disk change).

**üî∑** v1 / early draft: one popover row per file, **`approve_id`** / **`reject_id`** on each row. Richer per-chunk UI **‚è≥** later.

**üî∑** **List fetch:** one SQL query on **`filebase`** with correlated subqueries for **`approve_id`** / **`reject_id`**. Project scope: **`Folder.roots()`** on the daemon (same distinct paths as **`Folder.roots` RPC** / client **`yield project.roots()`**) ‚Üí **`instr(file_history.path, root + '/')`** per root, OR-combined inside the **`filebase.id IN (‚Ä¶)`** subquery (same path-condition pattern as **`Folder.load_children`**). **Not** **`project.path`** alone ‚Äî alias targets can live outside the project directory prefix. Returns **`FileWithHistory[]`**. **No** in-memory **`ReviewFiles`** scan.

## Naming

**‚ÑπÔ∏è** [`docs/coding-standards.md`](coding-standards.md): one word ideal, two okay, three risky, four you messed up.

- **`FileWithHistory`** ‚Äî approvals list row: **`filebase`** display fields + **`approve_id`** / **`reject_id`** (three words; name is explicit ‚Äî do not overload **`File`** or **`FileHistory`**)
- **`FileHistory`** ‚Äî unchanged: one **`file_history`** chunk / approve-revert RPC target
- **`FileHistory.approve`** / **`FileHistory.revert`** ‚Äî RPC; **`FileParams.id`** = `file_history` row id ( **`path`** = file)
- **`FileHistory.revert`** reply: updated **`File`** row only (**`msg = "ok"`**) ‚Äî **üö´** no content in **`response.msg`**; client calls **`File.read`**

---

## Daemon rules (several pending changes on one file)

**üî∑ Approve:** **`approve_id`** = newest pending row. Approve marks that row and every older pending row for the file approved; file leaves the list when nothing pending remains.

**üî∑ Reject:** **`reject_id`** = newest history row with a backup on disk (shipping **`File.revert()`** rule). Restore from backup; mark pending rejected; reply updated **`File`** row. Client **`File.read`** loads buffer content.

### 2. `ollmfilesd/FileWithHistory.vala` ‚Äî **Add** (new file)

**Add** ‚Äî Extends **`FileBase`**. Carries file display fields from **`filebase`** plus history ids from the list query. **`FileHistory`** remains one history chunk; this type is the approvals popover row.

```vala
namespace OLLMfilesd
{
	/**
	 * File pending approval with newest pending / backup history ids.
	 */
	public class FileWithHistory : FileBase
	{
		public FileWithHistory(ProjectManager manager)
		{
			base(manager);
		}

		public override string to_summary(
			Gee.HashMap<int, OLLMvector2.SQT.VectorMetadata> keymap,
			string indent)
		{
			return indent + this.path;
		}

		/** Newest pending {@code file_history.id}. */
		public int64 approve_id { get; set; default = 0; }

		/** Newest backup {@code file_history.id} for reject. */
		public int64 reject_id { get; set; default = 0; }

		/**
		 * {@code Folder.fetch_pending_approvals} ‚Äî one {@code selectQuery}.
		 * Project scope from {@link Folder.roots} (not {@code project.path} alone).
		 */
		public static Gee.ArrayList<GLib.Object> pending(
			ProjectManager manager,
			Folder project
		) throws Error {
			var list = new Gee.ArrayList<GLib.Object>();
			var root_folders = project.roots();
			string[] path_conds = {};
			foreach (var root in root_folders) {
				var escaped_path = root.path.replace("'", "''");
				path_conds += "(instr(file_history.path, '"
					+ escaped_path + "/') = 1 OR file_history.path = '"
					+ escaped_path + "')";
			}
			var root_scope = " AND (" + string.joinv(" OR ", path_conds) + ")";
			var q = """
SELECT
	filebase.id,
	filebase.path,
	filebase.last_change_type,
	(
		SELECT
			file_history.id
		FROM
			file_history
		WHERE
				file_history.filebase_id = filebase.id
			AND
				file_history.status = 0
		ORDER BY
			file_history.timestamp DESC
		LIMIT 1
	) AS approve_id,
	(
		SELECT
			file_history.id
		FROM
			file_history
		WHERE
				file_history.filebase_id = filebase.id
			AND
				file_history.backup_path != ''
		ORDER BY
			file_history.timestamp DESC
		LIMIT 1
	) AS reject_id
FROM
	filebase
WHERE
		filebase.is_need_approval = 1
	AND
		filebase.delete_id = 0
	AND
		filebase.base_type = 'f'
	AND
		filebase.id IN (
			SELECT
				file_history.filebase_id
			FROM
				file_history
			WHERE
				file_history.status = 0""" + root_scope + """
		)
""";
			var rows = new Gee.ArrayList<FileWithHistory>();
			var query = new SQ.Query<FileWithHistory>.with_properties(
				manager.db,
				"filebase",
				new string[] { "manager" },
				new Value[] { manager }
			);
			query.selectQuery(q, rows);
			foreach (var row in rows) {
				list.add(row);
			}
			return list;
		}
	}
}
```

**‚ÑπÔ∏è** **`SELECT`** lists **`id`**, **`path`**, **`last_change_type`** only. Project scope: **`project.roots()`** ‚Üí OR **`instr(file_history.path, root + '/')`** conditions inside the **`IN`** subquery ( **`Folder.load_children`** pattern). **üö´** Not **`project.path`** / **`LIKE`** alone. Display uses inherited **`FileBase.display_approval_*`**. **`File`** wire type unchanged.

```vala
		private ProjectManager rpc_manager;

		public signal void rpc_approve(OLLMrpc.Request request);
		public signal void rpc_revert(OLLMrpc.Request request);

		/**
		 * Wire dispatch singleton ({@code FileHistory.approve} / {@code revert}).
		 */
		public FileHistory.for_rpc(ProjectManager manager)
		{
			Object();
			this.rpc_manager = manager;
			this.rpc_approve.connect((request) => {
				var p = (FileParams) request.param;
				var file = this.rpc_manager.get_file_from_active_project(
					p.path
				);
				if (file == null) {
					request.reply(new OLLMrpc.Response() {
						id = request.id,
						error = new OLLMrpc.Error(
							OLLMrpc.RpcErrorCode.INTERNAL_ERROR,
							"file not found"
						)
					});
					return;
				}
				var rows = new Gee.ArrayList<FileHistory>();
				FileHistory.query(this.rpc_manager.db).select(
					"WHERE id = %lld".printf(p.id),
					rows
				);
				rows.get(0).approve(this.rpc_manager.db, file);
				var row = new File(this.rpc_manager);
				row.copy_from(file, {
					"manager",
					"buffer",
					"parent"
				});
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					result = row,
					result_type = "File",
					msg = "ok"
				});
			});
			this.rpc_revert.connect((request) => {
				var p = (FileParams) request.param;
				var rows = new Gee.ArrayList<FileHistory>();
				FileHistory.query(this.rpc_manager.db).select(
					"WHERE id = %lld".printf(p.id),
					rows
				);
				rows.get(0).revert.begin(
					request,
					this.rpc_manager,
					(obj, res) => {
						rows.get(0).revert.end(res);
					}
				);
			});
		}
```

**Add** ‚Äî Instance approve / revert (history row methods):

```vala
		/**
		 * Approve pending history through this row (head) for @file.
		 */
		public void approve(SQ.Database db, File file)
		{
			var pending = new Gee.ArrayList<FileHistory>();
			FileHistory.query(db).select(
				"WHERE filebase_id = %lld AND status = 0 "
				+ "AND timestamp <= %lld".printf(
					file.id,
					this.timestamp
				),
				pending
			);
			foreach (var row in pending) {
				row.status = 1;
				FileHistory.query(db).updateById(row);
			}
			file.is_need_approval = false;
			file.last_change_type = "";
			file.saveToDB(db, null, false);
		}

		/**
		 * Revert @file from this row's backup; reply with updated {@link File} row.
		 * Never throws ‚Äî every failure path calls {@code request.reply} and returns.
		 * Client loads buffer via {@code File.read} ‚Äî no content on this response.
		 */
		public async void revert(
			OLLMrpc.Request request,
			ProjectManager manager
		) {
			var p = (FileParams) request.param;
			var file = manager.get_file_from_active_project(p.path);
			if (file == null) {
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					error = new OLLMrpc.Error(
						OLLMrpc.RpcErrorCode.INTERNAL_ERROR,
						"file not found"
					)
				});
				return;
			}
			if (this.change_type == "added") {
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					error = new OLLMrpc.Error(
						OLLMrpc.RpcErrorCode.INTERNAL_ERROR,
						"Cannot revert added files"
					)
				});
				return;
			}
			if (this.backup_path == ""
				|| !GLib.FileUtils.test(
					this.backup_path,
					GLib.FileTest.EXISTS
				)) {
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					error = new OLLMrpc.Error(
						OLLMrpc.RpcErrorCode.INTERNAL_ERROR,
						"backup file does not exist"
					)
				});
				return;
			}
			var now = new GLib.DateTime.now_local();
			var revert_history = new FileHistory(
				manager.db,
				file,
				"revert",
				now
			);
			revert_history.status = 1;
			try {
				yield revert_history.commit();
			} catch (GLib.Error e) {
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					error = new OLLMrpc.Error(
						OLLMrpc.RpcErrorCode.INTERNAL_ERROR,
						e.message
					)
				});
				return;
			}
			var backup_file = GLib.File.new_for_path(this.backup_path);
			var target_file = GLib.File.new_for_path(this.path);
			var parent_dir = target_file.get_parent();
			try {
				if (parent_dir != null && !parent_dir.query_exists()) {
					parent_dir.make_directory_with_parents(null);
				}
				backup_file.copy(
					target_file,
					GLib.FileCopyFlags.OVERWRITE,
					null,
					null
				);
			} catch (GLib.Error e) {
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					error = new OLLMrpc.Error(
						OLLMrpc.RpcErrorCode.INTERNAL_ERROR,
						e.message
					)
				});
				return;
			}
			file.last_modified = now.to_unix();
			file.saveToDB(manager.db, null, false);
			this.status = -1;
			FileHistory.query(manager.db).updateById(this);
			file.last_change_type = "";
			file.is_need_approval = false;
			file.saveToDB(manager.db, null, false);
			var row = new File(manager);
			row.copy_from(file, {
				"manager",
				"buffer",
				"parent"
			});
			request.reply(new OLLMrpc.Response() {
				id = request.id,
				result = row,
				result_type = "File",
				msg = "ok"
			});
		}
```

**Note:** **`revert` never throws.** Known bad cases (missing file, added row, missing backup) ‚Üí **`request.reply` + `return`**. **`try`/`catch`** only on **`commit`** and filesystem restore (**mkdir** + **copy**) ‚Äî each block replies and returns; nothing re-throws. Daemon does not touch **`file.buffer`** or return file content; client **`File.read`** refreshes the buffer (**2.10.4.26** **G-2**).
