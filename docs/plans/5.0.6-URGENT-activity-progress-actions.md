## Purpose

- **рџ”·** Banner shows one optional action button from `action` + `action_label` on `OLLMrpc.Notification`.
- **рџ”·** Click emits **`notification_reply`**; tools and one RPC path **listen** (signals), no banner `switch` on job type.
- **рџ”·** Tool path: `History.Manager.notification_reply` в†’ browser Cancel.
- **рџ”·** RPC path: `action` prefixed with `rpc.` (e.g. `rpc.Codebase.stop`) в†’ strip prefix в†’ `rpc.call`.
- **рџ”·** Vector: Pause/Resume labels; filesystem scan: no button; browser + Hub download: Cancel.
- **вЏі** Ship before / with polished download cancel UX ([`5.0.5`](5.0.5-webkit-download.md)).

## Current behaviour

- **в„№пёЏ** `ActivityBanner` вЂ” label + progress only.
- **в„№пёЏ** No way to cancel a download or pause/resume a vector scan from the banner.
- **в„№пёЏ** `Codebase.stop` / `Codebase.start` already work on the daemon.

---

## Proposed behaviour

- **рџ”·** Out: emitter в†’ `Window.notification` в†’ `ActivityBanner.notification`.
- **рџ”·** In: button в†’ `ActivityBanner.notification_reply` в†’ Window fans to `History.Manager.notification_reply` and (when `action` has `rpc.` prefix) `rpc.call`.
- **рџ”·** Empty `action_label` в†’ no button (todayвЂ™s look).
- **рџ”·** Pause в†” Resume pair on vector; Cancel ends downloads.

---

## Notification fields

- **рџ”·** `action` вЂ” tool verb (`cancel`) or `rpc.` + RPC method (`rpc.Codebase.stop`, `rpc.Codebase.start`)
- **рџ”·** `action_label` вЂ” button text only (`Cancel`, `Pause`, `Resume`)
- **рџљ«** No new pause/resume RPC methods; no per-method hardcoding in the Window filter

### 6b. `libocwebkit/Browser.vala` вЂ” `notification_reply` (plan-named)

**Why:** `downloads_active` is private; Cancel must run inside `Browser`.

**Where:** class body вЂ” after `download` method (Linux section) or near other download helpers.

**Depends on:** В§3, В§6a.

#### Add вЂ” public reply handler (Linux)

```vala
#if LINUX
	/**
	 * Banner Cancel for an in-flight download (''action'' ''cancel'').
	 *
	 * @param notif ''event.browser.download.*'' with ''message'' = destination path
	 */
	public void notification_reply(OLLMrpc.Notification notif)
	{
		if (!notif.method.has_prefix("event.browser.download.")) {
			return;
		}
		if (notif.action != "cancel") {
			return;
		}
		foreach (var url in this.downloads_inflight.keys) {
			if (this.downloads_inflight.get(url) != notif.message) {
				continue;
			}
			if (!this.downloads_active.has_key(url)) {
				return;
			}
			var active = this.downloads_active.get(url) as Download;
			if (active == null) {
				return;
			}
			active.cancel();
			return;
		}
	}
#endif
```
