# 7.14.1-phase1-core — `ProgressItem`, `ToolList`, `ProgressList`, `PhaseEnum` + `to_human()` (Pango)

**Status:** proposed

**Parent:** `docs/plans/7.14.1-progresslist-runner.md`

**Checklist:** `.cursor/rules/CODING_STANDARDS.md` — **Checklist for all plans**.

---

## Purpose

### **PROGRESS UI: ACTIVITY FIRST — NOT LLM PROPOSALS**

**THE SIDEBAR SHOWS EXECUTION PASSES IN `Details.children` (`build_run_queue` → `run_exec`), NOT `proposed_tools`.**

**`ProgressItem.children` ON `Details` IS THE `ToolList` FIELD `children` (EXECUTION QUEUE).**

**`ProgressItem.status`** uses **`PhaseEnum`**. Default **`NONE`** until that phase is **live**; **`REFINEMENT`** only while **`refine()`** runs; **`TOOLS_RUNNING`** while **`run_exec`** is active; **`COMPLETED`** when the row is finished. **`status_str`** = **`status.to_human()`** — see § **`PhaseEnum`**.

**Naming:** no **`_`** prefix on fields/locals. **`var`** for method locals; **`0u`** for **`items_changed`** counters. **Negative guard** before **`items_changed`** (never **`if (added > 0 && call_changed) { … }`**). **`clear()`**: **`if (old_n == 0) return;`** before **`rows.clear()`**.

**No** GTK in `liboccoder` — model only.

**`to_human()`** — deliver as a **complete method** in **`PhaseEnum`** (below), not a signature-only stub. **`ProgressItem.status_str`** calls **`this.status.to_human()`**; strings are **Pango** markup for **`Gtk`** (**`use_markup`** on the cell). Tune labels in review; align with parent **`7.14`** (≤ 2 words inside **`<b>`** where applicable).

**`7.14.3`** duplicate work: once this lands, trim **`7.14.3`** so it does not redefine **`PhaseEnum`** / stage strings.

#### Add — `to_human()` (full method; place **after** existing **`to_string()`**, still inside **`PhaseEnum`**)

```vala
	/**
	 * Pango markup for the progress stage column ({@link ProgressItem.status_str}).
	 * Gtk: enable markup on the cell/renderer that binds **status-str**.
	 */
	public string to_human()
	{
		switch (this) {
		case NONE:
		case REFINE_COMPLETED:
			return "";
		case COARSE:
			return "<b>Planning</b>";
		case REFINEMENT:
			return "<b>Refine</b>";
		case LIST:
			return "<b>Creating</b>";
		case TASK_LIST_ITERATION:
			return "<b>Updating</b>";
		case EXECUTION:
		case TOOLS_RUNNING:
			return "<b>Executing</b>";
		case POST_EXEC:
			return "<b>Post review</b>";
		case EXEC_VALIDATE:
			return "<b>Review</b>";
		case COMPLETED:
			return "<span foreground=\"#808080\">✓</span>";
		default:
			return "";
		}
	}
```

* **`TOOLS_RUNNING`** / **`COMPLETED`** enum members must exist before this compiles — see **Add** block above for those members.
* **Retrying** (task-list parse / iteration) is **not** distinct in **`PhaseEnum`** today; if product needs it, extend **`PhaseEnum`** or drive markup from **`ProgressItem`** — do not add a second **`to_human`** overload here.

```vala
namespace OLLMcoder.Task
{

/**
 * List of {@link Tool} rows for one {@link Details} — used for **both** {@link Details.proposed_tools}
 * (refined **proposals**) and {@link Details.children} (**execution queue**). **UI** nested rows use **''children''** only.
 * Implements {@link GLib.ListModel} using a private
 * {@link Gee.ArrayList} as backing store (same pattern as {@link OLLMfiles.ProjectList}).
 *
 * Mutations update the array and emit {@link GLib.ListModel.items_changed}.
 *
 * @see Details.build_run_queue
 * @see Details.run_exec
 */
public class ToolList : GLib.Object, GLib.ListModel
{
	private Gee.ArrayList<Tool> items = new Gee.ArrayList<Tool>();

	public ToolList()
	{
		Object();
	}

	public Type get_item_type()
	{
		return typeof(ProgressItem);
	}

	public uint get_n_items()
	{
		return (uint) this.items.size;
	}

	public Object? get_item(uint position)
	{
		if (position >= this.items.size) {
			return null;
		}
		return this.items[(int) position];
	}

	/**
	 * Append — matches old ''Gee.ArrayList.add'' / {@link ResultParser} call sites.
	 */
	public void append(Tool t)
	{
		this.items.add(t);
		this.items_changed(this.items.size - 1, 0, 1);
	}

	/**
	 * Alias for {@link append}.
	 */
	public void add(Tool t)
	{
		this.append(t);
	}

	/**
	 * Index access — matches old ''Gee.ArrayList.get''.
	 */
	public Tool get (int index)
	{
		return this.items[index];
	}

	/**
	 * Item count — matches old ''.size'' on Gee list.
	 */
	public int size {
		get { return this.items.size; }
	}

	/**
	 * Remove all items. If already empty, return without calling ''items.clear()''.
	 */
	public void clear()
	{
		var old_n = (uint) this.items.size;
		if (old_n == 0) {
			return;
		}
		this.items.clear();
		this.items_changed(0, old_n, 0);
	}
}

}
```

- **`ToolList`** does not implement **`Gee.Iterable`** — index loops on **`this.proposed_tools`** / **`this.children`**.

## 4. `liboccoder/Task/ProgressList.vala` — new (**entire file**)

**Pattern:** **`OLLMfiles.ProjectList`** — **`items_changed`** on mutations; negative guards only.

```vala
/*
 * Copyright (C) 2026 Alan Knowles <alan@roojs.com>
 *
 * This library is free software; you can redistribute it and/or
 * modify it under the terms of the GNU Lesser General Public
 * License as published by the Free Software Foundation; either
 * version 3 of the License, or (at your option) any later version.
 *
 * This library is distributed in the hope that it will be useful,
 * but WITHOUT ANY WARRANTY; without even the implied warranty of
 * MERCHANTABILITY or FITNESS FOR A PARTICULAR PURPOSE.  See the GNU
 * Lesser General Public License for more details.
 *
 * You should have received a copy of the GNU Lesser General Public License
 * along with this library; if not, write to the Free Software Foundation,
 * Inc., 51 Franklin Street, Fifth Floor, Boston, MA  02110-1301  USA
 */

namespace OLLMcoder.Task
{

/**
 * {@link GLib.ListModel} of {@link ProgressItem} rows for the task progress sidebar: first all
 * {@link Details} from {@link OLLMcoder.Skill.Runner.completed}, then all from ''runner.pending''.
 * Phase 1 stores only {@link Details} instances; {@link GLib.ListModel.get_item_type} still returns
 * ''typeof(ProgressItem)'' so GTK factories bind to the interface.
 *
 * Task-list create / iterate UI (try loop, ''Creating'' / ''Retrying'') is not modelled here — see
 * {@link OLLMcoder.Skill.Runner} and 7.14.2.
 *
 * @see OLLMfiles.ProjectList
 */
public class ProgressList : GLib.Object, GLib.ListModel
{
	/**
	 * Owning runner; held weakly to avoid a reference cycle with {@link OLLMcoder.Skill.Runner.progress}.
	 */
	public weak OLLMcoder.Skill.Runner runner { get; private set; }

	private Gee.ArrayList<ProgressItem> rows = new Gee.ArrayList<ProgressItem>();

	/**
	 * @param r runner that owns this list; stored weakly
	 */
	public ProgressList(OLLMcoder.Skill.Runner r)
	{
		Object();
		this.runner = r;
	}

	/**
	 * {@link GLib.ListModel.get_item_type} — always ''ProgressItem'' (not ''Details'').
	 */
	public Type get_item_type()
	{
		return typeof(ProgressItem);
	}

	/**
	 * {@link GLib.ListModel.get_n_items}.
	 */
	public uint get_n_items()
	{
		return (uint) this.rows.size;
	}

	/**
	 * {@link GLib.ListModel.get_item}.
	 * @param position row index
	 * @return {@link Details} (phase 1) implementing {@link ProgressItem}, or ''null'' if out of range
	 */
	public Object? get_item(uint position)
	{
		if (position >= this.rows.size) {
			return null;
		}
		return this.rows[(int) position];
	}

	/**
	 * Resync **only** the uncompleted (pending) segment — {@link clear_pending(false)} then {@link add_pending(false)}, then one {@link GLib.ListModel.items_changed}.
	 * Completed-backed rows are left untouched. For a full wipe of all rows, use {@link clear}.
	 */
	public void rebuild()
	{
		var n0 = (uint) this.rows.size;
		this.clear_pending(false);
		var after_clear = (uint) this.rows.size;
		var k = (int) n0 - (int) after_clear;
		this.add_pending(false);
		var n1 = (uint) this.rows.size;
		/* Negative guard: no net change — return before items_changed. Do not replace with a positive if wrapping items_changed. */
		if (k == 0 && n1 == after_clear) {
			return;
		}
		this.items_changed(after_clear, (uint) k, n1 - after_clear);
	}

	/**
	 * Drop the non-completed tail: walk backward from the end, ''remove_at'' until the last row is
	 * a {@link Details} with ''status == PhaseEnum.COMPLETED''.
	 * Phase 1: every row is {@link Details} — cast with ''(Details)'', no null checks.
	 *
	 * @param call_changed if ''true'', emit ''items_changed'' when rows were removed; if ''false'' (default), do not (batch with {@link add_completed} / {@link rebuild}).
	 */
	public void clear_pending(bool call_changed = false)
	{
		var removed = 0u;
		while (this.rows.size > 0) {
			var d = (Details) this.rows.get(this.rows.size - 1);
			if (d.status == PhaseEnum.COMPLETED) {
				break;
			}
			this.rows.remove_at(this.rows.size - 1);
			removed++;
		}
		if (removed == 0 || !call_changed) {
			return;
		}
		this.items_changed((uint) this.rows.size, removed, 0);
	}

	/**
	 * Append one row per {@link Details} in ''runner.pending'' (in step order). Call after
	 * {@link clear_pending} when ''pending'' was replaced.
	 *
	 * @param call_changed if ''true'', emit ''items_changed'' for appended rows; if ''false'' (default), do not (batch with {@link add_completed} / {@link rebuild}).
	 */
	public void add_pending(bool call_changed = false)
	{
		var pos = (uint) this.rows.size;
		var added = 0u;
		foreach (var step in this.runner.pending.steps) {
			foreach (var d in step.children) {
				this.rows.add(d);
				added++;
			}
		}
		/* Negative guard — same rule as {@link clear_pending}: never wrap items_changed in a positive if. */
		if (added == 0 || !call_changed) {
			return;
		}
		this.items_changed(pos, 0, added);
	}

	/**
	 * ''clear_pending(false)''; add the single row; ''add_pending(false)''.
	 * Single ''items_changed'' at end.
	 */
	public void add_completed(OLLMcoder.Task.Step step)
	{
		var n0 = (uint) this.rows.size;
		this.clear_pending(false);
		var after_clear = (uint) this.rows.size;
		var k = (int) n0 - (int) after_clear;
		foreach (var d in step.children) {
			this.rows.add(d);
		}
		this.add_pending(false);
		var n1 = (uint) this.rows.size;
		/* Negative guard: no net change — return before items_changed. Same pattern as {@link rebuild}. */
		if (k == 0 && n1 == after_clear) {
			return;
		}
		this.items_changed(after_clear, (uint) k, n1 - after_clear);
	}

	/**
	 * Remove every row. If already empty, return without calling ''rows.clear()''. Else ''clear'' then ''items_changed(0, old_n, 0)''.
	 */
	public void clear()
	{
		var old_n = (uint) this.rows.size;
		if (old_n == 0) {
			return;
		}
		this.rows.clear();
		this.items_changed(0, old_n, 0);
	}
}

}
```
