## Concrete code mock-ups (in plan only)

### Client.Base — libocmcp/Client/Base.vala (from 2.11.2)

Abstract contract; no transport logic. Subclasses implement how to connect, send JSON-RPC, and receive responses.

```vala
namespace OLLMmcp.Client {
  public abstract class Base : Object {
    public abstract async void connect() throws Error;
    public abstract void disconnect();
    public abstract async Gee.ArrayList<OLLMmcp.Factory> list_tools() throws Error;
    public abstract async string call_tool(string name, Json.Object arguments) throws Error;
  }
}
```

- **connect()** — Establish the channel (subprocess started and ready, or HTTP connection open). For MCP, may run `initialize` handshake and store server capabilities.
- **disconnect()** — Tear down (kill subprocess, close HTTP). No async.
- **list_tools()** — Send MCP `tools/list` (or equivalent), parse response into a list of tool descriptors; return a `Gee.ArrayList<OLLMmcp.Factory>` (one Factory per tool: name, description, input_schema). Each Factory later creates an `OLLMmcp.Tool` that calls this client.
- **call_tool(name, arguments)** — Send MCP `tools/call` with tool name and arguments; return the tool result as a string (e.g. `content[].text` or JSON-serialized result).

### Client.Stdio — libocmcp/Client/Stdio.vala

Extends `Client.Base`. **Starts** the MCP server as a subprocess; talks JSON-RPC over stdin/stdout (newline-delimited JSON). Runs under bwrap when available; respects `Config.network`.

**Responsibilities:**

- **connect()**: Build argv from `config.command` and `config.args`. Optionally wrap in bwrap (minimal filesystem, `--unshare-net` unless `config.network`). Spawn subprocess, keep stdin/stdout pipes. Optionally send MCP `initialize` and wait for response; then call `initialized`. Store process and streams for later.
- **disconnect()**: Send optional shutdown/close notification if MCP supports it; then kill subprocess (SIGTERM), close pipes.
- **list_tools()**: Send JSON-RPC request `{"jsonrpc":"2.0","id":1,"method":"tools/list"}` (newline); read newline-delimited response; parse and build `Gee.ArrayList<OLLMmcp.Factory>` from the `tools` array (each item: name, description, inputSchema).
- **call_tool(name, arguments)**: Send JSON-RPC `{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":name,"arguments":arguments}}`; read response; extract result (e.g. `result.content[0].text`) and return as string.

```vala

    private Subprocess? process;
    private DataInputStream? stdin_reader;
    private DataOutputStream? stdout_writer;
    private uint next_id = 1;

    public Stdio(OLLMmcp.Config config) {
      this.config = config;
    }

    public override async void connect() throws Error {
      // Build argv: config.command + config.args
      // Optional: bwrap with --ro-bind /, --unshare-net unless config.network
      // Spawn Subprocess.new(argv, SubprocessFlags.STDIN_PIPE | STDOUT_PIPE)
      // stdin_reader = new DataInputStream(process.get_stdin_pipe())
      // stdout_writer = new DataOutputStream(process.get_stdout_pipe())
      // Optional: send initialize, wait for response, then initialized
    }

    public override void disconnect() {
      if (process != null) {
        process.send_signal(SubprocessSignal.TERM);
        process = null;
      }
      stdin_reader = null;
      stdout_writer = null;
    }

    public override async Gee.ArrayList<OLLMmcp.Factory> list_tools() throws Error {
      var response = yield json_rpc_request("tools/list", new Json.Object());
      // Parse response.result.tools → build Factory for each (name, description, inputSchema)
      var list = new Gee.ArrayList<OLLMmcp.Factory>();
      // ... append factories ...
      return list;
    }

    public override async string call_tool(string name, Json.Object arguments) throws Error {
      var params = new Json.Object();
      params.set_string_member("name", name);
      params.set_object_member("arguments", arguments);
      var response = yield json_rpc_request("tools/call", params);
      // Extract result content (e.g. result.content[0].text) and return
      return extract_result_text(response);
    }

    private async Json.Node? json_rpc_request(string method, Json.Object params) throws Error {
      var req = new Json.Object();
      req.set_string_member("jsonrpc", "2.0");
      req.set_int_member("id", (int) next_id++);
      req.set_string_member("method", method);
      req.set_object_member("params", params);
      string payload = Json.to_string(req, false) + "\n";
      yield stdout_writer.write_async(payload.data);
      var line = yield stdin_reader.read_line_async();
      return Json.from_string(line);
    }
  }
}
```

### Client.Http — libocmcp/Client/Http.vala

Extends `Client.Base`. **Connects** to an already-running MCP server at `config.url`; no process lifecycle. JSON-RPC via HTTP POST (and optionally SSE for streaming; for list_tools/call_tool, request/response is enough).

**Responsibilities:**

- **connect()**: Optionally validate URL and/or perform MCP `initialize` over HTTP (POST to e.g. `config.url` with JSON-RPC body). Store base URL and session state if needed.
- **disconnect()**: No process to kill; close any HTTP session / clear state.
- **list_tools()**: HTTP POST to `config.url` (or endpoint like `/mcp`) with body `{"jsonrpc":"2.0","id":1,"method":"tools/list"}`; parse response and build `Gee.ArrayList<OLLMmcp.Factory>`.
- **call_tool(name, arguments)**: HTTP POST with `{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":name,"arguments":arguments}}`; parse response and return result text.

```vala
namespace OLLMmcp.Client {
  public class Http : Base {
    private OLLMmcp.Config config;
    private string base_url;
    private uint next_id = 1;
    private Soup.Session? session;

    public Http(OLLMmcp.Config config) {
      this.config = config;
      this.base_url = config.url;
    }

    public override async void connect() throws Error {
      session = new Soup.Session();
      // Optional: POST initialize, wait for response
    }

    public override void disconnect() {
      session = null;
    }

    public override async Gee.ArrayList<OLLMmcp.Factory> list_tools() throws Error {
      var response = yield json_rpc_request("tools/list", new Json.Object());
      var list = new Gee.ArrayList<OLLMmcp.Factory>();
      // Parse response.result.tools → build Factory for each
      return list;
    }

    public override async string call_tool(string name, Json.Object arguments) throws Error {
      var params = new Json.Object();
      params.set_string_member("name", name);
      params.set_object_member("arguments", arguments);
      var response = yield json_rpc_request("tools/call", params);
      return extract_result_text(response);
    }

    private async Json.Node? json_rpc_request(string method, Json.Object params) throws Error {
      var req = new Json.Object();
      req.set_string_member("jsonrpc", "2.0");
      req.set_int_member("id", (int) next_id++);
      req.set_string_member("method", method);
      req.set_object_member("params", params);
      string body = Json.to_string(req, false);
      var msg = new Soup.Message("POST", base_url);
      msg.set_request_body_from_bytes("application/json", new Bytes.take(body.data));
      var bytes = yield session.send_and_read_async(msg, GLib.Priority.DEFAULT, null);
      return Json.from_string((string) bytes.get_data());
    }
  }
}
```

### Registry.create_client

```vala
// In Registry (libocmcp/Registry.vala):
public OLLMmcp.Client.Base? create_client(Config config) {
  switch (config.transport) {
    case "stdio": return new OLLMmcp.Client.Stdio(config);
    case "http":  return new OLLMmcp.Client.Http(config);
    default: return null;
  }
}
```
