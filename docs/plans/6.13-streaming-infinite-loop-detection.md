## Purpose

Detect when streamed assistant output repeats the same delta pattern (infinite generation loop). **`bool`** returns flag failure; **`OllmError.FAILED`** is thrown at the NDJSON reader boundary (`**process_json_streaming**` / `**process_streaming**`) and in **`ChatCompletions`** SSE when the check fails, so upper layers still see **`Error**` like other stream failures (e.g. retry).

**Design (bool chain, minimal throws):**

- **Chunk / loop state lives on `**Response.Chat`** only: **`back_tokens`** (and **`check_back_token()`**) belong with the streaming **`Response`** you already construct per request — **no** duplicate array on **`Call`**, **no** `**clear()**` in **`send()`** / **`exec_stream()`**; a new **`Response.Chat`** has an empty list by construction.
- `**check_back_token()**` → **`Response.Chat`** instance method → `**bool**` (`**false**` = loop pattern). No throw inside it.
- `**process_streaming_chunk()**` on **`Call.Chat**` → `**bool**` (`**false**` = stop this stream). Uses **`response.back_tokens`** and **`response.check_back_token()`**. **No throw** for loop detection; `**return false**` when `**!response.check_back_token()**`, otherwise `**return true**` at normal exits.
- `**process_json_chunk()**` → `**bool**`: `**true**` = keep reading NDJSON lines (including “parse failed, skip line”); `**false**` = `**process_streaming_chunk**` returned `**false**`. **No other structural change** to that method: same **`try`** / nesting as **`main`** — only the return type, **`return true`** on existing early exits / parse **`catch`**, and **`return this.process_streaming_chunk(stream_chunk);`** instead of a bare call.
- **Single throw at the NDJSON boundary:** `**process_json_streaming**` does `**if (trimmed != "" && !this.process_json_chunk(trimmed)) { throw new OllmError.FAILED("Streaming stopped: …"); }**` (or `**if (trimmed == "") continue;**` then `**if (!this.process_json_chunk(trimmed)) throw …**` in the line loop) so `**send()**` / `**handle_streaming_response**` still propagate `**Error**` like today. **`ChatCompletions`** SSE loop: same message when `**!resp.check_back_token()**` (no `**process_json_chunk**` on that path).

## Acceptance criteria

1. **`back_tokens`** is empty for each new streaming run because it lives on a **new `**Response.Chat`** (`**send()`** / **`exec_stream()`** construct a fresh response) — **no** explicit `**clear()**` on **`Call`**.
2. `**check_back_token()**` on **`Response.Chat`** matches the **verbatim** method in `**### 1b**` below (`**bool**` return; no silent drift).
3. On loop detection, `**process_streaming_chunk**` returns `**false**`; `**process_json_chunk**` returns `**false**`; callers of `**process_json_chunk**` in `**Base**` throw `**OllmError.FAILED**` (same user-visible outcome as today). `**ChatCompletions**` SSE path: throw when `**!resp.check_back_token()**`.
4. Parse / deserialize: only inside `**process_json_chunk**` (`**try**` / `**catch**`, log, return `**true**` to continue); no new helper methods for parsing.
5. Build passes (`**meson compile -C build**`). No new `***.vala**` files. Automated `**libollmchat**` unit tests are not in scope unless you approve adding a harness — **manual** verification on both Ollama and OpenAI streaming paths.

```vala
		public bool check_back_token()
		{
			if (this.back_tokens.size < 10) {
				return true;
			}

			var t0 = this.back_tokens.get(0);
			this.back_tokens.set(0, "");

			int[] matches = { 0 };

			for (int i = 0; i < 4; i++) {
				int pos = this.back_tokens.index_of(t0);

				if (pos < 0 || pos + 5 > this.back_tokens.size) {
					foreach (int m in matches) {
						this.back_tokens.set(m, t0);
					}
					return true;
				}

				matches += pos;

				if (matches.length > 2) {
					int n = matches.length;
					int dist = matches[n - 1] - matches[n - 2];
					if (dist != matches[n - 2] - matches[n - 3] || dist <= 5) {
						matches.resize(matches.length - 1);
						continue;
					}
				}
			}

			foreach (int m in matches) {
				this.back_tokens.set(m, t0);
			}

			var str = this.back_tokens.to_array();

			foreach (int match in matches) {
				if (match == 0) {
					continue;
				}
				if (string.joinv(" ", str[match:match + 5]) != string.joinv(" ", str[0:5])) {
					return true;
				}
			}

			return false;
		}
```
