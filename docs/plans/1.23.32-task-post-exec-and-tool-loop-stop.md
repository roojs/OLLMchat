## 1. Stop remaining tools — tail `contains` (no constant)

**Idea:** After a successful executor parse, `ex.document` is the full response. Do **not** scan every line from the start. Take only the **last two** `\n\n`-separated chunks (final paragraphs / sections), `to_markdown()` is already the body — so **last two blocks** of that string. If fewer than two chunks, use the whole thing. Then **`tail.down().contains("no further tool calls needed")`**. One substring, no `public const`, no backward line loop.

**`liboccoder/Task/Tool.vala`** — method only (no property). Call **`has_task_complete()`** only right after a successful **`run()`** when **`this.document`** is the executor output; then post-exec runs and these tools are not used again for that task.

```vala
	/**
	 * True when the tail of this run's executor output says the task
	 * is complete — no further tool calls in this task. Call only after
	 * exec_extract set this.document.
	 */
	public bool has_task_complete()
	{
		var md = this.document.to_markdown().strip();
		var chunks = md.split("\n\n");
		string tail;
		if (chunks.length >= 2) {
			tail = chunks[chunks.length - 2] + "\n\n" + chunks[chunks.length - 1];
		} else {
			tail = md;
		}
		return tail.down().contains("no further tool calls needed");
	}
```

**No extra storage on `Tool`.** On **`exec_ok`**, behaviour stays as today (**`return`** after parse). Each run already keeps **tool call + result** and **executor markdown** on **`ex.document`** (and **`tool_call_details()`** when there was a tool). **`has_task_complete()`** decides **`break`** in **`run_exec`** (UI message when true). **`run_post_exec()`** builds **`tool_runs_combined`** the same way: walk **`exec_runs` in order**, append each run’s tool output + executor doc, **`break`** on **`has_task_complete()`** — never touches Tools that **`run()`** never started. If every run finished with no **`has_task_complete()`**, the loop walks all **`exec_runs`** (all ran).

Prompt: model puts **no further tool calls needed** near the end when that run fully answers the task.

**`run_exec()`** — keep **`exec_runs`** unchanged; index loop, **`break`** on **`has_task_complete()`**.

```vala
	public async void run_exec() throws GLib.Error
	{
		var task_name = this.task_data.get("Name").to_markdown().strip();
		for (var i = 0; i < this.exec_runs.size; i++) {
			var ex = this.exec_runs.get(i);
			this.add_message(new OLLMchat.Message("ui",
				"Running Tools for Task " + task_name + " — Tool call " +
				(i + 1).to_string()));
			yield ex.run();
			if (ex.has_task_complete()) {
				this.add_message(new OLLMchat.Message("ui",
					"Tool evaluation indicated that **We have enough " +
					"information to complete task**."));
				break;
			}
		}
		yield this.run_post_exec();
		this.exec_done = true;
	}
```

**`run_post_exec()`** — same stop rule: append per run, **`break`** on **`has_task_complete()`**; else continue through every run that **`run_exec`** reached (if no early break, full **`exec_runs`**).

```vala
	public async void run_post_exec() throws GLib.Error
	{
		yield this.fill_model();
		this.chat_call.tools.clear();
		var tpl = OLLMcoder.Skill.PromptTemplate.template("task_post_exec.md");
		var task_base = "task://" + this.slug() + ".md";
		var definition = this.skill_manager.fetch(this);
		string[] run_blocks = {};
		for (var i = 0; i < this.exec_runs.size; i++) {
			var ex = this.exec_runs.get(i);
			run_blocks += ex.tool_call_details() + "\n\n" +
				ex.document.to_markdown();
			if (ex.has_task_complete()) {
				break;
			}
		}
		tpl.system_fill(0);
		tpl.fill(5,
			"task_link_base", task_base,
			"task_definition", this.to_markdown(MarkdownPhase.REFINEMENT),
			"tool_runs_combined", string.joinv("\n\n---\n\n", run_blocks),
			"skill_execute_body", definition.execute,
			"project_description",
			this.runner.sr_factory.project_manager.active_project == null ? "" :
				this.runner.sr_factory.project_manager.active_project.project_description());
		var task_name = this.task_data.get("Name").to_markdown().strip();
		var model_label = this.session.model_usage.model != "" ?
			this.session.model_usage.display_name_with_size() : "";
		var model_part = model_label != "" ? " with " + model_label : "";
		var full_prompt = "## System\n\n" + tpl.filled_system + "\n\n## User\n\n" +
			tpl.filled_user;
		this.add_message(new OLLMchat.Message("ui",
			OLLMchat.Message.fenced(
				"markdown.oc-frame-info.collapsed Summarizing Tool outputs for " +
				task_name + model_part,
				full_prompt)));
		var messages = new Gee.ArrayList<OLLMchat.Message>();
		messages.add(new OLLMchat.Message("system", tpl.filled_system));
		messages.add(new OLLMchat.Message("user", tpl.filled_user));
		this.add_message(new OLLMchat.Message("ui-waiting",
			model_label != "" ?
				"Waiting for response from " + model_label :
				"Waiting for response"));
		var response = yield this.chat_call.send(messages, null);
		var response_text = response != null ? response.message.content : "";
		var parser = new ResultParser(this.runner, response_text);
		parser.exec_post_extract(this);
		if (parser.issues != "") {
			this.issues += "\n" + parser.issues;
			throw new GLib.IOError.INVALID_ARGUMENT(
				"task_post_exec: " + parser.issues);
		}
	}
```

**`ui-waiting`:** **`Waiting for response from `** + **`display_name_with_size()`** when model set; else **`Waiting for response`** (standard for task LLM waits).

Make **`tool_call_details()`** **`internal`** if **`Details`** is another class — same namespace **`OLLMcoder.Task`** allows call from **`Details`**.

### `Details.vala` — **`validate_link`** + **three** helpers only (full body)

**What the old helper was trying to do:** Two places held “the task result” — **`exec_runs[0].document`** (per run) and **`task_output_document`** (post-exec). Picking between them in **every** validate path is noisy.

**Avoid that:** There is **one** place that defines **`task://slug#section`** for a completed task: **`task_output_document`**, filled at the end of **`run_post_exec`**. **`validate_link`** always uses **`ref_task.task_output_document`** for **task://** section checks — no helper, no branch.

**Legacy / replay:** Sessions from before post-exec may still have only **`exec_runs`**. **One** backfill when a task enters **`completed`** (or when loading from disk): if **`task_output_document.headings.size == 0`** and **`exec_runs.get(0).document`** exists, set **`task_output_document`** to that document **once** (same reference is fine for read-only validate), **or** re-parse **`slug.md`** from the task dir. After that, the invariant holds — never repeat during **`validate_link`**.

---

**Split:** **`validate_link`** dispatches by shape; **only three** extra methods — **`validate_link_http`**, **`validate_link_file`**, **`validate_link_task`**. `#anchor`-only, directory-by-stage, task section text, and ordering stay **inline** (no `_section` / `_ordering` / `_directory_stage` helpers).

```vala
	internal void validate_link(Markdown.Document.Format link, MarkdownPhase stage)
	{
		var href = link.href;
		if (link.path == "") {
			var anchor = link.hash;
			if (this.runner.user_request != null &&
					this.runner.user_request.headings.has_key(anchor)) {
				return;
			}
			switch (stage) {
				case MarkdownPhase.POST_EXEC:
					if (this.task_output_document.headings.has_key(anchor)) {
						return;
					}
					break;
				default:
					break;
			}
			this.issues += "\n" + "Invalid reference target \"" + href +
				"\": unknown anchor \"" + anchor + "\".";
			return;
		}
		if (link.scheme == "http" || link.scheme == "https") {
			this.validate_link_http(stage, href);
			return;
		}
		if (link.path != "" && link.scheme == "file") {
			this.validate_link_file(link, stage, href);
			return;
		}
		if (link.scheme == "task") {
			this.validate_link_task(link, stage, href);
			return;
		}
		this.issues += "\n" + "Invalid reference target \"" + href + "\". " +
			"Use only: #anchor (document sections), task://taskname.md or " +
			"task://taskname.md#section, http(s) URL, or absolute file path (must exist).";
	}

	private void validate_link_http(MarkdownPhase stage, string href)
	{
		switch (stage) {
			case MarkdownPhase.LIST:
			case MarkdownPhase.POST_EXEC:
				return;
			case MarkdownPhase.REFINEMENT:
			default:
				this.issues += "\n" +
					"References must not contain http(s) URLs. Do not put URLs in References " +
					"- create a tool call (e.g. web_fetch) in ## Tool Calls to fetch the content instead.";
				return;
		}
	}

	private void validate_link_file(Markdown.Document.Format link,
		MarkdownPhase stage, string href)
	{
		var project = this.runner.sr_factory.project_manager.active_project;
		string resolved_path;
		if (link.is_relative) {
			if (project == null) {
				this.issues += "\n" + "Invalid reference target \"" + href +
					"\": relative file path requires an active project.";
				return;
			}
			resolved_path = link.abspath(project.path);
		} else {
			resolved_path = link.path;
		}
		if (resolved_path == "") {
			return;
		}
		var lookup_path = resolved_path.has_suffix("/") ?
			resolved_path.substring(0, resolved_path.length - 1) : resolved_path;
		if (project != null) {
			if (project.project_files.child_map.has_key(lookup_path)) {
				return;
			}
			if (project.project_files.folder_map.has_key(lookup_path)) {
				switch (stage) {
					case MarkdownPhase.REFINEMENT:
					case MarkdownPhase.POST_EXEC:
						this.issues += "\n" + "Invalid reference target \"" + href +
							"\": path is a directory; use a file path, not a folder.";
						return;
					case MarkdownPhase.LIST:
					default:
						return;
				}
			}
			var under_project = GLib.File.new_for_path(resolved_path);
			var project_root = GLib.File.new_for_path(project.path);
			var is_under_project = under_project.has_prefix(project_root) ||
				under_project.equal(project_root);
			if (link.is_relative && !is_under_project) {
				this.issues += "\n" + "Invalid reference target \"" + href +
					"\": path is outside project folder.";
				return;
			}
		}
		var resolved_file = GLib.File.new_for_path(resolved_path);
		if (!resolved_file.query_exists()) {
			this.issues += "\n" + "Invalid reference target \"" + href +
				"\": file does not exist (resolved from project folder).";
			return;
		}
		if (resolved_file.query_file_type(GLib.FileQueryInfoFlags.NONE) !=
				GLib.FileType.DIRECTORY) {
			return;
		}
		switch (stage) {
			case MarkdownPhase.REFINEMENT:
			case MarkdownPhase.POST_EXEC:
				this.issues += "\n" + "Invalid reference target \"" + href +
					"\": path is a directory; use a file path, not a folder.";
				return;
			case MarkdownPhase.LIST:
			default:
				return;
		}
	}

	private void validate_link_task(Markdown.Document.Format link,
		MarkdownPhase stage, string href)
	{
		var path = link.path.strip();
		if (path.contains("/")) {
			this.issues += "\n" + "Invalid reference target \"" + href +
				"\": task path must not contain '/'.";
			return;
		}
		var slug = path.has_suffix(".md") ?
			path.substring(0, path.length - 3) : path;
		slug = slug.strip();
		if (slug == "") {
			this.issues += "\n" + "Invalid reference target \"" + href +
				"\": task path is empty.";
			return;
		}
		switch (stage) {
			case MarkdownPhase.POST_EXEC:
				if (slug == this.slug()) {
					if (link.hash == "") {
						this.issues += "\n" + "Invalid reference target \"" + href +
							"\": task reference must include a section anchor (e.g. task://" +
							slug + ".md#section-name).";
						return;
					}
					if (!this.task_output_document.headings.has_key(link.hash)) {
						this.issues += "\n" + "Invalid reference target \"" + href +
							"\": task \"" + slug + "\" has no section \"" + link.hash +
							"\". Available: " +
							this.task_output_document.header_links("task://" + slug + ".md") + ".";
					}
					return;
				}
				if (!this.runner.completed.slugs.has_key(slug)) {
					this.issues += "\n" + "Invalid reference target \"" + href +
						"\": no completed task for \"" + slug + "\".";
					return;
				}
				var ref_done = this.runner.completed.slugs.get(slug);
				if (link.hash == "") {
					this.issues += "\n" + "Invalid reference target \"" + href +
						"\": task reference must include a section anchor.";
					return;
				}
				var doc_done = ref_done.task_output_document;
				if (!doc_done.headings.has_key(link.hash)) {
					this.issues += "\n" + "Invalid reference target \"" + href +
						"\": task \"" + slug + "\" has no section \"" + link.hash +
						"\". Available: " +
						doc_done.header_links("task://" + slug + ".md") + ".";
				}
				return;
			case MarkdownPhase.REFINEMENT:
				var ref_rf = this.runner.completed.slugs.get(slug);
				if (ref_rf == null) {
					this.issues += "\n" + "Invalid reference target \"" + href +
						"\": no completed task for \"" + slug +
						"\" (references must be to completed tasks only).";
					return;
				}
				if (link.hash == "") {
					this.issues += "\n" + "Invalid reference target \"" + href +
						"\": task reference must include a section anchor (e.g. task://" +
						slug + ".md#section-name).";
					return;
				}
				var rdoc = ref_rf.task_output_document;
				if (!rdoc.headings.has_key(link.hash)) {
					this.issues += "\n" + "Invalid reference target \"" + href +
						"\": task \"" + slug + "\" has no section \"" + link.hash +
						"\". Available: " +
						rdoc.header_links("task://" + slug + ".md") + ".";
				}
				if (this.step_index >= 0 && ref_rf.step_index >= 0 &&
						ref_rf.step_index >= this.step_index) {
					this.issues += "\n" + "Reference target \"" + href +
						"\" refers to a task in the same or later section. " +
						"A task may only reference the output of a task from an earlier section. " +
						"I suggest you split this task into its own section.";
				}
				return;
			case MarkdownPhase.LIST:
			default:
				Details? ref_task = this.runner.completed.slugs.get(slug);
				if (ref_task == null) {
					ref_task = this.runner.pending.slugs.get(slug);
				}
				if (ref_task == null) {
					this.issues += "\n" + "Invalid reference target \"" + href +
						"\": no task for \"" + slug + "\".";
					return;
				}
				if (this.step_index != 0) {
					return;
				}
				if (link.hash == "") {
					this.issues += "\n" + "Invalid reference target \"" + href +
						"\": task reference must include a section anchor (e.g. task://" +
						slug + ".md#section-name).";
					return;
				}
				if (ref_task.exec_done) {
					var ldoc = ref_task.task_output_document;
					if (!ldoc.headings.has_key(link.hash)) {
						this.issues += "\n" + "Invalid reference target \"" + href +
							"\": task \"" + slug + "\" has no section \"" + link.hash +
							"\". Available: " +
							ldoc.header_links("task://" + slug + ".md") + ".";
					}
				}
				if (this.step_index >= 0 && ref_task.step_index >= 0 &&
						ref_task.step_index >= this.step_index) {
					this.issues += "\n" + "Reference target \"" + href +
						"\" refers to a task in the same or later section. " +
						"A task may only reference the output of a task from an earlier section. " +
						"I suggest you split this task into its own section.";
				}
				return;
		}
	}
```

```vala
						ret += "#### Task result\n\n";
						ret += this.task_post_exec_summary.to_markdown_with_content()
							.strip() + "\n\n";
						if (this.task_output_document.header_list.size > 0) {
							ret += "**Available sections:** " +
								this.task_output_document.header_links(task_ref) +
								"\n\n";
						}
					}
					continue;
```

**Note:** `task_post_exec_summary` must be non-empty after post-exec; adjust condition if summary stored only in Block.

## 6. Details.write

Replace multi-file per run with single artifact:

```vala
	public void write(string suffix = "")
	{
		var base_dir = this.runner.session.task_dir();
		var path = GLib.Path.build_filename(
			base_dir, this.slug() + suffix + ".md");
		var content = this.task_output_document.to_markdown();
		if (content.strip() == "" && this.exec_runs.size > 0) {
			var ex = this.exec_runs.get (0);
			content = ex.document != null ? ex.document.to_markdown() : "";
		}
		try {
			GLib.FileUtils.set_contents(path, content);
		} catch (GLib.FileError e) {
			GLib.critical("Details.write: failed to write %s: %s", path, e.message);
		}
	}
```
