# 2.11.2. MCP Loader

## Overview

Phase 2 of the MCP loader: abstract Client, **Factory** (libocmcp/Factory/), **Tool** (libocmcp/Tool/), and Loader. Registry is the client factory (creates Client from Config; no ClientFactory/Stdio.Factory/Http.Factory). Loader takes configs and gets Client per config from Registry, fetches factories from client, registers Tools. Standard design: **Mcp/Tool** and **Mcp/Factory**.

Parent: [2.11-mcp-loader-tool.md](2.11-mcp-loader-tool.md)

## Scope

- ✅ **Client directory/namespace:** `libocmcp/Client/` — `Base`, `Stdio`, `Http`
- ✅ **Abstract base:** `connect`, `disconnect`, `tools()`, `call()`; initialize in Stdio/Http
- ✅ **Registry is the client factory:** `create_client(Config)` stdio / http
- ✅ **Factory:** MCP metadata + `create_tool(client, server_id)` → `Tool`
- ✅ **Tool:** `BaseTool`; name `mcp:{server_id}:{tool_name}`; `Request` delegates to `client.call()`
- ✅ **Loader:** per enabled Config → connect → `tools()` → register
- ✅ **Registry.fill_tools():** invokes Loader (sync via `MainLoop`)

## Implemented (Tool / Factory / Request)

- **`OLLMmcp.Factory`:** `name`, `description`, `inputSchema` (`Json.Object`, default empty); wire deserialize normalizes schema; `create_tool()` returns `new Tool(client, server_id, this)`.
- **`OLLMmcp.Tool`:** construct properties `client`, `server_id`, `factory`; `parameter_description` getter builds `@param` lines from `factory.inputSchema` (no `properties` → `""`); `deserialize()` → `new Request(parameters_node.get_object())`; `clone()` non-null.
- **`OLLMmcp.Request`:** `arguments` + constructor; `execute_request()` → `((Tool) tool).client.call(factory.name, arguments)`. `deserialize()` return type is `RequestBase?` because `BaseTool` requires it.

### Registry creates Client

✅

```vala
OLLMmcp.Client.Base create_client(Config config) {
  switch (config.transport) {
    case "stdio": return new OLLMmcp.Client.Stdio(config);
    case "http":  return new OLLMmcp.Client.Http(config);
    default: return null;
  }
}
```

### Loader

✅ — `libocmcp/Loader.vala`

```vala
namespace OLLMmcp {
  public class Loader : Object {
    private Gee.ArrayList<OLLMmcp.Client.Base> clients;
    public async void run(Manager manager, Gee.ArrayList<Config> configs) {
      foreach (var config in configs) {
        if (!config.enabled) continue;
        var client = registry.create_client(config);
        yield client.connect();
        clients.add(client);
        foreach (var f in yield client.tools()) {
          manager.register_tool(f.create_tool(client, config.id));
        }
      }
    }
  }
}
```

✅

var configs = Config.load();
