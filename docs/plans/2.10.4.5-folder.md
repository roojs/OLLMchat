**Client-only:** full `Folder.children` tree in UI — **do not** hydrate whole tree in v1 client; daemon holds graph ([`2.10.4` parent](2.10.4-semantic-index-daemon.md) UI memory model).

## Server — `ollmfilesd`

`OLLMfiles.Folder` in daemon memory; dispatch **`Folder.{method}`**.

**Daemon-internal (no `Folder` RPC):**

- `GitProvider` during scan
- `FolderAnalysis`, `DocumentationAnalysis` inside index pipeline
