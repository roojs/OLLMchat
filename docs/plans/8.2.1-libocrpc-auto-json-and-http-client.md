## Part A — `Bin.Json` (done)

Source: `libocrpc/Bin/Json.vala`. Stdio: `ollmfilesd/StdioConnection.vala`.

### API (implemented)

- **🔷** `✔️` `new Bin.Json(Mode mode = EXPLICIT)`
- **🔷** `✔️` `json_to_bin(Json.Object src, Bin.Stream bin, GLib.Type type = GLib.Type.INVALID)`
- **🔷** `✔️` `bin_to_json(Bin.Stream bin)` → `Json.Node` — AUTO omits `*` meta on encode
- **🔷** `✔️` `json_to_bin_object`, `bin_to_json_object`, `json_member_to_bin`, `bin_member_to_json`

Encode: JSON inputs first, `Bin.Stream bin` last. Decode: `bin` first, return value is output.

### Inbound (stdio, HTTP, tests)

1. Transport parses JSON text → **`Json.Object`** (object root only).
2. `json_to_bin(obj, mem_bin, type)` — `type` when **`Mode.AUTO`** and no `*type`.
3. `Bin.Stream.parse()` → **`Bin.Serializable`**.

### Stdio today

- Inbound: NDJSON → `json_to_bin(root, bin)` → `bin.parse()` → **`Request`**
- Outbound: Serializable → bin → `bin_to_json(bin)` → NDJSON line

### Top-level JSON arrays (upstream)

HF **`GET /api/models`** returns bare **`[…]`** — not `Bin.Json`. Sort out in HTTP / **`libochf`** when implementing (flag, wrap, or loop). See [`8.0.1`](8.0.1-libochf-model-catalog.md).

### Tests

- **🔷** `✔️` Rename call sites (`StdioConnection`)
- **🔷** `⏳` AUTO fixtures under `tests/data/libocrpc/auto-json/`
- **🔷** `⏳` `json_to_bin(obj, bin, typeof(TestModel))` → `bin.parse()` round-trip

## Part B — HTTP on `OLLMrpc.Client`

From [`8.2` Phase 3](8.2-full-rpc-system.md#phase-3--http-transport-bin-inside-json-outside) — **client only**, **JSON metadata** (Hub search/detail). No GGUF download path here.

### Current socket client (`libocrpc/Client.vala`)

- **`connect(Request hello_request)`** — boot **`ollmfilesd`**, open Unix/TCP socket, start **`IOChannel`** read loop, queue hello via **`send_head`**, wait for **`Response`**. **Socket-only** — does nothing useful for HTTP Hub calls.
- **`call(Request request)`** — assign **`param`**, append to **`pending`**, **`send_head`** writes **`bin.write(request)`** on the persistent socket, **`wait_response`** matches **`Response.id`**.
- **`Request`** — **`method`** (e.g. `"Folder.list"`), **`param`** (`CallParam` subclass), **`id`** (assigned by client).
- Inbound: **`bin.parse()`** → **`Response`** or **`Notification`**; **`dispatch_message`** resolves pending by id.

Today there is no **`send_request`** on **`Client`** — that name lives in **`libollmchat/Call/Base`** (libsoup per call). Part B adds the HTTP equivalent **on the same `Client` type**, not a parallel **`Http.Client`** class.

### HTTP mapping (reuse wire types)

Same **`Request`** / **`Response`** / **`call()`** entry point:

- **🔷** No persistent socket — one libsoup round-trip per **`call()`**
- **🔷** **`Request.method`** — URL path (e.g. `"/api/models"`, `"/api/models/{repo}"`)
- **🔷** **`Request.param`** — query fields (`CallParam` subclass properties → `?key=value`)
- **🔷** **`Request.result_type`** — client-only; JSON decode type for **`Response.result`** (not on wire)
- **🔷** **`connect()`** — no-op success when **`socket_path`** is `http://` / `https://` base URL
- **ℹ️** Socket path today: **`bin.write(request)`** on persistent **`IOChannel`** read loop

### 6. `libocrpc/Client.vala` — `send_head()`: HTTP branch (ordered chunks)

**Why:** One libsoup GET per queued **`Request`**; JSON body → **`json_to_bin`** → **`Response.result`**.

**Where:** **`send_head()`** — replace connected guard and split socket vs HTTP send.

**Depends on:** §2, §3, §5.

##### Part 1 — guard

#### Remove

```vala
			if (!this.connected || this.output == null || this.bin == null) {
				return;
			}
			this.sending = true;
			try {
				GLib.debug(
					"send id=%d method=%s",
					head.request.id,
					head.request.method
				);
				this.bin.write(head.request);
				yield this.output.flush_async(
					GLib.Priority.DEFAULT,
					null
				);
				head.sent = true;
			} catch (GLib.Error e) {
				this.complete_pending(
					head.request.id,
					null,
					e
				);
			}
			this.sending = false;
```

#### Replace with

```vala
			var http = this.socket_path.has_prefix ("http://")
				|| this.socket_path.has_prefix ("https://");
			if (!http && (!this.connected || this.output == null || this.bin == null)) {
				return;
			}
			this.sending = true;
			if (http) {
				try {
					GLib.debug(
						"http send id=%d path=%s",
						head.request.id,
						head.request.method
					);
					if (this.http_session == null) {
						this.http_session = new Soup.Session ();
					}
					var url = this.socket_path + head.request.method;
					var param = head.request.param;
					var qs = new GLib.StringBuilder ();
					var first = true;
					foreach (var pspec in param.get_class ().list_properties ()) {
						if (pspec.name == "args") {
							continue;
						}
						var val = GLib.Value ();
						param.get_property (pspec.name, ref val);
						if (val.type () == GLib.Type.STRING) {
							var s = val.get_string ();
							if (s == null || s == "") {
								continue;
							}
							qs.append (first ? "?" : "&");
							first = false;
							qs.append (GLib.Uri.escape_string (pspec.name));
							qs.append ("=");
							qs.append (GLib.Uri.escape_string (s));
							continue;
						}
						if (val.type () == GLib.Type.INT) {
							var iv = val.get_int ();
							qs.append (first ? "?" : "&");
							first = false;
							qs.append (GLib.Uri.escape_string (pspec.name));
							qs.append ("=");
							qs.append (iv.to_string ());
						}
					}
					url += qs.str;
					var message = new Soup.Message ("GET", url);
					var bytes = yield this.http_session.send_and_read_async (
						message,
						GLib.Priority.DEFAULT,
						null
					);
					if (message.status_code < 200
						|| message.status_code >= 300) {
						throw new GLib.IOError.FAILED (
							"HTTP %u for %s",
							message.status_code,
							url
						);
					}
					if (head.request.result_type == GLib.Type.INVALID) {
						throw new Bin.StreamError.PROTOCOL (
							"HTTP call missing result_type"
						);
					}
					var parser = new Json.Parser ();
					parser.load_from_data ((string) bytes.get_data ());
					var root = parser.get_root ();
					if (root.get_node_type () != Json.NodeType.OBJECT) {
						throw new Bin.StreamError.PROTOCOL (
							"HTTP JSON root must be object"
						);
					}
					var mem = new GLib.MemoryOutputStream.resizable ();
					var out_stream = new GLib.DataOutputStream (mem);
					var encode_ctx = new Bin.Stream (null, out_stream);
					this.http_json.json_to_bin (
						root.get_object (),
						encode_ctx,
						head.request.result_type
					);
					out_stream.close ();
					var in_base = new GLib.MemoryInputStream.from_bytes (
						mem.steal_as_bytes ()
					);
					var read_ctx = new Bin.Stream (
						new GLib.DataInputStream (in_base),
						null
					);
					var obj = read_ctx.parse ();
					var response = new Response () {
						id = head.request.id,
					};
					response.result.add (obj);
					head.sent = true;
					this.complete_pending (
						head.request.id,
						response,
						null
					);
				} catch (GLib.Error e) {
					this.complete_pending (
						head.request.id,
						null,
						e
					);
				}
				this.sending = false;
				if (this.pending.size > 0
					&& !this.pending.get(0).sent) {
					this.send_head.begin ();
				}
				return;
			}
			try {
				GLib.debug(
					"send id=%d method=%s",
					head.request.id,
					head.request.method
				);
				this.bin.write(head.request);
				yield this.output.flush_async(
					GLib.Priority.DEFAULT,
					null
				);
				head.sent = true;
			} catch (GLib.Error e) {
				this.complete_pending(
					head.request.id,
					null,
					e
				);
			}
			this.sending = false;
```
