## Part A — `Bin.Json` (done)

Source: `libocrpc/Bin/Json.vala`. Stdio: `ollmfilesd/StdioConnection.vala`.

### API (implemented)

- **🔷** `✔️` `new Bin.Json(Mode mode = EXPLICIT)`
- **🔷** `✔️` `json_to_bin(Json.Object src, Bin.Stream bin, GLib.Type type = GLib.Type.INVALID)`
- **🔷** `✔️` `bin_to_json(Bin.Stream bin)` → `Json.Node` — AUTO omits `*` meta on encode
- **🔷** `✔️` `json_to_bin_object`, `bin_to_json_object`, `json_member_to_bin`, `bin_member_to_json`

Encode: JSON inputs first, `Bin.Stream bin` last. Decode: `bin` first, return value is output.

### Inbound (stdio, HTTP, tests)

1. Transport parses JSON text → **`Json.Object`** (object root only).
2. `json_to_bin(obj, mem_bin, type)` — `type` when **`Mode.AUTO`** and no `*type`.
3. `Bin.Stream.parse()` → **`Bin.Serializable`**.

### Stdio today

- Inbound: NDJSON → `json_to_bin(root, bin)` → `bin.parse()` → **`Request`**
- Outbound: Serializable → bin → `bin_to_json(bin)` → NDJSON line

### Top-level JSON arrays (upstream)

HF **`GET /api/models`** returns bare **`[…]`** — not `Bin.Json`. Sort out in HTTP / **`libochf`** when implementing (flag, wrap, or loop). See [`8.0.1`](8.0.1-libochf-model-catalog.md).

### Tests

- **🔷** `✔️` Rename call sites (`StdioConnection`)
- **🔷** `⏳` AUTO fixtures under `tests/data/libocrpc/auto-json/`
- **🔷** `⏳` `json_to_bin(obj, bin, typeof(TestModel))` → `bin.parse()` round-trip

## Part B — HTTP on `OLLMrpc.Client`

From [`8.2` Phase 3](8.2-full-rpc-system.md#phase-3--http-transport-bin-inside-json-outside) — **client only**, **JSON metadata** (Hub search/detail). No GGUF download path here.

### Current socket client (`libocrpc/Client.vala`)

- **`connect(Request hello_request)`** — boot **`ollmfilesd`**, open Unix/TCP socket, start **`IOChannel`** read loop, queue hello via **`send_head`**, wait for **`Response`**. **Socket-only** — does nothing useful for HTTP Hub calls.
- **`call(Request request)`** — assign **`param`**, append to **`pending`**, **`send_head`** writes **`bin.write(request)`** on the persistent socket, **`wait_response`** matches **`Response.id`**.
- **`Request`** — **`method`** (e.g. `"Folder.list"`), **`param`** (`CallParam` subclass), **`id`** (assigned by client).
- Inbound: **`bin.parse()`** → **`Response`** or **`Notification`**; **`dispatch_message`** resolves pending by id.

Today there is no **`send_request`** on **`Client`** — that name lives in **`libollmchat/Call/Base`** (libsoup per call). Part B adds the HTTP equivalent **on the same `Client` type**, not a parallel **`Http.Client`** class.

### HTTP mapping (reuse wire types)

Same **`Request`** / **`Response`** / **`call()`** entry point:

- **🔷** **`Client.protocol`** — nested **`Client.Protocol`** enum (`SOCKET`, `STDIO`, `TCP`, `HTTP`); set once in constructor from **`socket_path`**
- **🔷** Branches use **`this.protocol == Protocol.HTTP`** (not repeated prefix checks; not **`OLLMrpc.Transport`**)
- **🔷** **`Request.method`** — path + query suffix (caller builds with **`+`**); **`Request.result_type`** — JSON decode type (not on wire)
- **🔷** **`connect()`** — creates **`http_session`** once for **`HTTP`**; caller must **`connect`** before **`call`**
- **🔷** **`call()`** — **`!this.connected`** only (no lazy session, no **`output`** check on HTTP)
- **🔷** **`send_head()`** — **`if (this.protocol == Protocol.HTTP)`** → **`send_http()`**; else existing socket send

**`Client.Protocol` detection** (constructor only; enum nested in **`Client`**, no extra file):

- **`http://`** / **`https://`** → **`Protocol.HTTP`**
- **`tcp://`** → **`Protocol.TCP`**
- **`stdio`** (exact) → **`Protocol.STDIO`**
- else → **`Protocol.SOCKET`**

### 6. `libocrpc/Client.vala` — `send_http()` helper

**Why:** HTTP GET + JSON decode; **`send_head()`** dispatches here (**🔷** user-approved helper).

**Where:** new **`private async`** method before **`send_head()`**.

**Depends on:** §2, §3.

**Coding standards:** match **`Client.vala`** — **no space before `(`**; keep short calls on one line. URL is **`this.socket_path + head.request.method`** — no query reflection in **`Client`**.

#### Add — before `private async void send_head()`:

```vala
		private async void send_http(PendingWrite head) throws GLib.Error
		{
			var url = this.socket_path + head.request.method;
			GLib.debug("id=%d url=%s", head.request.id, url);
			var message = new Soup.Message("GET", url);
			var bytes = yield this.http_session.send_and_read_async(message, GLib.Priority.DEFAULT, null);
			if (message.status_code < 200 || message.status_code >= 300) {
				throw new GLib.IOError.FAILED("HTTP %u for %s", message.status_code, url);
			}
			if (head.request.result_type == GLib.Type.INVALID) {
				throw new Bin.StreamError.PROTOCOL("HTTP call missing result_type");
			}
			var parser = new Json.Parser();
			parser.load_from_data((string) bytes.get_data());
			var root = parser.get_root();
			if (root.get_node_type() != Json.NodeType.OBJECT) {
				throw new Bin.StreamError.PROTOCOL("HTTP JSON root must be object");
			}
			var mem = new GLib.MemoryOutputStream.resizable();
			var encode_ctx = new Bin.Stream(null, new GLib.DataOutputStream(mem));
			this.http_json.json_to_bin(root.get_object(), encode_ctx, head.request.result_type);
			encode_ctx.out_stream.close();
			var read_ctx = new Bin.Stream(new GLib.DataInputStream(new GLib.MemoryInputStream.from_bytes(mem.steal_as_bytes())), null);
			var obj = read_ctx.parse();
			var response = new Response() {
				id = head.request.id,
			};
			response.result.add(obj);
			this.complete_pending(head.request.id, response, null);
		}
```

### 7. `libocrpc/Client.vala` — `send_head()`: HTTP branch

**Why:** One **`if (protocol == HTTP)`** early path; socket send unchanged below. Narrow **`try`/`catch`** in each path.

**Where:** **`send_head()`** — replace socket-only guard + send body.

**Depends on:** §5, §6.

#### Remove

```vala
			if (!this.connected || this.output == null || this.bin == null) {
				return;
			}
			this.sending = true;
			try {
				GLib.debug(
					"send id=%d method=%s",
					head.request.id,
					head.request.method
				);
				this.bin.write(head.request);
				yield this.output.flush_async(
					GLib.Priority.DEFAULT,
					null
				);
				head.sent = true;
			} catch (GLib.Error e) {
				this.complete_pending(
					head.request.id,
					null,
					e
				);
			}
			this.sending = false;
```

#### Replace with

```vala
			if (!this.connected) {
				return;
			}
			this.sending = true;
			if (this.protocol == Protocol.HTTP) {
				try {
					yield this.send_http(head);
					head.sent = true;
				} catch (GLib.Error e) {
					this.complete_pending(head.request.id, null, e);
				}
				this.sending = false;
				return;
			}
			try {
				GLib.debug("id=%d method=%s", head.request.id, head.request.method);
				this.bin.write(head.request);
				yield this.output.flush_async(GLib.Priority.DEFAULT, null);
				head.sent = true;
			} catch (GLib.Error e) {
				this.complete_pending(head.request.id, null, e);
			}
			this.sending = false;
```

## LLM notes

- **🚫** No **`Bin.Json.read`** — inbound is **`json_to_bin`** then **`Bin.Stream.parse()`**.
- **🚫** No top-level **`Json.Array`** in generic HTTP JSON path.
- **🚫** No binary download API in **`libocrpc`**.
- **🚫** Reuse **`Client.call`**; branch on **`Client.protocol`** / **`Protocol`**, not prefix checks. Do not name the enum **`Transport`** — **`OLLMrpc.Transport`** is the wire namespace.
- **🚫** No lazy **`http_session`** init outside **`connect()`**; **`call()`** checks **`connected`** only.
- **🚫** Helper is **`send_http()`**, not **`send_head_http()`**.
- **🚫** **No space before `(`** — match **`Client.vala`** (`write(head)`, `flush_async(...)`, not `write (head)`). Do not break short lines; only break when genuinely long.
- **🚫** **`try`/`catch`** — narrow, around the throwing calls only.
- **🚫** HTTP URL: **`this.socket_path + request.method`** only — caller builds path/query in **`method`** with **`+`**; **`Client`** does not read **`param`** for HTTP.
- **🚫** Read **`docs/coding-standards-router.md`** universal table before implementing Part B.
