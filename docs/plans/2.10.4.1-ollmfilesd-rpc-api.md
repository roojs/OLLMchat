## `method` = `{Object}.{method}`

Same names as `libocfiles` classes and methods. Object not repeated in `params`.

```json
{"method":"Daemon.hello","params":{"protocol":1,"client":"ollmchat"}}
{"method":"ProjectManager.create_project","params":{"path":"/home/alan/proj/newapp"}}
```

## `vector` — not split yet (shape TBD)

Stays here until class names match `libocvector`. Daemon hosts FAISS + SQL; client calls from tools / `BackgroundScan`.

| Wire *(TBD)* | Today |
| ------------ | ----- |
| `vector.status` | `CodebaseSearchTool.init_databases` |
| `vector.search` | `Search.execute`, `RequestCodebaseSearch` — `format`: tool \| json |
| `vector.list` | `oc-vector-search --show-info` |
| `vector.get` | `oc-vector-search --dump-vector` |
| `vector.reset` | `Indexer.reset_database` |
| `vector.project.describe` | `Folder.get_project_description`, `ProjectAnalysis` |

**Daemon-internal:** `VectorBuilder`, `Database.save_index`, `add_vectors_batch`, analysis passes inside index queue.

## Shared wire shapes

**`filebase`** — DB row (`FileBase` fields; omit `parent`, `manager`, `buffer`).

**`vector_metadata`** — one indexed element (`SQT/VectorMetadata`).

**Errors:** `-32600`…`-32603`, `-32001` not found, `-32002` busy, `-32003` io.
