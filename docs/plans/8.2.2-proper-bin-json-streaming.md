## Purpose

- **üî∑** **`gobject_to_json_async`** ‚Äî pipe-backed GObject ‚Üí **`Json.Node`**; decode **`Stream`** exists **before** **`write`**; callback on main context.
- **üî∑** **Not** mem ‚Üí **`steal_as_bytes()`** ‚Üí second **`Stream`** ‚Üí **`bin_to_json`**.
- **üî∑** **GObject ‚Üí JSON only.** JSON ‚Üí GObject stays **`json_to_bin`** + **`parse`** (out of scope).
- **üö´** **No** sync **`begin_gobject_to_json`** / **`finish`**.
- **üö´** **No** ‚Äúcodec‚Äù / **`json_codec`** in new code ‚Äî **`bin_json`** field name at call sites.

---

## Stream design `üî∑`

```text
  encode_bin.write(src)  ‚îÄ‚îÄ‚ñ∫  [pipe]  ‚îÄ‚îÄ‚ñ∫  bin_to_json(decode_bin)  ‚îÄ‚îÄ‚ñ∫  Idle ‚Üí ready(node)
        step 4                  live              step 2 (thread, before step 4 ends)
```

- **üî∑** **`Posix.pipe`** ‚Üí **`UnixInputStream`** (read) + **`UnixOutputStream`** (write).
- **üî∑** **`decode_bin = Stream(read_ds, null)`** ‚Äî created **before** encode thread runs.
- **üî∑** **`encode_bin = Stream(null, write_ds)`** ‚Äî **`write(src)`** then **`write_ds.close()`** (EOF).
- **üî∑** **Two `Stream` instances** on one pipe ‚Äî avoid concurrent **`write`** + **`bin_to_json`** on one object (**`names` / `name_to_token`** race).
- **üî∑** Decode thread **`join()`** after encode; then **`GLib.Idle.add`** ‚Üí **`ready(node)`**.

## Phase 1 ‚Äî `libocrpc/Bin/Json.vala` `‚è≥`

### 1. `libocrpc/Bin/Json.vala` ‚Äî `GObjectToJsonReady` + `gobject_to_json_async`

**Why:** Pipe GObject ‚Üí JSON with decode wired before write; async callback.

**Where:** end of **`Json`** class body ‚Äî immediately before the class closing **`}`** (after **`bin_member_to_json`**).

**Depends on:** none. **`libocrpc/meson.build`** already has **`--pkg=posix`**.

#### Add ‚Äî delegate + `gobject_to_json_async` (pipe, decode thread, join, Idle callback)

```vala
		/** Main-context callback after {@link gobject_to_json_async} finishes. */
		public delegate void GObjectToJsonReady(global::Json.Node node);

		/**
		 * Encode {@link Serializable} to wire bytes on a pipe; decode to JSON tree.
		 *
		 * Creates {@link decode_bin} before {@link encode_bin.write}; decode runs
		 * on a worker thread blocked on the pipe until bytes arrive. Invokes
		 * {@link ready} on the thread-default main context with the {@link Json.Node}.
		 */
		public void gobject_to_json_async(
			Serializable src,
			GObjectToJsonReady ready
		)
		{
			int[] fds = new int[2];
			if (Posix.pipe(fds) != 0) {
				GLib.error("gobject_to_json_async: pipe(2) failed");
			}

			var read_ds = new GLib.DataInputStream(
				new GLib.UnixInputStream(fds[0], true)
			);
			var write_ds = new GLib.DataOutputStream(
				new GLib.UnixOutputStream(fds[1], true)
			);
			var decode_bin = new Stream(read_ds, null);
			var encode_bin = new Stream(null, write_ds);

			var decoded_box = new global::Json.Node[1];
			GLib.Error? decode_err = null;

			var decode_thread = GLib.Thread.try_new<void*>(
				"ollmrpc-gobject-to-json",
				() => {
					try {
						decoded_box[0] = this.bin_to_json(decode_bin);
					} catch (GLib.Error e) {
						decode_err = e;
					}
					return null;
				}
			);
			if (decode_thread == null) {
				GLib.error("gobject_to_json_async: decode thread failed");
			}

			try {
				encode_bin.write(src);
			} catch (GLib.Error e) {
				write_ds.close();
				decode_thread.join();
				GLib.error(
					"gobject_to_json_async: encode failed: %s",
					e.message
				);
			}
			write_ds.close();
			decode_thread.join();

			if (decode_err != null) {
				GLib.error(
					"gobject_to_json_async: decode failed: %s",
					decode_err.message
				);
			}
			if (decoded_box[0] == null) {
				GLib.error("gobject_to_json_async: decode returned null");
			}

			var node = decoded_box[0];
			var invoke = ready;
			GLib.Idle.add(() => {
				invoke(node);
				return false;
			});
		}
```

### 2. `examples/oc-hf.vala` ‚Äî search loop: `gobject_to_json_async`

**Why:** Replace steal workaround with pipe API.

**Where:** **`run_test`** ‚Äî **`foreach (var model in bag.items)`** body.

**Depends on:** ¬ß1.

#### Remove

```vala
				var mem = new GLib.MemoryOutputStream.resizable();
				var encode_ctx = new OLLMrpc.Bin.Stream(
					null,
					new GLib.DataOutputStream(mem)
				);
				encode_ctx.write(model);
				encode_ctx.out_stream.close();
				var read_ctx = new OLLMrpc.Bin.Stream(
					new GLib.DataInputStream(
						new GLib.MemoryInputStream.from_bytes(mem.steal_as_bytes())
					),
					null
				);
				arr.add_element(json_codec.bin_to_json(read_ctx));
```

#### Replace with

```vala
				var waited = false;
				json_codec.gobject_to_json_async(
					model,
					(node) => {
						arr.add_element(node);
						waited = true;
					}
				);
				while (!waited) {
					GLib.MainContext.default().iteration(true);
				}
```

### 4. `ollmfilesd/StdioConnection.vala` ‚Äî outbound NDJSON `write`

**Why:** Stdio outbound GObject ‚Üí JSON via pipe API.

**Where:** **`write`** ‚Äî **`try`** block from **`mem = ‚Ä¶`** through **`bin_to_json`**.

**Depends on:** ¬ß1.

#### Remove

```vala
			var mem = new GLib.MemoryOutputStream.resizable();
			var encode_out = new GLib.DataOutputStream(mem);
			var encode_ctx = new OLLMrpc.Bin.Stream(null, encode_out);
			try {
				encode_ctx.write(serializable);
				encode_out.close();

				var in_base = new GLib.MemoryInputStream.from_bytes(
					mem.steal_as_bytes()
				);
				var read_ctx = new OLLMrpc.Bin.Stream(
					new GLib.DataInputStream(in_base),
					null
				);
				var node = this.json_codec.bin_to_json(read_ctx);
				var gen = new Json.Generator();
				gen.set_pretty(false);
				gen.set_root(node);
				var line = gen.to_data(null);
				this.bin.out_stream.put_string(line);
				this.bin.out_stream.put_byte((uint8) '\n');
				this.bin.out_stream.flush();
			} catch (GLib.Error e) {
				GLib.error("stdio write: %s", e.message);
			}
```

#### Replace with

```vala
			var write_done = false;
			GLib.Error? write_err = null;
			this.json_codec.gobject_to_json_async(
				serializable,
				(node) => {
					try {
						var gen = new Json.Generator();
						gen.set_pretty(false);
						gen.set_root(node);
						var line = gen.to_data(null);
						this.bin.out_stream.put_string(line);
						this.bin.out_stream.put_byte((uint8) '\n');
						this.bin.out_stream.flush();
					} catch (GLib.Error e) {
						write_err = e;
					}
					write_done = true;
				}
			);
			while (!write_done) {
				GLib.MainContext.default().iteration(true);
			}
			if (write_err != null) {
				GLib.error("stdio write: %s", write_err.message);
			}
```
