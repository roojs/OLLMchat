### 1. `libollmchat/Response/Chat.vala` — `detect_looping(string token)`

**Why:** Same back-token insert/trim/**`check_back_token`** loop is duplicated in **`Call.Chat`** and **`Call.ChatCompletions`**.

**Where:** After **`check_back_token()`**, before **`serialize_property`**.

**Depends on:** none.

#### Keep

```vala
			return false;
		}

		public override Json.Node serialize_property(string property_name, Value value, ParamSpec pspec)
```

#### Add — insert **`detect_looping`** between **`check_back_token`** and **`serialize_property`**.

```vala
		/**
		 * Feed streaming delta words into {@link back_tokens} and run {@link check_back_token}.
		 *
		 * @return false when a repeating pattern is detected
		 */
		public bool detect_looping(string token)
		{
			foreach (string w in Regex.split_simple("\\s+", token)) {
				if (w.length == 0) {
					continue;
				}
				this.back_tokens.insert(0, w);
				if (this.back_tokens.size > 100) {
					this.back_tokens.remove_at(this.back_tokens.size - 1);
				}
				if (!this.check_back_token()) {
					return false;
				}
			}
			return true;
		}

```

### 3. `libollmchat/Call/ChatCompletions.vala` — `exec_stream()`: stream/agent hooks + `detect_looping`

**Why:** Agent/session UI depends on **`stream_start`** / **`stream_chunk`** / **`handle_stream_*`**. **`exec_stream`** only accumulates chunks today.

**Where:** **`exec_stream()`**, inside the SSE loop, replace the block from **`var token = resp.addChunk(chunk);`** through the **`back_tokens`** loop.

**Depends on:** §1.

#### Keep

```vala
				if (chunk == null) {
					continue;
				}
```

#### Remove

```vala
				var token = resp.addChunk(chunk);
				if (token == "") {
					continue;
				}
				foreach (string w in Regex.split_simple("\\s+", token)) {
					if (w.length == 0) {
						continue;
					}
					resp.back_tokens.insert(0, w);
					if (resp.back_tokens.size > 100) {
						resp.back_tokens.remove_at(resp.back_tokens.size - 1);
					}
					if (!resp.check_back_token()) {
						throw new OllmError.FAILED(
							"Streaming stopped: output repeated; possible infinite generation loop.");
					}
				}
```

#### Replace with

```vala
				var token = resp.addChunk(chunk);

				if (resp.is_first_chunk) {
					resp.is_first_chunk = false;
					this.stream_start();
					if (this.agent != null) {
						this.agent.handle_stream_started();
					}
				}

				if (resp.new_thinking.length == 0 &&
					resp.new_content.length == 0 &&
					!resp.done &&
					token == "") {
					continue;
				}

				if (resp.new_thinking.length > 0 ||
					resp.new_content.length > 0 ||
					resp.done) {
					bool is_thinking = resp.new_thinking.length > 0;
					string new_text = is_thinking ? resp.new_thinking : resp.new_content;
					this.stream_chunk(new_text, is_thinking, resp);
					if (this.agent != null) {
						this.agent.handle_stream_chunk(new_text, is_thinking, resp);
					}
				}

				if (token == "") {
					continue;
				}
				if (!resp.detect_looping(token)) {
					throw new OllmError.FAILED(
						"Streaming stopped: output repeated; possible infinite generation loop.");
				}
```
