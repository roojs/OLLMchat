## Purpose

- **🔷** **`Bin.Json.from_gobject(src)`** — **`async`**, returns **`Json.Node`**; JSON direction implied by class name; pipe-backed; decode **`Stream`** before **`write`**.
- **🔷** **Not** mem → **`steal_as_bytes()`** → second **`Stream`** → **`bin_to_json`**.
- **🔷** **GObject → JSON only.** JSON → GObject stays **`json_to_bin`** + **`parse`** (out of scope).
- **🚫** **No** **`GObjectToJsonReady`** or other **`delegate`** types.
- **🚫** **No** sync session API (**`begin_from_gobject`** / **`finish`**, etc.).
- **🚫** **No** “codec” naming — field **`bin_json`** at call sites.

- **🔷** **Usage (always, from `async` code):** **`var node = yield bin_json.from_gobject(src);`**
- **🚫** **No** fire-and-forget — always **`yield`** and use **`node`**.
- **🚫** **No** **`.begin`/`.end`** on **`from_gobject`** — sync **`write`** would block-wait inside the override; **`Connection.write`** must be **`async`** so **`StdioConnection`** can **`yield`**.

## Stream design `🔷`

```text
  encode_bin.write(src)  ──►  [pipe]  ──►  bin_to_json(decode_bin)  ──►  return Json.Node
        step 3                  live              step 1 (thread, before step 3 ends)
```

- **🔷** **`Posix.pipe`** → **`UnixInputStream`** + **`UnixOutputStream`**
- **🔷** **`decode_bin = Stream(read_ds, null)`** — before encode
- **🔷** **`encode_bin = Stream(null, write_ds)`** — **`write(src)`**, **`write_ds.close()`**
- **🔷** Two **`Stream`** instances on one pipe (**`names` / `name_to_token`** must not race)
- **🔷** Decode thread **`join()`** after encode; return node (or **`yield promise.future.wait_async()`** if decode resolves via **`GLib.Promise`**)

---

## Thread safety `🔷`

- **🔷** **FG thread:** reads live **`src`** during **`encode_bin.write(src)`** only.
- **🔷** **Worker thread:** builds **`Json.Node`** from pipe bytes (**`bin_to_json`**) — **no** access to **`src`**.
- **🔷** **`from_gobject` is not a general-purpose thread-safe snapshot API.** Same rules as **`Stream.write`**: **do not mutate `src` (or nested GObjects it exposes) while the call is in flight.**
- **ℹ️** Two **`Stream`** instances on one pipe avoid **`names` / `name_to_token`** races between writer and reader on the **same** call.
- **ℹ️** **Concurrent `from_gobject` on different `src` objects** — OK if each **`src`** is stable for its own call.
- **ℹ️** **Concurrent `from_gobject` on the same `src`** while **`src`** is still mutating elsewhere — **not** safe; caller must serialize or stop mutating until the call returns.
- **ℹ️** **Process-wide `Bin.register` maps** — unchanged; register types before use, not mid-flight.
- **🚫** **No** automatic deep-copy of **`src`** in **`from_gobject`** — caller owns consistency (e.g. **`Download`**: finish in-memory updates before flush, or don't overlap flushes on the same **`model`**).
- **💩** If overlapping async flushes become real: caller queues them or copies state — **not** solved inside **`Json.from_gobject`**.

## Phase 1 — `libocrpc/Bin/Json.vala` `✔️`

### 1. `libocrpc/Bin/Json.vala` — `from_gobject`

**Why:** Pipe GObject → JSON; decode wired before write; Vala **`async`** return.

**Where:** end of **`Json`** class — before closing **`}`** (after **`bin_member_to_json`**).

**Depends on:** none. **`libocrpc/meson.build`** already has **`--pkg=posix`**.

#### Add — `from_gobject` (pipe, decode thread, join, return node)

```vala
		/**
		 * Encode {@link Serializable} on a pipe; decode to a JSON tree.
		 *
		 * {@link decode_bin} exists before {@link encode_bin.write}; decode runs
		 * on a worker thread blocked on the pipe until bytes arrive.
		 */
		public async global::Json.Node from_gobject(
			Serializable src
		) throws GLib.Error
		{
			int[] fds = new int[2];
			if (Posix.pipe(fds) != 0) {
				throw new StreamError.PROTOCOL("from_gobject: pipe(2) failed");
			}

			var read_ds = new GLib.DataInputStream(
				new GLib.UnixInputStream(fds[0], true)
			);
			var write_ds = new GLib.DataOutputStream(
				new GLib.UnixOutputStream(fds[1], true)
			);
			var decode_bin = new Stream(read_ds, null);
			var encode_bin = new Stream(null, write_ds);

			var decoded_box = new global::Json.Node[1];
			GLib.Error? decode_err = null;

			var decode_thread = GLib.Thread.try_new<void*>(
				"ollmrpc-gobject-to-json",
				() => {
					try {
						decoded_box[0] = this.bin_to_json(decode_bin);
					} catch (GLib.Error e) {
						decode_err = e;
					}
					return null;
				}
			);
			if (decode_thread == null) {
				throw new StreamError.PROTOCOL(
					"from_gobject: decode thread failed"
				);
			}

			try {
				encode_bin.write(src);
			} catch (GLib.Error e) {
				write_ds.close();
				decode_thread.join();
				throw e;
			}
			write_ds.close();
			decode_thread.join();

			if (decode_err != null) {
				throw decode_err;
			}
			if (decoded_box[0] == null) {
				throw new StreamError.PROTOCOL(
					"from_gobject: decode returned null"
				);
			}

			return decoded_box[0];
		}
```
