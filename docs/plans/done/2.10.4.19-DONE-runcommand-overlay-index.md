## Purpose

- **🔷** **`RunCommandV2`** must **not** call **`fetch_files`** before bubble exec.
- **🔷** **`Scan`** keeps classify + **`handle_*`**; **`FileVerification`** keeps **`created`** / **`modified`** / **`removed`** / **`finish`**.
- **🔷** Rename **`indexed_file_type`** → **`has_file`**; make it **async** with lazy **`fetch_file`** on cache miss — preserves **`libocbwrap`** boundary ([`2.10.4.17`](2.10.4.17-DONE-libocbwrap-sandbox-library.md)).

---

## Mental model

**Shipping `Sandbox/Scan`** — index lookup inline via **`all_files.get`**, then classify.

**V2 split ([`2.10.4.17`](2.10.4.17-DONE-libocbwrap-sandbox-library.md)):** **`Scan`** (generic, no **`libocfiles`**) asks **`FileVerification.has_file`** — the seam so impls can read the index without leaking client types into **`libocbwrap`**.

| Impl | **`has_file`** source |
|------|----------------------|
| **`OLLMtools.FileVerification`** | cache miss → **`yield fetch_file`**; **`base_type`** → **`GLib.FileType`** |
| **`OLLMfilesd.FileVerification`** (future) | **`project_files.all_files`** (no RPC) |

**Bug today:** sync read on empty **`file_cache`** + whole-project **`fetch_files()`** hack in **`Request.vala`**. **Fix:** async **`has_file`** + delete hack.

---

## Agreed direction (🔷)

1. **Remove** unfiltered **`fetch_files()`** from **`RunCommandV2/Request.vala`**.
2. **Extend **`Folder.fetch_files`** with **`paths[]`** + **`metadata_only`** (path-filter batch).
3. **Rename **`indexed_file_type`** → **`has_file`**; make **async**; **`OLLMtools`**: lazy **`fetch_file`**, classify via **`base_type`**.
4. **`Scan`**: **`yield verification.has_file(...)`** — **`handle_*`** and apply hooks unchanged.
5. **🚫** no **`project`** / **`manager`** on **`Bubble`** / **`Scan`** — keeps sandbox layer generic.

### ✅ 2. `ollmfilesd/Folder.vala` — `rpc_fetch_files`: path-filter via `all_files`

**Why:** Path-filter needs dirs / aliases / files from **`all_files`**, not **`cached_search`**.

**Where:** `rpc_fetch_files` handler, after **`project == null`** return, before **`p.limit`**.

**Depends on:** §1.

**Keep** — handler start through **`project == null`** block (unchanged).

**Add** — path-filter branch before dropdown pagination:

```vala
				if (p.paths.length > 0) {
					var list = new Gee.ArrayList<GLib.Object>();
					foreach (var lookup_path in p.paths) {
						if (
							!project.project_files.all_files.has_key(
								lookup_path
							)
						) {
							continue;
						}
						var source = project.project_files.all_files.get(
							lookup_path
						);
						var row = new File(this.manager);
						row.copy_from(
							source,
							{
								"manager",
								"buffer",
								"parent",
								"last_modified"
							}
						);
						list.add(row);
					}
					request.reply(new OLLMrpc.Response() {
						id = request.id,
						result = list,
						result_type = "File",
						is_array = true,
						msg = list.size.to_string()
					});
					return;
				}
```

**Keep** — **`p.limit`**, **`cached_search`**, rest of handler (unchanged).

### ✅ 6. `liboctools/FileVerification.vala` — lazy `fetch_file` + `base_type`

**Why:** On cache miss, **`yield fetch_file`** (metadata on wire).

**Where:** rename override; async body.

**Depends on:** §5.

**Keep** — **`created`**, **`modified`**, **`removed`**, **`finish`**.

#### Replace — `indexed_file_type` override with `has_file`

**Remove**

```vala
		public override GLib.FileType indexed_file_type(string real_path)
		{
			if (!this.manager.file_cache.has_key(real_path)) {
				return GLib.FileType.UNKNOWN;
			}
			var item = this.manager.file_cache.get(real_path);
			if (item is OLLMfiles.Folder) {
				return GLib.FileType.DIRECTORY;
			}
			if (item.is_alias) {
				return GLib.FileType.SYMBOLIC_LINK;
			}
			return GLib.FileType.REGULAR;
		}
```

**Replace with**

```vala
		public override async GLib.FileType has_file(string real_path)
		{
			if (this.project == null) {
				return GLib.FileType.UNKNOWN;
			}
			if (!this.manager.file_cache.has_key(real_path)) {
				yield this.project.fetch_file(real_path);
			}
			if (!this.manager.file_cache.has_key(real_path)) {
				return GLib.FileType.UNKNOWN;
			}
			var item = this.manager.file_cache.get(real_path);
			if (item.base_type == "d") {
				return GLib.FileType.DIRECTORY;
			}
			if (item.base_type == "fa") {
				return GLib.FileType.SYMBOLIC_LINK;
			}
			return GLib.FileType.REGULAR;
		}
```
