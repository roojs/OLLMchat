## Purpose

- **🔷** Wire the **V2 client** and **Approvals UI** to daemon **`FileHistory.approve`** / **`FileHistory.revert`** and **`Folder.fetch_pending_approvals` → `FileWithHistory[]`** (defined in **2.10.4.25**).
- **🔷** Split from **2.10.4.25** so the parent plan stays **daemon-only**; this plan holds client Meson, **`ReviewFiles`**, **`Approvals`**, and compile/design gaps found during a reverted implementation attempt.
- **ℹ️** [`2.10.4.24-ACTIVE-v2-build-gaps.md`](../2.10.4.24-ACTIVE-v2-build-gaps.md) **B-7d** ✅ — Approvals uses **`FileHistory.approve`** / **`revert`** RPC (§8).
- **🔷 G-2 — Reject end-to-end:** daemon **`FileHistory.revert`** restores on disk + replies **`File`** row (**🚫** no content in **`msg`**). Client **`V2/FileHistory.revert()`** merges the wire row, then **`yield cached.read()`** reloads the editor buffer. **Then** **`review_files.refresh()`** updates the popover. **🚫** Approvals must **not** call **`File.read`** or decode revert **`response.msg`** itself — that lives in §2.
- **🔷 G-7:** One **`ProjectManager.review_files`** for the window (**created once**, **🚫** not replaced per project). **`activate_project`** sets **`active_project`**, then **`refresh()`** clears and reloads from server. **`ReviewFiles.refreshed`** → **`update_button_visibility`**. Approvals **🚫** no local **`review_files`** field — use **`project_manager.review_files`** (same as shipping V1 **`project.project_files.review_files`**).

Each subsection below is **self-contained**: decision/status and code fences live together — **🚫** no separate “gaps” section elsewhere in the doc.

**Hunk labels:** every fence group is tagged **`ClassName.method()`**, **`ClassName()`** (ctor), or **`ClassName — field`**. Apply only the hunks for that method/region.

### 2. `libocfiles/V2/FileHistory.vala` — **Add** **✅**

**G-2 ✅ (2026-07-01) — reject contract:**

1. **Server:** **`FileHistory.revert`** RPC — restore file on disk, update DB, reply updated **`File`** row + **`msg = "ok"`** (**🚫** no file content on wire).
2. **Client buffer:** merge **`response.result`** onto cached **`File`**, then **`yield cached.read()`** — same path as any external disk change; reloads editor text via **`File.read`** RPC.
3. **List UI:** caller (**§8** **`on_reject_clicked`**) **`yield review_files.refresh()`** after **`hist.revert.end`** — popover + buttons via **`refreshed`**.

**Approve** has no buffer step — only RPC + list refresh.

**Add**

```vala
namespace OLLMfiles
{
	/**
	 * V2 approve/revert RPC handle ({@code file_history.id} + path).
	 */
	public class FileHistory : Object
	{
		public weak ProjectManager manager { get; set; }
		public int64 id { get; set; default = 0; }
		public string path { get; set; default = ""; }

		public async void approve()
		{
			yield this.manager.rpc.call(new OLLMrpc.Request() {
				method = "FileHistory.approve",
				param = new OLLMfilesd.FileParams() {
					path = this.path,
					id = this.id
				}
			});
		}

		/**
		 * Revert on daemon, then refresh buffer via {@link File.read}.
		 */
		public async void revert()
		{
			var response = yield this.manager.rpc.call(
				new OLLMrpc.Request() {
					method = "FileHistory.revert",
					param = new OLLMfilesd.FileParams() {
						path = this.path,
						id = this.id
					}
				}
			);
			if (response.error != null) {
				return;
			}
			var cached = this.manager.file_cache.get(this.path) as File;
			if (cached == null && response.result is File) {
				this.manager.file_cache.set(
					this.path,
					(File) response.result
				);
				cached = this.manager.file_cache.get(this.path) as File;
			}
			if (cached == null && this.manager.active_project != null) {
				cached = yield this.manager.active_project.fetch_file(
					this.path
				);
			}
			if (cached == null) {
				return;
			}
			cached.manager = this.manager;
			if (response.result is File) {
				cached.copy_from((File) response.result, {
					"manager",
					"buffer",
					"parent"
				});
			}
			yield cached.read();
		}
	}
}
```

**ℹ️** **`fetch_file`** fallback covers cache miss when the editor has not opened the path yet (**G-4 Option B** partial).
