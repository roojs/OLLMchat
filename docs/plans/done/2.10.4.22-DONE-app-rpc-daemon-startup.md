## V2 Window startup

**ℹ️** **`load_projects_from_db`** is legacy naming. On V2 the app has **🚫** no SQLite — the daemon owns **`files.sqlite`**.

**V2 `initialize_client` order (after §MOD-WIN-1):**

1. **`new ProjectManager()`** — local UI state + default **`OLLMrpc.Client`**; **🚫** no DB ctor, **🚫** no async work
2. **`yield project_manager.rpc.connect(hello)`** — boot **`ollmfilesd`** if needed, socket/TCP, **`Daemon.hello`**
3. If **`!rpc.connected`** — banner + **return** (**🚫** do not call **`load_projects_from_db`** or register tools)
4. **`yield project_manager.load_projects_from_db()`** — **`ProjectManager.load_projects_from_db`** RPC only
5. Rest of init (tools, agents, UI) — assumes RPC is up

**Shipping contrast:** migrate/cleanup against in-process SQLite, then **`load_projects_from_db`** hits local DB — no **`ollmfilesd`**, no **`rpc.connect`**.

---

## Why Window must be V2 (not in-place edit)

Shipping **`initialize_client`** ties **`ProjectManager`** to **`SQ.Database`** and local file-index work:

- **`new ProjectManager(SQ.Database…)`** — SQLite lives in the app process
- **`project_manager.db`**, **`ProjectMigrate`**, **`FileHistory.cleanup_old_backups`**
- **`git_provider`** on PM (shipping only)
- **`BackgroundScan`** wired to **`project_manager.db`** and in-process index

V2 **`libocfiles/ProjectManager`** has **🚫** no **`db`**, **🚫** no SQLite ctor — **`files.sqlite`** and migrate/cleanup live on **`ollmfilesd`**.

**🔷** You cannot reshape shipping **`Window.vala` in place** and keep **`v2testing=false`** working. Parallel **`ollmapp/V2/Window.vala`** + Meson source pick ([`2.10.4.21`](2.10.4.21-DONE-meson-v2-cutover.md) §9) matches **`libocfiles/V2/`** and the rest of the cutover.

### MOD-WIN-1. `ollmapp/V2/Window.vala` — `initialize_client` (strip DB, boot RPC)

**Type:** **MODIFY** (V2 copy)

**Why:** V2 PM has no SQLite. Async RPC boot cannot live in PM ctor — Window **`yield`s `rpc.connect`** before **`load_projects_from_db`**.

**Where:** **`// Create ProjectManager first`** through **`FileHistory.cleanup_old_backups`**.

**Depends on:** §NEW-1.

#### Remove — shipping SQLite + migrate + cleanup block

```vala
			// Create ProjectManager first to share with tools
			this.project_manager = new OLLMfiles.ProjectManager(
				new SQ.Database(GLib.Path.build_filename(this.app.data_dir, "files.sqlite"))
			);
			this.project_manager.buffer_provider = new OLLMcoder.BufferProvider();
			this.project_manager.git_provider = new OLLMcoder.GitProvider();
			
			// Run migration if database file doesn't exist (e.g. first run or new data dir)
			if (this.project_manager.db != null) {
				var db_file = GLib.File.new_for_path(this.project_manager.db.filename);
				if (!db_file.query_exists()) {
					var migrator = new OLLMfiles.ProjectMigrate(this.project_manager);
					yield migrator.migrate_all();
				}
				// Load project list from DB so session restore and UI can resolve project_path
				yield this.project_manager.load_projects_from_db();
			}
			// Cleanup old backup files and database records on startup
			if (this.project_manager.db != null) {
				var max_days = config.files_max_deleted_days > 0 ? config.files_max_deleted_days : 30;
				OLLMfiles.FileHistory.cleanup_old_backups.begin(this.project_manager.db, max_days);
			}
```

#### Replace with — boot RPC, then load project list

```vala
			this.project_manager = new OLLMfiles.ProjectManager();
			this.project_manager.buffer_provider = new OLLMcoder.BufferProvider();

			var hello = new OLLMrpc.Request() {
				method = "Daemon.hello",
				param = new OLLMfilesd.DaemonParams() {
					protocol = 1,
					client = "ollmchat"
				}
			};
			if (!yield this.project_manager.rpc.connect(hello)) {
				var msg = this.project_manager.rpc.connect_error;
				if (msg == "") {
					msg = "could not start or reach the filesystem daemon (ollmfilesd)";
				}
				GLib.warning("ollmchat: %s", msg);
				this.tool_error_banner.title = "Filesystem daemon: " + msg;
				this.tool_error_banner.revealed = true;
				return;
			}

			yield this.project_manager.load_projects_from_db();
```

				// When project changes, scan all files in the project
				// scanProject handles null project internally

					this.background_scan.scanProject(project);
				});
			} else {
				//GLib.debug("Background semantic search indexing disabled via --disable-indexer");
			}
```

```
