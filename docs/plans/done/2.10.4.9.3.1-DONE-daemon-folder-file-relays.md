### 1. `ollmfilesd/ProjectManager.vala` — `project_root`

**Why:** Lookup is manager state (`active_project`, `project_map`) — not a property of any single `Folder` tree node. **Not** `Folder.find_project_root()` (parent-chain walk).

**Where:** `ProjectManager` class, immediately before `get_folder_at_path`.

**Depends on:** none.

#### Add

```vala
		/**
		 * Loaded project root row by absolute path.
		 *
		 * Checks {@link active_project} before {@link ProjectList.project_map}.
		 * Does not query the database — use {@link get_folder_at_path} for any
		 * folder path including subfolders and DB-only rows.
		 *
		 * @param project_path Project root absolute path
		 * @return The project folder, or null if not loaded in memory
		 */
		public Folder? project_root(string project_path)
		{
			if (project_path == "") {
				return null;
			}
			if (this.active_project != null
				&& this.active_project.path == project_path) {
				return this.active_project;
			}
			foreach (var entry in this.projects.project_map.entries) {
				if (entry.value.path == project_path && entry.value.is_project) {
					return entry.value;
				}
			}
			return null;
		}
```

### 2. `ollmfilesd/File.vala` — `File.fetch` relay

**Why:** Backs client `Folder.fetch_file` ([`2.10.4.9.1`](2.10.4.9.1-DONE-design-project-files-and-client-api.md) §4).

**Where:** `File` class — signal list and `construct` block (mirror existing `rpc_read`).

**Depends on:** §1 `project_root`; parent §1 `FileParams.project_path`.

#### Add — `rpc_fetch` signal

Placement: after `rpc_read` signal declaration.

```vala
		public signal void rpc_fetch(OLLMrpc.Request request);
```

#### Add — `rpc_fetch` handler in `construct`

Placement: immediately after the closing `});` of the `rpc_read.connect` handler.

```vala
			this.rpc_fetch.connect((request) => {
				var p = (FileParams) request.param;
				var project = this.manager.project_root(p.project_path);
				if (project == null) {
					request.reply(new OLLMrpc.Response() {
						id = request.id,
						result_type = "File",
						msg = "project not found"
					});
					return;
				}
				var project_file = project.project_files.child_map.get(p.path);
				if (project_file == null) {
					request.reply(new OLLMrpc.Response() {
						id = request.id,
						result_type = "File",
						msg = "file not found"
					});
					return;
				}
				var source = project_file.file;
				var row = new File(this.manager) {
					last_modified = source.mtime_on_disk()
				};
				row.copy_from(
					source,
					{"manager", "buffer", "parent", "last_modified"}
				);
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					result = row,
					result_type = "File"
				});
			});
```

```vala
		public signal void rpc_contains_folder(OLLMrpc.Request request);
		public signal void rpc_fetch_file_list(OLLMrpc.Request request);

```

#### Add — `construct` block with handlers

Placement: immediately after the four signal declarations above.

```vala
		construct
		{
			this.rpc_fetch.connect((request) => {
				var path = ((FolderParams) request.param).path;
				var folder = this.manager.get_folder_at_path(path);
				if (folder == null) {
					request.reply(new OLLMrpc.Response() {
						id = request.id,
						result_type = "Folder",
						msg = "folder not found"
					});
					return;
				}
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					result = folder,
					result_type = "Folder"
				});
			});
			this.rpc_contains_folder.connect((request) => {
				var p = (FolderParams) request.param;
				var project = this.manager.project_root(p.project_path);
				if (project == null) {
					request.reply(new OLLMrpc.Response() {
						id = request.id,
						msg = "project not found"
					});
					return;
				}
				if (!project.project_files.folder_map.has_key(p.path)) {
					request.reply(new OLLMrpc.Response() {
						id = request.id,
						msg = "directory not in project"
					});
					return;
				}
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					msg = "true"
				});
			});
			this.rpc_fetch_file_list.connect((request) => {
				var path = ((FolderParams) request.param).path;
				var project = this.manager.project_root(path);
				if (project == null) {
					request.reply(new OLLMrpc.Response() {
						id = request.id,
						result = new Gee.ArrayList<GLib.Object>(),
						result_type = "File",
						is_array = true,
						msg = "project not found"
					});
					return;
				}
				var list = new Gee.ArrayList<GLib.Object>();
				foreach (var project_file in project.project_files) {
					list.add(project_file.file);
				}
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					result = list,
					result_type = "File",
					is_array = true
				});
			});
			this.rpc_fetch_pending_approvals.connect((request) => {
				var path = ((FolderParams) request.param).path;
				var project = this.manager.project_root(path);
				if (project == null) {
					request.reply(new OLLMrpc.Response() {
						id = request.id,
						result = new Gee.ArrayList<GLib.Object>(),
						result_type = "File",
						is_array = true,
						msg = "project not found"
					});
					return;
				}
				project.project_files.review_files.refresh();
				var list = new Gee.ArrayList<GLib.Object>();
				for (uint i = 0; i < project.project_files.review_files.get_n_items(); i++) {
					var pf = project.project_files.review_files.get_item(i)
						as ProjectFile;
					if (pf != null) {
						list.add(pf.file);
					}
				}
				request.reply(new OLLMrpc.Response() {
					id = request.id,
					result = list,
					result_type = "File",
					is_array = true
				});
			});
		}
```

## Notes (not implementation instructions)

- **🔷** `project_root` on `ProjectManager` — `active_project` before `project_map`; relays call `this.manager.project_root(…)`.
- **🔷** `File.fetch`: negative guard + `return` when project or `child_map` miss; single success reply path.
- **🔷** Wire row `last_modified`: GObject property initializer on `new File(…) { last_modified = … }`; exclude `last_modified` from `copy_from`.
- **🔷** `Folder.*` list handlers: if `project_root` is null, reply empty result immediately.
- **ℹ️** `OLLMrpc.Response` has a `msg` string field (`libocrpc/Response.vala`) — used for acks (`"ok"`), bools (`"true"`), and **not-found** text when `result` is null or empty.
- **🔷** Not-found replies set `msg` (no `error` object — soft miss, not JSON-RPC fault):

| Wire method | `msg` when missing |
| ----------- | ------------------ |
| `File.fetch` | `"project not found"` / `"file not found"` |
| `Folder.fetch` | `"folder not found"` |
| `Folder.contains_folder` | `"project not found"` / `"directory not in project"`; success `"true"` |
| `Folder.fetch_file_list` | `"project not found"` (+ empty `File[]`) |
| `Folder.fetch_pending_approvals` | `"project not found"` (+ empty `File[]`) |

- **ℹ️** Client [`2.10.4.9.1`](2.10.4.9.1-DONE-design-project-files-and-client-api.md) §4 `contains_folder` still treats only `msg == "true"` as success; other `msg` values → `false`. `fetch_file` / `fetch_folder` currently return null on missing `result` — `msg` is for logs / `RpcClient.failed` wiring later, not a separate client fence in this plan.
- **ℹ️** Internal `ProjectManager.get_folder_at_path` stays on the daemon for `Folder.fetch`; **🚫** no `ProjectManager.get_folder_at_path` RPC relay.
