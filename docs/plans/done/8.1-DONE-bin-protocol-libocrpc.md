## Purpose

Implement the binary payload codec described in [`bin-rpc-protocol.md`](../bin-rpc-protocol.md) in **`libocrpc`**, namespace **`OLLMrpc.Bin`**.

- **🔷** Types: **`Stream`**, **`Serializable`** — **🚫** no separate **`Registry`** or **`Object`** base class.
- **🔷** **`Stream`** — per-connection I/O + JIT wire-name tokens + **`write`** / **`parse`**; **process-wide static** alias → GType catalog via **`Stream.register`** (same call sites as **`rpc_register()`**).
- **🔷** **`Connection`** / **`Client`** own **`bin`** for the channel lifetime — **🚫** not one stream per message.
- **🔷** Primary API: **`write(Serializable)`** outbound, **`parse()`** inbound — root type from wire only.
- **🔷** Default **`bin_write_prop`** / **`bin_read_prop`** — scalars, **`string[]`** (`GStrv`), and nested **`Serializable`** objects.
- **🔷** Override **`bin_write_prop`** / **`bin_read_prop`** — **`Gee.ArrayList`**, other **`T[]`**, **`uint8[]`**, graph/runtime omits — see [Gee.ArrayList — two paths](#geearraylist--two-paths) at top.
- **✔️ 🔷** Phase 1: codec in **`libocrpc`** — **done** (see **Phase 1**).
- **✔️ 🔷** Phase 2: audit the **two live RPC sites**; port wire types from JSON workarounds to **`Bin.Serializable`**; consolidate registration — **done**
- **✔️ 🔷** Phase 3: attach **`bin`** to **`Connection`** / **`Client`**; hook read loops — **done**
- **✅ 🔷** Phase 4: remove JSON dual stack — [audit](#phase-4--json-removal-audit) — **done**
- **🚫** Domain types in **`libocrpc`** — sample types only in **`tests/rpc/bin-test.vala`**.

Proposed Vala uses **early return** and **inverted guards** — **🚫** no **`else`** where the happy path can return first ([`reducing-nesting`](../coding-standards.md#reducing-nesting)).

## Bin integration model

### Ownership

```text
Process-wide (static)
  └── Bin.Stream.register(alias, gtype)   ← alias → local GType; same call sites as OLLMrpc.register / rpc_register()

Connection (server) / Client (client)
  └── Bin.Stream bin                        ← in/out streams + JIT wire-name tokens + write/parse
```

- **🔷** Create **`bin`** once when the channel opens; clear on disconnect.
- **🔷** **Send:** caller invokes **`bin.write(obj)`** when outbound RPC carries a GObject body.
- **🔷** **Receive:** RPC read loop calls **`bin.parse()`** on the hot path (Phase 3).

### Type registration

- **🔷** **`Bin.Stream.register(alias, gtype)`** — **static**, process-wide (alias → local GType for decode).
- **🔷** Each wire type's existing **`rpc_register()`** also calls **`Bin.Stream.register`** — **same places** as **`OLLMrpc.register`** today:
  - **`libocfiles/V2/ProjectManager`** constructor (domain types)
  - **`OLLMrpc.Client`** static construct (**`Error`**, **`Notification`**)
  - **`ollmfilesd`** startup — add **`rpc_register()`** on server wire types and call before listen (server has no **`OLLMrpc.register`** today)
- **🔷** Client and server register the **same wire alias strings**; each maps to its **own** local **`typeof`**.
- **🚫** No **`register_wire_types(Bin.Stream bin)`** — no per-connection type catalog.
- **🚫** No type registration on **`connect()`** / **`start()`** — types are registered before the channel opens.

## Vala API (`OLLMrpc.Bin`)

### `Stream` (public)

| Method | Role |
| ------ | ---- |
| **`static register(alias, gtype)`** | Process-wide wire alias → GType (from each type's **`rpc_register()`**) |
| **`write(Serializable obj)`** | Outbound root object |
| **`parse()`** | Inbound root object; type from wire |

Also used by **`Serializable`**: **`write_tag`** / **`read_tag`**, **`write_gtype`** / **`parse_object`**.

### `Serializable` (override points, `bin_` prefix)

| Method | Role |
| ------ | ---- |
| **`bin_write`** | Walk properties, write **`TOKEN_END`** |
| **`bin_read`** | **`bin_pre`**, read tags until **`TOKEN_END`**, **`bin_post`** |
| **`bin_pre`** | Inbound hook before property decode (default empty) |
| **`bin_post`** | Inbound hook after property decode (default empty) |
| **`bin_write_prop`** | Per-prop encode — default: scalars, **`string[]`**, nested **`Serializable`**; override for other lists / omits |
| **`bin_read_prop`** | Per-prop decode — default: scalars, **`string[]`**, nested **`Serializable`**; override for other lists / omits |

- **🚫** No private **`bin_*`** helpers on the interface.
- **🚫** No nested **`switch`** in prop encode/decode.
- **🚫** Varint / LEB128 / zigzag.
- **🔷** Prop walk uses **`get_property`** / **`set_property`** / **`find_property`**.

Wire layout details: [`bin-rpc-protocol.md`](../bin-rpc-protocol.md).

### ✔️ 1. `libocrpc/Bin/Stream.vala` — static type registration

**✔️ Why:** Alias → GType is **process-wide** (like **`OLLMrpc.register`**), not per-connection. Per-connection **`Stream`** keeps only JIT wire-name tokens (**`names`**, **`name_to_token`**).

**✔️ Where:** Move **`alias_to_gtype`** / **`gtype_to_alias`** off the instance; make **`register`** static.

**✔️ Depends on:** Phase 1 **`Stream`** (instance **`register`** — fixed in Phase 2).

#### ✔️ Remove — instance maps + instance `register`

```vala
		private Gee.HashMap<string, GLib.Type> alias_to_gtype =
			new Gee.HashMap<string, GLib.Type> ();
		private Gee.HashMap<GLib.Type, string> gtype_to_alias =
			new Gee.HashMap<GLib.Type, string> ();
```

```vala
		public void register (string alias, GLib.Type gtype) throws GLib.Error
		{
			if (this.alias_to_gtype.has_key (alias)) {
				throw new Error.REGISTRATION (
					"duplicate register of alias '%s'",
					alias
				);
			}

			this.alias_to_gtype.set (alias, gtype);
			this.gtype_to_alias.set (gtype, alias);
		}
```

#### ✔️ Replace with — static maps + static `register`

```vala
		private static Gee.HashMap<string, GLib.Type>? alias_to_gtype;
		private static Gee.HashMap<GLib.Type, string>? gtype_to_alias;

		public static void register(string alias, GLib.Type gtype) throws GLib.Error
		{
			if (alias_to_gtype == null) {
				alias_to_gtype = new Gee.HashMap<string, GLib.Type>();
				gtype_to_alias = new Gee.HashMap<GLib.Type, string>();
			}
			if (alias_to_gtype.has_key(alias)) {
				throw new Error.REGISTRATION(
					"duplicate register of alias '%s'",
					alias
				);
			}
			alias_to_gtype.set(alias, gtype);
			gtype_to_alias.set(gtype, alias);
		}
```

**ℹ️** Instance methods **`write_gtype`**, **`parse_object`**, **`read_reg_gtype`** read the static maps (not **`this.alias_to_gtype`**).

**ℹ️** Update **`tests/rpc/bin-test.vala`** — **`Stream.register(...)`** static, not **`write_bin.register`**.
