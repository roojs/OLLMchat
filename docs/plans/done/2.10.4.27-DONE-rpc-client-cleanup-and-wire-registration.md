**Why:** Client registers **`typeof(Daemon)`** as the **`"Daemon"`** receiver for **`Json.gobject_deserialize`**. Must be **concrete** вЂ” not interface, not abstract.

**Depends on:** nothing

#### Add вЂ” `libocrpc/Daemon.vala`

```vala

{
	/**
	 * Wire type for {@code Daemon.hello} ({@code result_type = "Daemon"}).
	 * Client deserializes into this class. Server type is
	 * {@code ollmfilesd/Daemon.vala} (unchanged).

		}

		public string server { get; set; default = ""; }
		public bool ready { get; set; default = false; }
	}

}
```
