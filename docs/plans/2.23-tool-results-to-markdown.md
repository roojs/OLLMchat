# Tool results as markdown in skills agent only

**Status:** Plan  
**Scope:** BaseTool; skills agent (Task.Tool, Details.executor_precursor); tools used by skills agent.

**Goal:** Skills agent path calls `execute(chat_call, tool_call, true)` (last argument `is_markdown = true`) and uses **header_raw** for tool output. Normal path calls `execute(chat_call, tool_call)` or `execute(..., false)`. Tool output is whatever the tool returns. **No tool overrides that reformat request output** — that is not the point of this plan. Wrapper provides "Tool call xxx Result".

**Rejected:** Any design that wraps the raw result in markdown (e.g. `result_to_markdown(result)`).

---

## 1. BaseTool — execute(..., bool is_markdown)

**File:** `libollmchat/Tool/BaseTool.vala`

**1a) Change `execute()`** to add a third parameter `is_markdown = false`; keep the same body (one method only).

**Replace:**

```vala
		public virtual async string execute(Call.Chat chat_call, Response.ToolCall tool_call)
		{
			// Convert parameters Json.Object to Json.Node for deserialization
			var parameters_node = new Json.Node(Json.NodeType.OBJECT);
			parameters_node.set_object(tool_call.function.arguments);
			
			// Check if this is a wrapped tool
			RequestBase? request = null;
			if (this.is_wrapped && this is WrapInterface) {
				// Wrapped tool execution flow
				var wrapped_tool = this as WrapInterface;
				request = wrapped_tool.deserialize_wrapped(parameters_node, this.command_template);
				if (request == null) {
					return "ERROR: Failed to create wrapped tool request object";
				}
				// Mark request as wrapped for tracking
				request.is_wrapped = true;
			} else {
				// Normal tool execution flow
				request = this.deserialize(parameters_node);
				if (request == null) {
					return "ERROR: Failed to create request object";
				}
			}
			
			// Set tool and agent (not in JSON, set after deserialization)
			request.tool = this;
			// Set agent property (from chat_call, set after deserialization)
			request.agent = chat_call.agent;
			
			// request_id is auto-generated via default value in RequestBase
			// Register for monitoring (works for both Agent.Base and dummy agents)
			// Interface methods have default no-op implementations
			request.agent.register_tool_monitoring(request.request_id, request);
			
			return yield request.execute();
		}
```

**With:**

```vala
		/** @param is_markdown When true, called from skills agent path; output is shown with header_raw. No reformatting of request output. */
		public virtual async string execute(Call.Chat chat_call, Response.ToolCall tool_call, bool is_markdown = false)
		{
			var parameters_node = new Json.Node(Json.NodeType.OBJECT);
			parameters_node.set_object(tool_call.function.arguments);
			RequestBase? request = null;
			if (this.is_wrapped && this is WrapInterface) {
				var wrapped_tool = this as WrapInterface;
				request = wrapped_tool.deserialize_wrapped(parameters_node, this.command_template);
				if (request == null) {
					return "ERROR: Failed to create wrapped tool request object";
				}
				request.is_wrapped = true;
			} else {
				request = this.deserialize(parameters_node);
				if (request == null) {
					return "ERROR: Failed to create request object";
				}
			}
			request.tool = this;
			request.agent = chat_call.agent;
			request.agent.register_tool_monitoring(request.request_id, request);
			return yield request.execute();
		}
```

## 5. GoogleSearch ResultItem and Result

**File:** `liboctools/GoogleSearch/Request.vala`

**6a) ResultItem.to_markdown** — Replace:

```vala
		public string to_markdown()
		{
			return "### [" + this.title + "](" + this.link + ")\n" + this.snippet + "\n";
		}
```

**With:**

```vala
		public string to_markdown(int index = 0)
		{
			var head = (index > 0) ? "#### Result (" + index.to_string() + ")\n\n" : "#### " + this.title + "\n\n";
			return head
				+ "- **Link** [" + this.title + "](" + this.link + ")\n\n"
				+ "- **Description**\n\n```markdown\n" + this.snippet + "\n```\n";
		}
```

**6b) Result.to_markdown** — Replace:

```vala
			foreach (var item in this.items) {
				result += item.to_markdown();
			}
```

**With:**

```vala
			var i = 1;
			foreach (var item in this.items) {
				result += item.to_markdown(i);
				i++;
			}
```

## 6. read_file — no tool-specific changes

Summary format is identical for the tools that need it; no ReadFile-specific properties or to_markdown() permitted. **read_file** uses the default `execute()`; when skills agent calls `execute(..., true)` the same return is used. No changes to ReadFile Request or ReadFile Tool.

## Order of work

1. BaseTool: change `execute(chat_call, tool_call)` to `execute(chat_call, tool_call, bool is_markdown = false)`; one method only, no run_request.
2. Task.Tool: call `execute(chat_call, tool_call, true)` for skills agent.
3. Details: header_raw for tool output.
4. SearchResult: `### Result` → `#### Result`.
5. GoogleSearch: ResultItem.to_markdown(int), Result.to_markdown() loop with index.
6. read_file: no tool-specific changes (uses default execute(..., is_markdown)).

## Out of scope

- **edit_mode:** Stream-based; no single execute() return.
- **result_to_markdown / wrapping:** Explicitly out of scope.
