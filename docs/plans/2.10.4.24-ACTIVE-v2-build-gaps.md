### B-2 — RPC hydrate: `manager` assign after `register()`

**Symptoms:**

**Files:** **`V2/Folder.vala`**, **`V2/ProjectManager.vala`**, **`V2/ReviewFiles.vala`**

**✔️** Applied in tree.

**🚫** Do not add a second PM reference or wrapper type just to avoid **`set`**.

### B-3 — `DeleteManager.remove`: `switch (get_type())` case labels

**Symptoms:**

```
V2/DeleteManager.vala:69 … Expression must be constant
V2/DeleteManager.vala:81 … Expression must be constant
```

**Cause:** **`switch (filebase.get_type())`** with **`case typeof(File):`** — case labels must be constant; **`typeof(...)`** is not.

**🔷 Resolution:** keep **`switch`**, dispatch on **`filebase.base_type`** (**`"f"`**, **`"d"`**, **`"da"`**, …) — string literals are valid case labels.

**Replace with:**

```vala
			switch (filebase.base_type) {
				case "f":
					var file = (File) filebase;
					// … yield file.delete(), cache cleanup …
					break;
				case "d":
				case "da":
					throw new IOError.NOT_SUPPORTED(
						"Folder deletion is daemon-side only"
					);
				default:
					throw new IOError.NOT_SUPPORTED(
						"Unsupported type for client delete (base_type=%s)".printf(
							filebase.base_type
						)
					);
			}
```

**✔️** Applied in tree.

**🚫** Do not use **`switch (get_type())`** on the client.

**ℹ️** Values from **`V2/FileBase.vala`**: **`"f"`** File, **`"d"`** Folder, **`"fa"`** / **`"da"`** aliases, **`"pf"`** project-file row.

### B-4 — `Folder.fetch_files`: `out int total` in `async` method

**Symptoms:**

```
Synchronous out-parameters are not supported in async methods
only `null' is allowed as default value for out parameters
```

**Cause:** Vala allows **`async` return values**, not **`out`** parameters. The RPC wire is fine — each **`Folder.fetch_files`** response already has **`result`** (file page) and **`msg`** (total count string). The broken code copied a sync-style **`out int total`** into an **`async`** method.

**Files:**

| File | Method |
|------|--------|
| **`libocfiles/V2/Folder.vala`** | **`fetch_files`** |
| **`libocfiles/V2/ProjectFiles.vala`** | **`refresh`**, **`load_more`** |

**🔷 Resolution (user confirmed):** **`fetch_files` returns `OLLMrpc.Response`**. Callers read **`response.msg`** and **`response.result`**.

**✔️** Applied in tree.

**🚫 Rejected:**

| Approach | Why |
|----------|-----|
| **`out int total`** on **`async`** | Vala compile error |
| **`last_fetch_total`** property on **`Folder`** | Side-channel state; response already carries **`msg`** |
| **`FetchFilesPage` struct** | Invented wrapper; response already has **`result`** + **`msg`** |

**ℹ️** [`2.10.4.9` B-10](done/2.10.4.9-DONE-v2-caller-cutover.md) wire unchanged.

#### `Folder.vala` — `fetch_files`

**Before** (broken — **`out int total`** on **`async`**):

```vala
		/**
		 * Fetch a page of file rows for this project (file dropdown).
		 *
		 * Returns daemon metadata rows; the caller builds its own list model.
		 * Total match count (before pagination) is in {@code Response.msg} on the
		 * wire; exposed here as {@code total}.
		 *
		 * @param offset Rows to skip (default 0)
		 * @param limit Page size (default 50)
		 * @param query Dropdown filter (default browse all)
		 * @param total Total matching rows (output)
		 * @return Files in this page for list UI
		 */
		public async Gee.ArrayList<File> fetch_files(
			int offset = 0,
			int limit = 50,
			string query = "",
			out int total = 0,
			string[] paths = {},
			bool metadata_only = false
		)
		{
			var response = yield this.manager.rpc.call(new OLLMrpc.Request() {
				method = "Folder.fetch_files",
				param = new OLLMfilesd.FolderParams() {
					path = this.path,
					offset = offset,
					limit = limit,
					query = query,
					paths = paths,
					metadata_only = metadata_only
				}
			});
			total = 0;
			if (response.error != null) {
				return new Gee.ArrayList<File>();
			}
			if (response.msg != "") {
				total = int.parse(response.msg);
			}
			var files = (Gee.ArrayList<File>) response.result;
			foreach (var file in files) {
				file.manager = this.manager;
				this.manager.file_cache.set(file.path, file);
			}
			return files;
		}
```

**After:**

```vala
		/**
		 * Fetch a page of file rows for this project (file dropdown).
		 *
		 * Returns the daemon RPC response. Caller reads {@code result} (file page)
		 * and {@code msg} (total match count before pagination).
		 *
		 * @param offset Rows to skip (default 0)
		 * @param limit Page size (default 50)
		 * @param query Dropdown filter (default browse all)
		 * @return {@link OLLMrpc.Response} — {@code result} = file page, {@code msg} = total
		 */
		public async OLLMrpc.Response fetch_files(
			int offset = 0,
			int limit = 50,
			string query = "",
			string[] paths = {},
			bool metadata_only = false
		)
		{
			var response = yield this.manager.rpc.call(new OLLMrpc.Request() {
				method = "Folder.fetch_files",
				param = new OLLMfilesd.FolderParams() {
					path = this.path,
					offset = offset,
					limit = limit,
					query = query,
					paths = paths,
					metadata_only = metadata_only
				}
			});
			if (response.error == null && response.result != null) {
				var files = (Gee.ArrayList<File>) response.result;
				foreach (var file in files) {
					file.manager = this.manager;
					this.manager.file_cache.set(file.path, file);
				}
			}
			return response;
		}
```
