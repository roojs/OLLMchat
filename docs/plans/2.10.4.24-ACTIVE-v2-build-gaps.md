### B-3 ‚Äî `DeleteManager.remove`: `switch (get_type())` case labels

**Symptoms:**

```

V2/DeleteManager.vala:81 ‚Ä¶ Expression must be constant
```

**Cause:** **`switch (filebase.get_type())`** with **`case typeof(File):`** ‚Äî case labels must be constant; **`typeof(...)`** is not.

**üî∑ Resolution:** keep **`switch`**, dispatch on **`filebase.base_type`** (**`"f"`**, **`"d"`**, **`"da"`**, ‚Ä¶) ‚Äî string literals are valid case labels.

```vala
					// ‚Ä¶ yield file.delete(), cache cleanup ‚Ä¶
```

**‚úîÔ∏è** Applied in tree.

### B-4 ‚Äî `Folder.fetch_files`: `out int total` in `async` method

**Symptoms:**

```
Synchronous out-parameters are not supported in async methods
only `null' is allowed as default value for out parameters
```

**Cause:** Vala allows **`async` return values**, not **`out`** parameters. The RPC wire is fine ‚Äî each **`Folder.fetch_files`** response already has **`result`** (file page) and **`msg`** (total count string). The broken code copied a sync-style **`out int total`** into an **`async`** method.

**Files:**

| File | Method |
|------|--------|
| **`libocfiles/V2/Folder.vala`** | **`fetch_files`** |
| **`libocfiles/V2/ProjectFiles.vala`** | **`refresh`**, **`load_more`** |

**üî∑ Resolution (user confirmed):** **`fetch_files` returns `OLLMrpc.Response`**. Callers read **`response.msg`** and **`response.result`**.

**‚úîÔ∏è** Applied in tree.

**üö´ Rejected:**

| Approach | Why |
|----------|-----|
| **`out int total`** on **`async`** | Vala compile error |
| **`last_fetch_total`** property on **`Folder`** | Side-channel state; response already carries **`msg`** |
| **`FetchFilesPage` struct** | Invented wrapper; response already has **`result`** + **`msg`** |

**‚ÑπÔ∏è** [`2.10.4.9` B-10](done/2.10.4.9-DONE-v2-caller-cutover.md) wire unchanged.

```vala
		 * Fetch a page of file rows for this project (file dropdown).

			int offset = 0,
			int limit = 50,
			string query = "",
			out int total = 0,
			string[] paths = {},
			bool metadata_only = false
		)
		{
			var response = yield this.manager.rpc.call(new OLLMrpc.Request() {
				method = "Folder.fetch_files",
				param = new OLLMfilesd.FolderParams() {
```
