# 1.20. URGENT - About Dialog

## Overview

Add an about dialog to the OLLMchat application to display application information, version, license, and credits.

## Problem

The application currently lacks an about dialog, which is a standard feature for desktop applications. Users have no way to view:
- Application version
- License information
- Credits/attributions
- Application description

## Goals

1. **About Dialog Widget**: Create an about dialog using GTK/Adwaita components
2. **Application Information**: Display application name, version, description
3. **License Information**: Show license (LGPL-3.0)
4. **Credits**: Display author and any third-party attributions
5. **Accessibility**: Make dialog accessible from application menu or window menu

## Implementation Plan

### Phase 1: Create About Dialog

1. **Choose Dialog Type**
   - Option A: `Gtk.AboutDialog` (standard GTK about dialog)
   - Option B: `Adw.AboutWindow` (Adwaita about window, more modern)
   - **Recommendation**: Use `Adw.AboutWindow` for consistency with Adwaita UI

2. **Dialog Properties**
   - `application_name`: "OLLMchat"
   - `application_icon`: Application icon
   - `version`: Get from build system or version constant
   - `developer_name`: "Alan Knowles"
   - `website`: Project website (if available)
   - `issue_url`: Issue tracker URL (if available)
   - `license_type`: `Gtk.License.LGPL_3_0`
   - `copyright`: Copyright notice
   - `comments`: Application description

### Phase 2: Add to Application Menu

1. **Menu Integration**
   - Add "About" item to application menu (if using `Gtk.Application`)
   - Or add to window menu/header bar menu
   - Connect to show about dialog action

2. **Action Handler**
   - Create action handler that shows the about dialog
   - Use `present()` method to show the dialog

### Phase 3: Version Management

1. **Version Source**
   - Get version from meson.build or version constant
   - Store in application class or constant
   - Display in about dialog

## Files to Modify

- `ollmapp/Window.vala` - Add about dialog and menu integration
- `ollmapp/Application.vala` - Add version constant/property (if needed)
- `meson.build` - Ensure version is available to application

## Example Implementation

```vala
private void show_about_dialog()
{
    var about = new Adw.AboutWindow() {
        application_name = "OLLMchat",
        application_icon = "org.roojs.ollmchat",
        version = "1.0.0", // Get from build system
        developer_name = "Alan Knowles",
        license_type = Gtk.License.LGPL_3_0,
        copyright = "Copyright ¬© 2025 Alan Knowles",
        comments = "A chat interface for local LLM servers",
        website = "https://github.com/...", // If available
    };
    
    about.present(this);
}
```
