### `ProgressRunner` — UI columns (`7.14-task-progress-tree-ui.md`)

| Left (`title`) | Right (`status_str`) |
|----------------|----------------------|
| **Task Creation** + optional **` (k/n)`** (`try_no` / `try_max`, **omit** when first attempt) | **Creating** \| **Retrying** \| grey **✓** |
| **Reviewing Task List** + same suffix rules | **Reviewing** \| **Retrying** \| grey **✓** |

Switch left label with **`bool in_creation`**. **`Runner`** does **not** keep a field for the row. **Two phases → two instances:** **task creation** uses a local **`rp`** and **`add(rp)`** once; **reviewing / iteration** uses a **new** local **`ir`** (**`in_creation = false`**) and **`add(ir)`** when **`run_task_list_iteration`** runs — **do not** reuse **`rp`** for iteration. Each instance’s **`try_no` / `try_max`** drives its own retry loop. When a slice is done, **`status = PhaseEnum.COMPLETED`** on **that** instance so **`clear_pending`** can stop at it.

### `ProgressList.add`

**`add(ProgressRunner r)`** — **`Runner`** builds and configures **`r`** (including try counts), then passes it in; **`ProgressList`** does not allocate the row. Begin with **`clear_pending(false)`** (tail strip until **`COMPLETED`**) so the session row is not stacked on stale pending **`Details`**. Append **`r`**, then emit **one** **`items_changed`** covering both the removals (if any) and the insert. No return value. No **`call_changed`** parameter; **`add`** is always a top-level call from **`Runner`**, not nested inside another **`ProgressList`** mutator.

**Row order** ( **`Runner`** order of calls): completed **`Details`** prefix, then **`add(rp)`** (creation) so the runner sits before **`add_pending`** appends pending **`Details`**. A later **`add(ir)`** (iteration) appends another **`ProgressRunner`** after completed work / **`COMPLETED`** rows as per your ordering; **`clear_pending`** inside **`add`** still applies. Each attempt inside **`run_task_list_iteration`** uses **`clear_pending(false)`** on the **existing** list — separate from **`add(ir)`**.

## `liboccoder/Task/ProgressRunner.vala`

```vala
namespace OLLMcoder.Task
{

public class ProgressRunner : GLib.Object, ProgressItem
{
	public weak OLLMcoder.Skill.Runner runner { get; private set; }

	/** true = Task Creation row; false = Reviewing Task List row */
	public bool in_creation { get; set; default = true; }
	public uint try_no  { get; set; default = 0; }
	public uint try_max { get; set; default = 0; }

	private PhaseEnum status_value = PhaseEnum.NONE;

	public PhaseEnum status {
		get { return this.status_value; }
		set {
			if (this.status_value == value) {
				return;
			}
			this.status_value = value;
			this.notify_property("status_str");
		}
	}

	public string title {
		owned get {
			var base = this.in_creation ? "Task Creation" : "Reviewing Task List";
			if (this.try_no == 0 || this.try_max == 0) {
				return base;
			}
			return "%s (%u/%u)".printf(base, this.try_no + 1, this.try_max);
		}
	}

	public string status_str {
		owned get { return this.status.to_human(); }
	}

	public GLib.ListModel children {
		get;
		default = new GLib.ListStore(typeof (ProgressItem));
	}

	public ProgressRunner(OLLMcoder.Skill.Runner r)
	{
		Object();
		this.runner = r;
	}
}

}
```

### `add` (new)

```vala
	/**
	 * Appends a {@link ProgressRunner} row supplied by {@link Runner}. Caller constructs
	 * {@link r} and sets try fields / phase before calling. Starts with {@link clear_pending}
	 * (no signal); then one {@link GLib.ListModel.items_changed} for tail removals (if any)
	 * plus the new row.
	 */
	public void add(ProgressRunner r)
	{
		var old_size = this.rows.size;
		this.clear_pending(false);
		var after_clear = this.rows.size;
		var removed = old_size - after_clear;
		this.rows.add(r);
		this.items_changed((uint) after_clear, (uint) removed, 1u);
	}
```

### `clear_pending` (same as **7.14.1-phase1-core**)

Strip non-**`COMPLETED`** rows from the tail; **stop at the first `COMPLETED` row**. No type tests, no other branches — same single rule for every **`ProgressItem`**.

```vala
	public void clear_pending(bool call_changed = false)
	{
		var old_size = this.rows.size;
		for (var i = old_size; i > 0; i--) {
			var pi = this.rows.get(this.rows.size - 1);
			if (pi.status == PhaseEnum.COMPLETED) {
				break;
			}
			this.rows.remove_at(this.rows.size - 1);
		}
		var removed = old_size - this.rows.size;
		if (removed == 0 || !call_changed) {
			return;
		}
		this.items_changed((uint) this.rows.size, (uint) removed, 0);
	}
```

### `add_pending` (replace body)

```vala
	public void add_pending(bool call_changed = false)
	{
		var pos = this.rows.size;
		var added = 0;
		foreach (var step in this.runner.pending.steps) {
			foreach (var d in step.children) {
				this.rows.add(d);
				added++;
			}
		}
		if (added == 0 || !call_changed) {
			return;
		}
		this.items_changed((uint) pos, 0, (uint) added);
	}
```

### `add_completed` (replace body)

```vala
	public void add_completed(Step step)
	{
		var n0 = this.rows.size;
		this.clear_pending(false);
		var after_clear = this.rows.size;
		var k = n0 - after_clear;
		var insert_at = this.rows.size;
		for (var ii = 0; ii < this.rows.size; ii++) {
			if (this.rows.get(ii) is ProgressRunner) {
				insert_at = ii;
				break;
			}
		}
		foreach (var d in step.children) {
			this.rows.insert(insert_at, d);
			insert_at++;
		}
		this.add_pending(false);
		var n1 = this.rows.size;
		if (k == 0 && n1 == after_clear) {
			return;
		}
		this.items_changed((uint) after_clear, (uint) k, (uint) (n1 - after_clear));
	}
```

### `clear` (replace body)

		if (old_n == 0) {
			return;
		}
		this.rows.clear();

	}
