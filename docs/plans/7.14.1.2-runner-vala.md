# 7.14.1.2 — `Runner.vala` — `progress` + sync hooks

**Status:** proposed

**Parent:** `docs/plans/7.14.1-progresslist-runner.md`

**Depends on:** **`docs/plans/7.14.1-phase1-core.md`** (**`ProgressList`**).

---

## Purpose

- **`public ProgressList progress { get; private set; }`**
- Constructor: **`this.progress = new ProgressList(this);`**
- Call **`clear_pending(true)`** / **`add_pending(true)`** when **`pending`** resyncs (first task-list parse, **`handle_task_list`**, **`run_task_list_iteration`** success).
- **Optional:** central place to nudge **`Details`** / **`Tool`** **`status`** when session phase changes — if not here, document in **`7.14.1.3`**.
