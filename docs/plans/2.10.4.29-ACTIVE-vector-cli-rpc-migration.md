### Two scans (do not conflate)

| Step | What | Who | Code |
|------|------|-----|------|
| **Filesystem scan** | Walk disk тЖТ update **`filebase`** rows | **CLI / app** (not **`Codebase.start`**) | **`Folder.read_dir`** via **`ProjectManager.activate_project`** |
| **Queue build** | DB тЖТ enqueue where **`last_vector_scan < mtime_on_disk()`** | **`Codebase.start`** | **`BackgroundScan`** **`queueProject`** path тАФ **ЁЯЪл** no disk walk |
| **Index run** | FAISS embed + metadata | daemon (async) | **`BackgroundScan.startQueue`** |

**ЁЯФ╖** **`Codebase.start`** = pause тЖТ wait idle тЖТ enqueue stale files from DB тЖТ resume queue. **ЁЯЪл** no **`read_dir`**, **ЁЯЪл** no **`create_project`**. The CLI (or app) must filesystem-scan **before** **`start`** so **`filebase`** is current.

**`stop`** does **not** clear the queue тАФ a later **`start`** appends newly stale files and resumes pending work.

### Progress notification (тЬЕ)

```json
{"method":"event.vector.scan_update","object-type":"Vector","message":"12 /path/to/file.vala"}
```

- **`message`** = `queue_size` + space + `current_file`; **`"0 "`** when queue empty
- **`Codebase.start`** / **`stop`** reply immediately тАФ **ЁЯЪл** do not block RPC for full index run

### `Response.result` тАФ always a list

**`Response.result`** is **`Gee.ArrayList<GLib.Object>`** (never null) тАФ **`default = new Gee.ArrayList<GLib.Object>()`**. Length 0, 1, or N after handlers populate it. **ЁЯЪл** no bare single-object **`result`**. One row тЖТ one-element list. **ЁЯЪл** no **`result_type`** / **`is_array`** тАФ see **Phase 1.5**.

**Clients:** **`response.result`** is always a list. One **`(Gee.ArrayList<T>) response.result`** cast per site тАФ **ЁЯЪл** no **`(Gee.ArrayList<GLib.Object>)`** intermediate (**`raw`**, etc.). **ЁЯЪл** no **`response.result == null`** / **`!= null`** тАФ guard **`response.error`** only, then cast. List-returning methods: cast and return (empty list casts fine). Single-row methods: cast typed list, then **`list.get(0)`** when **`list.size > 0`**. **ЁЯЪл** no **`response.result as File`** / **`(File) response.result`**. **ЁЯЪл** no copy-**`foreach`** loops.

**Encode:** **`bin_write_prop`** omits the **`result`** property when **`result.size == 0`** (default empty, msg-only, and intentional empty replies). **Decode:** property absent тЖТ default empty list remains.

**Depends on:** ┬з1.5.2тАУ┬з1.5.3.

#### Replace with

```markdown
**Property type:** **`Gee.ArrayList<GLib.Object>`** тАФ **`default = new Gee.ArrayList<GLib.Object>()`**; never null. Handlers populate **`result`** (length 0, 1, or N). Single-row RPCs use a one-element list тАФ **ЁЯЪл** no bare object on **`result`**.

**Decode:** property present тЖТ **`Stream.parse_object_array()`**; property absent тЖТ default empty list.
```
