## Meson option: `v2testing`

- **🔷** Root boolean option **`v2testing`** in **`meson_options.txt`** — default **`true`** (V2 build).
  - Shipping v1: **`meson setup build -Dv2testing=false`**
  - Meson option names are lowercase; use **`v2testing`** (not `V2TESTNG` / env-style names).
- **🔷** **`get_option('v2testing')` only in `meson.build` files** — pick source lists, deps, and targets.
- **🚫 Pure Meson — do not pass into the codebase:**
  - **🚫** no **`add_project_arguments`**, **`add_global_arguments`**, or **`-D` / `-C` Vala defines** for this flag
  - **🚫** no **`#if` / `const bool` / runtime checks** in **`.vala`** keyed off a compile-time V2 switch
  - **🚫** no reading **`v2testing`** from env at runtime in app code
- **🔷** Behaviour differences come from **which sources are compiled** (e.g. **`libocfiles/V2/*`** vs shipping, **`ollmapp/V2/Window.vala`** vs shipping **`Window.vala`**) — not from `#if v2testing` inside one file.
- **🛑 Stop and rethink** if implementation needs **`#if`**, Vala **`--define`**, or runtime “am I on V2?” branching — update this plan and get approval before touching **`.vala`**.

**When `v2testing` is `true` (Meson only):**

- swap each library's source list to **`V2/*`** / **`*V2/*`** copies documented in cutover inventory
- **`ollmapp`** builds the V2 startup path (spawn **`ollmfilesd`**, no in-process **`BackgroundScan`**) via **swapped sources**, not `#if`
- drop shipping **`libocfiles/Sandbox/*`** from **`ocfiles_src`**

**🚫** Green **`ninja -C build`** (default V2) before treating flip as done.

**✔️** Default flipped **`true`** (`2026-07-04`). Shipping v1: **`-Dv2testing=false`**.

**⏳** Build gaps — [`2.10.4.24-ACTIVE-v2-build-gaps.md`](2.10.4.24-ACTIVE-v2-build-gaps.md) (inventory + **🔷** user decisions; **🚫** no compile-only stubs).
