## Purpose

- **🔷** Local inference ([`8.0`](8.0-libollmchat-local-gguf-backend.md)) needs a reliable way to **find** and **fetch** GGUF weights from Hugging Face Hub.
  - **ℹ️** Hub exposes **public JSON search** and **unauthenticated file resolve** for most open models.
  - **ℹ️** Unlike `libollamaweb`, we are **not** scraping a third-party site — **no catalog-on-disk cache** for search hits.
- **🔷** **Near-term goal:** search Hub for **speculative-decoding** GGUF (draft / auxiliary models), **fetch detail** for one **model ref**, **download** one file — **not** BGE embeddings first.
  - **💩** `⏳` Probe: `oc-hf --search speculative` (and `draft`, `eagle`) before locking Phase 2 target.
  - **ℹ️** **`libllama` speculative decode runtime** — follow-up in [`8.0`](8.0-libollmchat-local-gguf-backend.md) Phase 6+; **out of scope** here.
- **🔷** `✔️` Standalone library **`libochf`** — namespace **`OLLMhf`**, **no** `libollmchat` / `ollmapp` dependency.
- **🔷** `✔️` **All Hub JSON** through **`Client.call(Request)`** — **`method`** + typed **`param`** (`CallParam` subclass) + **`result_type`**; **no** direct Soup / **`Json.Parser`** in **`libochf`**.
- **🔷** `⏳` GGUF bytes — **`OLLMhf.Download`** GObject (one class, **`Download.vala`**); **`start()`** / **`stop()`**, **`progress`** signal, libsoup stream-to-disk; **not** `libocrpc`.
- **🔷** `✔️` Mini example **`examples/oc-hf.vala`** — **`--search`**, **`--detail`**, later **`--download`**.
- **🔷** `✔️` Phase 0 — [`8.2.1`](8.2.1-libocrpc-auto-json-and-http-client.md) HTTP + AUTO JSON on **`OLLMrpc.Client`**.
- **🔷** `✔️` Phase 1 — **`libochf`** + **`oc-hf --search`** / **`--detail`**.
- **🔷** `⏳` Phase 2 — **`oc-hf --download`** one speculative GGUF into [`8.0` model layout](8.0-libollmchat-local-gguf-backend.md#model-layout).
- **💩** `⏳` Phase 3 (deferred) — BGE-m3 for [`8.0` Phase 4](8.0-libollmchat-local-gguf-backend.md#phase-4--bge-download).

## Terminology `🔷`

- **🔷** **Model ref** — Hub repo string **`author/name`** (e.g. `groonga/bge-m3-Q4_K_M-GGUF`). **Not** a numeric id. Stored on **`OLLMhf.Model.id`**; plan text may say **`model_ref`** when describing callers.
- **🔷** **Install dir** — `{models_dir}/{author}/{name}/` derived from **`model.id`** (split **`/`** into path segments). **No** separate destination alias.
- **🔷** **Filename** — one entry from **`siblings[]`** (e.g. `bge-m3-q4_k_m.gguf`). A repo may have **many** `.gguf` files; optional **`--file`** limits which siblings **`Download`** fetches (default: all **`.gguf`** in **`model.siblings`**).

### Landed ([`8.2.1`](8.2.1-libocrpc-auto-json-and-http-client.md)) `✔️`

- **🔷** `✔️` **`OLLMrpc.Client`** HTTP — **`connect()`** + **`call()`**.
- **🔷** `✔️` **`Request`**: **`method`**, **`param`** (`CallParam`), **`result_type`** (HTTP decode hint).
- **🔷** `✔️` **`Request.result-type`** omitted on bin wire.

### HTTP `call()` contract (Hub metadata) `🔷`

- **🔷** **`Client.socket_path`** — Hub base URL (e.g. **`https://huggingface.co`**)
- **🔷** **`Request.method`** — Hub path (e.g. **`"/api/models"`**; detail adds **`"/" + model_ref`** in caller)
- **🔷** **`Request.param`** — query fields → **`?name=value&…`** (generic; **`send_http()`** builds this)
- **🔷** **`Client.send_http()`** — **`socket_path + method`**, then append **`?key=value&…`** from **`param`** properties (incremental **`qs`** string); array root wrap
- **🚫** **No** `to_method()` · **🚫** no **`http_url()`** helper · **🚫** no **`string[]`** / **`joinv`** for query parts

## API investigation (2026-07-09) `ℹ️`

### Base URL and auth `ℹ️`

- **ℹ️** Base: `https://huggingface.co`
- **ℹ️** **No token** for public metadata + open-repo resolve.
- **ℹ️** **Gated** repos → **401** on resolve — skip in Phase 1–2 defaults.

### Search — `GET /api/models` `ℹ️`

- **ℹ️** Response: **JSON array** of shallow model objects.
- **ℹ️** Params: `search`, `filter=gguf`, `sort=downloads`, `direction=-1`, `limit` (≤100).
- **💩** `⏳` Probe queries: `speculative`, `draft`, `eagle` (+ `filter=gguf`).

### Detail — `GET /api/models/{model_ref}` `ℹ️`

- **ℹ️** **`{model_ref}`** = `author/name` in the URL path.
- **ℹ️** **`siblings[]`**: `rfilename`, `size` — filter **`.gguf`** → **`OLLMhf.ModelFile`**.

### Download — `GET /{model_ref}/resolve/{rev}/{filename}` `ℹ️`

- **ℹ️** Redirect to CDN; stream bytes to disk.
- **ℹ️** Default rev: **`main`** unless caller passes another.
- **ℹ️** LFS files: **`ETag`** on the resolved response is the file **SHA-256** (use for final checksum compare).
- **ℹ️** **Not** Ollama-style progress JSON — no server **`digest`** / per-layer chunks. Progress is **client-side** (bytes read vs **`ModelFile.size`**).
- **ℹ️** libsoup delivers **read buffers** (~8–64 KiB per `read()`); update **`GLib.Checksum`** after each buffer write. **Final** validation: size + SHA-256.
- **ℹ️** **Resume:** load **`Model`** from **`download.json`** (**`Bin.Json`** round-trip) → restore **`ModelFile`** download fields → HTTP **`Range`** append to **`{rfilename}.partial`**.

### Local install layout ([`8.0`](8.0-libollmchat-local-gguf-backend.md#model-layout)) `🔷`

- **🔷** **`models_dir`** — root for all local weights (default `~/.local/share/ollmchat/models`). Only caller-supplied path; everything else comes from **`Model`**.
- **🔷** Per-model dir: **`{models_dir}/{author}/{name}/`** where **`author/name`** = **`model.id`** (Hub repo id).
- **🔷** Sibling files land as **`{models_dir}/{author}/{name}/{rfilename}`** (+ **`manifest.json`**, checksum sidecars).
- **🔷** `⏳` While downloading: **`{rfilename}.partial`** + **`download.json`** (**`Model`** via **`Bin.Json`**; removed when complete).
- **💩** `⏳` Phase 2 proof **model ref** TBD from search probe.

		{

			GLib.debug("id=%d url=%s", head.request.id, url);
			var message = new Soup.Message("GET", url);

#### Replace with `✔️` — full method (match **`Client.vala`** style; query inline):

```vala
		private async void send_http(PendingWrite head) throws GLib.Error
		{
			GLib.debug("id=%d path=%s", head.request.id, head.request.method);
			var url = this.socket_path + head.request.method;
			var qs = "";
			foreach (var pspec in head.request.param.get_class().list_properties()) {
				if (pspec.name == "args") {
					continue;
				}
				var val = Value(pspec.value_type);
				head.request.param.get_property(pspec.name, ref val);
				switch (val.type()) {
					case GLib.Type.STRING:
						var s = val.get_string();
						if (s == "") {
							continue;
						}
						qs += (qs == "" ? "?" : "&")
							+ GLib.Uri.escape_string(pspec.name, null)
							+ "=" + GLib.Uri.escape_string(s, null);
						break;
					case GLib.Type.INT:
						qs += (qs == "" ? "?" : "&")
							+ GLib.Uri.escape_string(pspec.name, null)
							+ "=" + val.get_int().to_string();
						break;
				}
			}
			url += qs;
			GLib.debug("id=%d url=%s", head.request.id, url);
			var message = new Soup.Message("GET", url);
			var bytes = yield this.http_session.send_and_read_async(
				message, GLib.Priority.DEFAULT, null);
			if (message.status_code < 200 || message.status_code >= 300) {
				throw new GLib.IOError.FAILED("HTTP %u for %s", message.status_code, url);
			}
			if (head.request.result_type == GLib.Type.INVALID) {
				throw new Bin.StreamError.PROTOCOL("HTTP call missing result_type");
			}
			var parser = new Json.Parser();
			parser.load_from_data((string) bytes.get_data());
			var root = parser.get_root();
			if (root.get_node_type() == Json.NodeType.ARRAY) {
				var wrap = new Json.Object();
				wrap.set_array_member("items", root.get_array());
				root = new Json.Node(Json.NodeType.OBJECT);
				root.set_object(wrap);
			}
			if (root.get_node_type() != Json.NodeType.OBJECT) {
				throw new Bin.StreamError.PROTOCOL("HTTP JSON root must be object or array");
			}
			var mem = new GLib.MemoryOutputStream.resizable();
			var encode_ctx = new Bin.Stream(null, new GLib.DataOutputStream(mem));
			this.http_json.json_to_bin(root.get_object(), encode_ctx, head.request.result_type);
			encode_ctx.out_stream.close();
			var read_ctx = new Bin.Stream(new GLib.DataInputStream(
				new GLib.MemoryInputStream.from_bytes(mem.steal_as_bytes())), null);
			var obj = read_ctx.parse();
			var response = new Response() {
				id = head.request.id,
			};
			response.result.add(obj);
			this.complete_pending(head.request.id, response, null);
		}
```

#### Add `✔️` — `libochf/ModelArray.vala`:

Decode-only wrapper for Hub search **`[…]`** → **`{ "items": […] }`**. JSON has no element type; **`send_http()`** knows elements are **`Model`** via **`result_type`**. **🚫** No **`bin_write_prop`** — HTTP receive path only.

```vala
namespace OLLMhf
{
	public class ModelArray : GLib.Object, OLLMrpc.Bin.Serializable
	{
		public Gee.ArrayList<Model> items {
			get; set; default = new Gee.ArrayList<Model>();
		}

		public static void rpc_register() {
			OLLMrpc.Bin.register("ModelArray", typeof(ModelArray));
		}

		public override void bin_read_prop(
			OLLMrpc.Bin.Stream ctx,
			GLib.ParamSpec prop,
			uint8 type_byte
		) throws GLib.Error {
			switch (prop.name) {
				case "items":
					if ((type_byte & 0x7F) != GLib.Type.OBJECT
						|| (type_byte & 0x80) == 0) {
						throw new OLLMrpc.Bin.SerializableError.PROPERTY(
							"prop '%s' expected object array", prop.name);
					}
					ctx.read_gtype();
					var count = (uint) ctx.in_stream.read_byte();
					if ((count & 0x80) != 0) {
						count = ((count & 0x7F) << 8) | ctx.in_stream.read_byte();
					}
					this.items = new Gee.ArrayList<Model>();
					for (var i = 0; i < count; i++) {
						var child = (Model) GLib.Object.new(typeof(Model));
						child.bin_read(ctx);
						this.items.add(child);
					}
					return;
				default:
					this.bin_default_read_prop(ctx, prop, type_byte);
					return;
			}
		}
	}
}
```

### 6. `libochf/Download.vala` — download handle: stream resolve → disk `⏳`

**Why:** GGUF bytes are not JSON; libsoup stream-to-disk in **`libochf`** only. Object + signals mirrors **`OLLMchat.Call.Pull`** (library layer); app-level multi-download orchestration (like **`PullManager`**) stays out of scope.

**Where:** new **`libochf/Download.vala`**; add to **`ochf_src`** in **`libochf/meson.build`**.

**Depends on:** §3 `⏳`.

**Contrast — Ollama pull `ℹ️`**

- **ℹ️** Ollama: server streams NDJSON **`progress_chunk`** per **layer** (`status`, `digest`, `completed`, `total`).
- **ℹ️** HF Hub: one HTTP byte stream; **`progress`** signal is **synthetic** (`completed`/`total` from read loop).
- **ℹ️** Ollama validates each **layer** digest when that layer finishes; HF validates the **whole file** SHA-256 once at end (incremental hash during read buffers).

#### Add `⏳` — `OLLMhf.Model` + `OLLMhf.ModelFile` local download fields:

**Why:** Crash-safe resume by serializing **`Model`** / **`ModelFile`** — **no** separate download-status schema.

**Where:** extend **`libochf/Model.vala`** and **`libochf/ModelFile.vala`** (§3).

```vala
// Model.vala — local-only (not from Hub JSON / not on bin wire)
		/** Revision used for resolve URLs during an in-progress download. */
		public string download_revision { get; set; default = "main"; }

// ModelFile.vala — local-only
		/** Bytes written so far for this sibling. */
		public int64 bytes_written { get; set; default = 0; }
		/** LFS ETag from HEAD (SHA-256) for final verify. */
		public string etag { get; set; default = ""; }
		/** Incremental SHA-256 hex while streaming (resume). */
		public string sha256_partial { get; set; default = ""; }
		/** True when this sibling is fully downloaded and verified. */
		public bool download_complete { get; set; default = false; }
```

- **🔷** All fields above are **local-only on Hub HTTP** — Hub JSON does not set them; they **do** round-trip in **`download.json`** / **`manifest.json`** via **`Bin.Json`** (AUTO mode).
- **🔷** **`Download.start()`** loads **`download.json`** when restored **`id`** matches **`this.model.id`**.
- **🔷** Queue order: filtered **`.gguf`** siblings; skip **`download_complete`**; resume first incomplete (**`bytes_written`** &lt; **`size`**).
- **🚫** **No** **`Json.gobject_*`**.

#### Add `⏳` — `{model_dir}/download.json` + `manifest.json` (`Bin.Json` + `Model`):

**Why:** Crash-safe resume; human-readable JSON on disk, encoded through the **bin JSON bridge** — not ad-hoc GObject JSON.

**Where:** **`{model_dir}`** built inline in **`start()`**; read/write inline there too.

**Codec:** **`new OLLMrpc.Bin.Json(OLLMrpc.Bin.Json.Mode.AUTO)`** on **`Download`**. **`OLLMhf.rpc_register()`** before encode/decode.

**Write** (`Model` → `download.json`):

```vala
			var mem = new GLib.MemoryOutputStream.resizable();
			var encode_ctx = new OLLMrpc.Bin.Stream(
				null,
				new GLib.DataOutputStream(mem)
			);
			encode_ctx.write(this.model);
			encode_ctx.out_stream.close();
			var decode_ctx = new OLLMrpc.Bin.Stream(
				new GLib.DataInputStream(
					new GLib.MemoryInputStream.from_bytes(mem.steal_as_bytes())
				),
				null
			);
			var json_text = Json.to_string(
				this.json_codec.bin_to_json(decode_ctx),
				true
			);
			GLib.FileUtils.set_contents(path, json_text);
```

**Read** (JSON → `Model`):

```vala
			string contents;
			GLib.FileUtils.get_contents(path, out contents, null);
			var parser = new Json.Parser();
			parser.load_from_data(contents);
			var root = parser.get_root();
			var mem = new GLib.MemoryOutputStream.resizable();
			var encode_ctx = new OLLMrpc.Bin.Stream(
				null,
				new GLib.DataOutputStream(mem)
			);
			this.json_codec.json_to_bin(
				root.get_object(),
				encode_ctx,
				typeof(Model)
			);
			encode_ctx.out_stream.close();
			var decode_ctx = new OLLMrpc.Bin.Stream(
				new GLib.DataInputStream(
					new GLib.MemoryInputStream.from_bytes(mem.steal_as_bytes())
				),
				null
			);
			var restored = (Model) decode_ctx.parse();
```

- **🔷** During transfer: rate-limited flush → **`download.json`** (e.g. every 5 s or every 8 MiB).
- **🔷** On **`start()`**: if **`download.json`** exists and restored **`id`** matches, **`this.model = restored`**; else keep construct-time detail.
- **🔷** On all siblings done: write **`manifest.json`**, delete **`download.json`** and **`.partial`** files.
- **🚫** **No** custom JSON schema — extend **`Model`** / **`ModelFile`** properties; **`Bin.Json`** maps them.
- **ℹ️** **`Json.Parser`** here is **only** for local **`download.json`** / **`manifest.json`** — **not** for Hub HTTP (that stays **`Client.call`**).

#### Download flow `⏳`

**Entry:** detail **`Model`** → **`new Download(model)`** → connect **`progress`** → **`yield dl.start()`**.

**Public API only:** **`start()`**, **`stop()`**, **`progress`** signal. **🚫** No private helpers — path build, JSON read/write, and per-sibling HTTP streaming are **inline in `start()`** (use **ordered chunks** in the plan fence below).

1. **Install dir** — inline: default **`models_dir`** if empty; foreach **`model.id.split("/")`** → **`Path.build_filename`**; **`make_dir_with_parents`**.
2. **Restore** — inline: **`OLLMhf.rpc_register()`**; read **`download.json`** via **`Json.Parser`** → **`json_to_bin`** → **`parse`** → **`Model`** when **`id`** matches.
3. **`model.download_revision = this.revision`**; build **`.gguf`** queue ( **`file_filter`**, skip **`download_complete`**).
4. **Per sibling** — inline in queue loop:
   - HEAD resolve URL → **`file.etag`**
   - **`{rfilename}.partial`**; GET + **`Range`** when **`bytes_written > 0`**
   - Buffer loop → checksum → **`progress`** → rate-limited **`download.json`**
   - Verify; rename; **`download_complete = true`**
5. **Finish** — **`manifest.json`** (same path); delete **`download.json`** + **`.partial`**

**HTTP `ℹ️`**

- **ℹ️** URL: **`file.to_url(model.id, model.download_revision)`** — [API investigation — Download](#download--get-model_refresolverevfilename-ℹ️).
- **ℹ️** **`Soup.Session`** on **`Download`**; follow redirects.
- **ℹ️** **`stop()`** cancels internal **`Cancellable`**; partial + **`download.json`** remain for resume.
