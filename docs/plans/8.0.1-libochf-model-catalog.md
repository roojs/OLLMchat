## Purpose

- **🔷** Local inference ([`8.0`](8.0-libollmchat-local-gguf-backend.md)) needs a reliable way to **find** and **fetch** GGUF weights from Hugging Face Hub.
  - **ℹ️** Hub exposes **public JSON search** and **unauthenticated file resolve** for most open models.
  - **ℹ️** Unlike `libollamaweb`, we are **not** scraping a third-party site — **no catalog-on-disk cache** for search hits.
- **🔷** **Near-term goal:** search Hub for **speculative-decoding** GGUF (draft / auxiliary models), **fetch detail** for one **model ref**, **download** one file — **not** BGE embeddings first.
  - **💩** `⏳` Probe: `oc-hf --search speculative` (and `draft`, `eagle`) before locking Phase 2 target.
  - **ℹ️** **`libllama` speculative decode runtime** — follow-up in [`8.0`](8.0-libollmchat-local-gguf-backend.md) Phase 6+; **out of scope** here.
- **🔷** `⏳` Standalone library **`libochf`** — namespace **`OLLMhf`**, **no** `libollmchat` / `ollmapp` dependency.
- **🔷** `⏳` **All Hub JSON** through **`Client.call(Request)`** — **`method`** + typed **`param`** (`CallParam` subclass) + **`result_type`**; **no** direct Soup / **`Json.Parser`** in **`libochf`**.
- **🔷** `⏳` GGUF bytes — **`OLLMhf.Download`** (one class, **`Download.vala`**); libsoup stream-to-disk only; **not** `libocrpc`.
- **🔷** `⏳` Mini example **`examples/oc-hf.vala`** — **`--search`**, **`--detail`**, later **`--download`**.
- **🔷** `✔️` Phase 0 — [`8.2.1`](8.2.1-libocrpc-auto-json-and-http-client.md) HTTP + AUTO JSON on **`OLLMrpc.Client`**.
- **🔷** `⏳` Phase 1 — **`libochf`** + **`oc-hf --search`** / **`--detail`**.
- **🔷** `⏳` Phase 2 — **`oc-hf --download`** one speculative GGUF into [`8.0` model layout](8.0-libollmchat-local-gguf-backend.md#model-layout).
- **💩** `⏳` Phase 3 (deferred) — BGE-m3 for [`8.0` Phase 4](8.0-libollmchat-local-gguf-backend.md#phase-4--bge-download).

## Terminology `🔷`

- **🔷** **Model ref** — Hub repo string **`author/name`** (e.g. `groonga/bge-m3-Q4_K_M-GGUF`). **Not** a numeric id. Plan text uses **`model_ref`**; JSON field from Hub is often **`id`** or **`modelId`** — map into **`OLLMhf.Model.model_ref`**.
- **🔷** **Filename** — one entry from **`siblings[]`** (e.g. `bge-m3-q4_k_m.gguf`). A repo may have **many** `.gguf` files; **`--file`** selects which one when ambiguous.

## Prerequisite — `libocrpc` `✔️` + array HTTP `⏳`

### Landed ([`8.2.1`](8.2.1-libocrpc-auto-json-and-http-client.md)) `✔️`

- **🔷** `✔️` **`Bin.Json`** AUTO — **`json_to_bin(obj, bin, type)`**.
- **🔷** `✔️` **`OLLMrpc.Client`** HTTP — **`connect()`** + **`call()`**.
- **🔷** `✔️` **`Request`**: **`method`**, **`param`** (`CallParam`), **`result_type`** (HTTP decode hint).
- **🔷** `✔️` **`Request.result-type`** omitted on bin wire.

### HTTP `call()` contract (Hub metadata) `🔷`

- **🔷** **`Client.socket_path`** — Hub base URL (e.g. **`https://huggingface.co`**)
- **🔷** **`Request.method`** — Hub path (e.g. **`"/api/models"`**; detail adds **`"/" + model_ref`** in caller)
- **🔷** **`Request.param`** — query fields → **`?name=value&…`** (generic; **`send_http()`** builds this)
- **🔷** **`Client.send_http()`** — **`socket_path + method`**, then append **`?key=value&…`** from **`param`** properties (incremental **`qs`** string); array root wrap
- **🚫** **No** `to_method()` · **🚫** no **`http_url()`** helper · **🚫** no **`string[]`** / **`joinv`** for query parts

		{
			var url = this.socket_path + head.request.method;
			GLib.debug("id=%d url=%s", head.request.id, url);
			var message = new Soup.Message("GET", url);
			var bytes = yield this.http_session.send_and_read_async(
				message, GLib.Priority.DEFAULT, null);
			if (message.status_code < 200 || message.status_code >= 300) {
				throw new GLib.IOError.FAILED("HTTP %u for %s", message.status_code, url);
			}
			if (head.request.result_type == GLib.Type.INVALID) {
				throw new Bin.StreamError.PROTOCOL("HTTP call missing result_type");
			}
			var parser = new Json.Parser();
			parser.load_from_data((string) bytes.get_data());
			var root = parser.get_root();
			if (root.get_node_type() != Json.NodeType.OBJECT) {
				throw new Bin.StreamError.PROTOCOL("HTTP JSON root must be object");
			}
			var mem = new GLib.MemoryOutputStream.resizable();
			var encode_ctx = new Bin.Stream(null, new GLib.DataOutputStream(mem));
			this.http_json.json_to_bin(root.get_object(), encode_ctx, head.request.result_type);
			encode_ctx.out_stream.close();
			var read_ctx = new Bin.Stream(new GLib.DataInputStream(
				new GLib.MemoryInputStream.from_bytes(mem.steal_as_bytes())), null);
			var obj = read_ctx.parse();

#### Replace with `⏳` — full method (match **`Client.vala`** style; query inline):

```vala
		private async void send_http(PendingWrite head) throws GLib.Error
		{
			GLib.debug("id=%d path=%s", head.request.id, head.request.method);
			var url = this.socket_path + head.request.method;
			var qs = "";
			foreach (var pspec in head.request.param.get_class().list_properties()) {
				if (pspec.name == "args") {
					continue;
				}
				var val = Value(pspec.value_type);
				head.request.param.get_property(pspec.name, ref val);
				switch (val.type()) {
					case GLib.Type.STRING:
						var s = val.get_string();
						if (s == "") {
							continue;
						}
						qs += (qs == "" ? "?" : "&")
							+ GLib.Uri.escape_string(pspec.name, null)
							+ "=" + GLib.Uri.escape_string(s, null);
						break;
					case GLib.Type.INT:
						qs += (qs == "" ? "?" : "&")
							+ GLib.Uri.escape_string(pspec.name, null)
							+ "=" + val.get_int().to_string();
						break;
				}
			}
			url += qs;
			GLib.debug("id=%d url=%s", head.request.id, url);
			var message = new Soup.Message("GET", url);
			var bytes = yield this.http_session.send_and_read_async(
				message, GLib.Priority.DEFAULT, null);
			if (message.status_code < 200 || message.status_code >= 300) {
				throw new GLib.IOError.FAILED("HTTP %u for %s", message.status_code, url);
			}
			if (head.request.result_type == GLib.Type.INVALID) {
				throw new Bin.StreamError.PROTOCOL("HTTP call missing result_type");
			}
			var parser = new Json.Parser();
			parser.load_from_data((string) bytes.get_data());
			var root = parser.get_root();
			if (root.get_node_type() == Json.NodeType.ARRAY) {
				var wrap = new Json.Object();
				wrap.set_array_member("items", root.get_array());
				root = new Json.Node(Json.NodeType.OBJECT);
				root.set_object(wrap);
			}
			if (root.get_node_type() != Json.NodeType.OBJECT) {
				throw new Bin.StreamError.PROTOCOL("HTTP JSON root must be object or array");
			}
			var mem = new GLib.MemoryOutputStream.resizable();
			var encode_ctx = new Bin.Stream(null, new GLib.DataOutputStream(mem));
			this.http_json.json_to_bin(root.get_object(), encode_ctx, head.request.result_type);
			encode_ctx.out_stream.close();
			var read_ctx = new Bin.Stream(new GLib.DataInputStream(
				new GLib.MemoryInputStream.from_bytes(mem.steal_as_bytes())), null);
			var obj = read_ctx.parse();
			var response = new Response() {
				id = head.request.id,
			};
			response.result.add(obj);
			this.complete_pending(head.request.id, response, null);
		}
```

#### Add `⏳` — `libochf/ModelArray.vala`:

Decode-only wrapper for Hub search **`[…]`** → **`{ "items": […] }`**. JSON has no element type; **`send_http()`** knows elements are **`Model`** via **`result_type`**. **🚫** No **`bin_write_prop`** — HTTP receive path only.

```vala
namespace OLLMhf
{
	public class ModelArray : GLib.Object, OLLMrpc.Bin.Serializable
	{
		public Gee.ArrayList<Model> items {
			get; set; default = new Gee.ArrayList<Model>();
		}

		public static void rpc_register() {
			OLLMrpc.Bin.register("ModelArray", typeof(ModelArray));
		}

		public override void bin_read_prop(
			OLLMrpc.Bin.Stream ctx,
			GLib.ParamSpec prop,
			uint8 type_byte
		) throws GLib.Error {
			switch (prop.name) {
				case "items":
					if ((type_byte & 0x7F) != GLib.Type.OBJECT
						|| (type_byte & 0x80) == 0) {
						throw new OLLMrpc.Bin.SerializableError.PROPERTY(
							"prop '%s' expected object array", prop.name);
					}
					var elem_gtype = ctx.read_gtype();
					if (elem_gtype != typeof(Model)) {
						throw new OLLMrpc.Bin.SerializableError.PROPERTY(
							"prop '%s' expected Model elements", prop.name);
					}
					var count = (uint) ctx.in_stream.read_byte();
					if ((count & 0x80) != 0) {
						count = ((count & 0x7F) << 8) | ctx.in_stream.read_byte();
					}
					this.items = new Gee.ArrayList<Model>();
					for (var i = 0; i < count; i++) {
						var child = (Model) GLib.Object.new(typeof(Model));
						child.bin_read(ctx);
						this.items.add(child);
					}
					return;
				default:
					this.bin_default_read_prop(ctx, prop, type_byte);
					return;
			}
		}
	}
}
```

- **🚫** No **`Session`** — callers own **`OLLMrpc.Client`** directly.
- **🚫** No **`Search.Service`**, disk catalog cache.
- **🚫** Say **model ref** (`author/name`), not numeric **id**.
