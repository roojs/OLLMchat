write_changes

Description:

public void write_changes (DataOutputStream output_stream, int current_line) throws Error

Writes the replacement text to the output stream for insertions at end of file. Only writes if this is an insertion (start == end) and it's past the current line.

Parameters:

output_stream

The output stream to write replacement to

current_line

The current line number in the file

Exceptions:

Error

if I/O operations fail
