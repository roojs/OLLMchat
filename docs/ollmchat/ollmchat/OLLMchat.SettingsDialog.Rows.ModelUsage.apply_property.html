public void apply_property (Object config)

Applies the ModelUsage object to the parent config object.

The nested widgets (Connection, Model, Options) already update the ModelUsage object directly via their apply_property methods. This method applies the ModelUsage object back to the parent config object.

Parameters:

config

The parent config object to apply the ModelUsage to
