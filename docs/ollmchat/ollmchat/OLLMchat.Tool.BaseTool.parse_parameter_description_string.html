Description:

protected void parse_parameter_description_string (string in_desc, HashMap<string,ParamObject> type_definitions)

Parses a single parameter description and adds it to the function's parameters property.

Format: @param parameter_name {type} [required|optional] Parameter description here Format: @param parameter_name {array<type>} [required|optional] Parameter description here
