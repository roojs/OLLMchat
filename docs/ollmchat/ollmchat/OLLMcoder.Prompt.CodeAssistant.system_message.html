system_message

Description:

public override string system_message (AgentHandler? handler = null) throws Error

Generates the complete system prompt for a code-assistant agent.

Overrides BaseAgent.system_message() to generate system prompt with current context. This includes: introduction, communication rules, tool calling, search/reading rules, code changes rules, debugging, external APIs, user info (OS, workspace, shell), and citation format.

Parameters:

handler

Optional AgentHandler instance (can access session, client, etc.)

Returns:

Complete system prompt string

Exceptions:

Error

if system message generation fails
