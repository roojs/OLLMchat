public void setup_defaults (string connection_url)

Sets up default values for embed and analysis model configurations.

Sets default model names, options, and connection: - Embed: model "bge-m3:latest", temperature 0.0, num_ctx 2048 - Analysis: model "qwen3 -coder:30b", temperature 0.0 - Both use the provided connection URL

Parameters:

connection_url

The connection URL to use for both embed and analysis models
