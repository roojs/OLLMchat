tool_message

Description:

public signal void tool_message (string message, Object? widget = null)

Emitted when a tool sends a status message during execution.

Parameters:

message

The status message from the tool

widget

Optional widget parameter (default null). Expected to be a Gtk.Widget, but typed as Object? since the Ollama base library should work without Gtk. A cast will be needed when using this parameter in Gtk-based applications.

1.0
