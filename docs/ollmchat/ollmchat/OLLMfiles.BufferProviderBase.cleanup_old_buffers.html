cleanup_old_buffers

Description:

protected void cleanup_old_buffers (File current_file)

Cleanup old buffers to free memory.

Keeps buffers for: - Open files (is_open == true) - Top 10 most recently used files (by last_viewed)

Sets file.buffer = null for all other files.

Parameters:

current_file

The file currently being accessed (always keeps its buffer)
