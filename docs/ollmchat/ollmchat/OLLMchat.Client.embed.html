embed

Description:

public async Embed embed (string input, int dimensions = -1, bool truncate = false, Cancellable? cancellable = null) throws Error

Generates embeddings for the input text.

Creates vector embeddings representing the input text using the configured model. Returns a single Response.Embed object containing the embeddings and metadata.

Parameters:

input

The text to generate embeddings for

dimensions

Optional number of dimensions to generate embeddings for (default: -1, not set)

truncate

Optional whether to truncate inputs that exceed context window (default: false)

cancellable

Optional cancellable for cancelling the request

Returns:

Response.Embed object with embeddings and timing information

Since:

1.0
