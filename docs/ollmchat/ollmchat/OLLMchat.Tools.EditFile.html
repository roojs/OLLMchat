Description:

public class EditFile : Tool

Tool for editing files by applying a list of edits (ranges with replacement text).

This tool applies edits to a file using a streaming approach to minimize memory usage. Supports two-step permission flow: first READ permission to generate diff, then WRITE permission with diff display.
