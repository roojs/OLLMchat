Description:

public static Config from_file (string config_path)

Creates a new Config object loaded from a file.

If the file doesn't exist or cannot be read/parsed, returns a Config object with default values and loaded set to false.

Parameters:

config_path

Path to the configuration file

Returns:

A new Config instance loaded from the file
