analyze_tree

Description:

public async Tree analyze_tree (Tree tree) throws Error

Analyzes a Tree object and generates descriptions for elements.

Iterates over Tree.elements and: - Skips LLM for simple elements (enum types without docs, simple properties, etc.) - Calls LLM for complex elements (classes, methods, properties with docs, etc.) - Stores descriptions in VectorMetadata.description property

Parameters:

tree

The Tree object from Tree layer

Returns:

The same Tree object with descriptions populated
