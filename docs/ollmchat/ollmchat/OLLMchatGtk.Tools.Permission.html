Permission

Description:

public class Permission : Provider

Permission provider that displays permission requests using ChatWidget's permission widget.

This provider delegates to ChatWidget's integrated permission widget, which is positioned between ChatView and ChatInput for stable display. Also sends a system notification to inform the user about the permission request.
