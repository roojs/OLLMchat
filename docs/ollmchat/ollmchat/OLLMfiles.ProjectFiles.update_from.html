public void update_from (Folder folder)

Recursively update project files from a folder tree.

Scans the folder and its children recursively, adding only real files (not aliases) to the project files list. Handles FileAliases by following them to their target files. Prevents recursing the same folder more than once by tracking scanned folders using a HashSet of folder IDs.

Parameters:

folder

The folder to start scanning from (should be a project folder)
