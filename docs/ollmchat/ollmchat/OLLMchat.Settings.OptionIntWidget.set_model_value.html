set_model_value

Description:

public override void set_model_value (Value value)

Sets the model's default value (only called by model, not user).

This sets model_value and updates the Auto button label to show the model's default. This does NOT change default_value (hardcoded default) or the actual saved value in Options. It only affects: - What label is shown on Auto button when value is unset - What value appears in spin button when user clicks "Auto" to set custom value
