send_message

Description:

public override async Chat send_message (string text, Cancellable? cancellable = null) throws Error

Sends a message using this session's client.

Sets streaming mode and either continues an existing conversation with reply() or starts a new conversation with chat().

Parameters:

text

The message text to send

cancellable

Optional cancellable for canceling the request

Returns:

The response from the chat API
