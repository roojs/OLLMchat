Classes:

  • Base - Abstract base class for Ollama API responses.
  • CallFunction - Represents a function call with name and arguments. Used within ToolCall to represent the function being called.
  • Chat - Response from a chat API call.
  • Embed - Represents the response from the embeddings API.
  • Model - Represents model information from the Ollama server.
  • ToolCall - Represents a tool call from the assistant. Used in assistant messages with tool_calls array.
