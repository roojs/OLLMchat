Description:

public class ModelsPage : SettingsPage

Models tab content for settings dialog.

Manages model list and model-specific options configuration. Uses Adw.PreferencesGroup with Adw.BoxedList for model list. Models are grouped by connection with section headers.

Since:

1.0
