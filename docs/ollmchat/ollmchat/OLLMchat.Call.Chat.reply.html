reply

Description:

public async Chat reply (string new_text, Chat previous_response) throws Error

Sets up this Chat as a reply to a previous conversation and executes it. Appends the previous assistant response and new user message to the messages array, then calls exec_chat().

Parameters:

new_text

The new user message text

previous_response

The previous Response from the assistant

Returns:

The Response from executing the chat call
