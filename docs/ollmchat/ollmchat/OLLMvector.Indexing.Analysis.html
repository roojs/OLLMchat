Description:

public class Analysis : Object

Analysis layer for code file processing.

Processes Tree objects from the Tree layer and generates one-line descriptions for code elements using LLM. Skips LLM analysis for simple elements (enum types without documentation, simple properties, enum values, fields without docs).
