Classes:

  • DummyRenderer - Dummy renderer for testing the Parser. Extends RenderBase and overrides methods to print callbacks instead of rendering.
  • HtmlParser - Converts HTML to Markdown format.
  • HtmlRender - Simple renderer that converts markdown to HTML strings.
  • PangoRender - Simple renderer that converts markdown and HTML tags to Pango markup strings.
  • Parser - Parser for markdown text that calls specific callbacks on Render.
  • RenderBase - Base abstract class for renderers that use the Parser.
  • TagAnchor - Handler for anchor tags.
  • TagBlockquote - Handler for blockquote tags.
  • TagBreak - Handler for break tags.
  • TagCode - Handler for code tags.
  • TagDiv - Handler for div tags.
  • TagIgnored - Base class for tag handlers. Default implementations do nothing (ignore behavior).
  • TagImage - Handler for image tags.
  • TagListItem - Handler for list item tags.
  • TagOption - Handler for option tags.
  • TagOrderedList - Handler for ordered list tags.
  • TagParagraph - Handler for paragraph tags.
  • TagPre - Handler for pre tags.
  • TagSeperator - Handler for separator tags.
  • TagSimple - Generic handler for simple tags that add the same string at opening and closing.
  • TagSimpleWithBreak - Generic handler for tags that add start on open, and check previous char before adding end on close.
  • TagTable - Handler for table tags.
  • TagTableData - Handler for table data tags.
  • TagTableHeader - Handler for table header tags.
  • TagTableRow - Handler for table row tags.
  • TagTitle - Handler for title tags.
  • TagUnderline - Handler for underline tags.
  • TagUnorderedList - Handler for unordered list tags.
  • Writer - Handles markdown output writing.
