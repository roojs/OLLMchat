execute

Description:

public virtual async string execute (Chat chat_call, Object parameters)

Public method that creates a Request and delegates execution to it.

Converts parameters to Json.Node, calls deserialize() to create a Request object, then sets tool and chat_call properties, and calls its execute() method.

Parameters:

chat_call

The chat call context for this tool execution

parameters

The parameters from the Ollama function call

Returns:

String result or error message (prefixed with "ERROR: " for errors)
