BufferProviderBase

Object Hierarchy:

Description:

public class BufferProviderBase : Object

Base class for buffer operations with default no-op implementations.

Provides a default implementation that does nothing, allowing libocfiles to work without GTK dependencies. Concrete implementations (e.g., in liboccoder) can override these methods to provide actual buffer functionality.
