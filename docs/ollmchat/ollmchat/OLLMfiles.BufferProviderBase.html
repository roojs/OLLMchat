BufferProviderBase

Object Hierarchy:

Description:

public class BufferProviderBase : Object

Base implementation for non-GTK contexts.

Provides a default implementation that creates DummyFileBuffer instances, allowing libocfiles to work without GTK dependencies. Concrete implementations (e.g., in liboccoder) can override create_buffer() to provide GTK buffers.
