OLLMchat.ChatPermission

Description:

Tool permission system namespace.

The OLLMchat.ChatPermission namespace provides permission checking for tool execution. Implement ChatPermission.Provider to control which tools can be executed and under what conditions. The Dummy provider allows all operations for testing.

Content:

Classes:

  • Dummy - Dummy implementation of Provider for testing.
  • NamespaceDoc - Namespace documentation marker. This file contains namespace-level documentation for OLLMchat.ChatPermission.
  • Provider - Abstract base class for requesting permission to execute tool operations.

Enums:

  • Operation - Operation types for permission requests. Can be combined using bitwise OR (e.g., READ | WRITE).
  • PermissionResponse - Permission response from user. Combines allow/deny decision with storage type.
  • PermissionResult - Permission check result.
