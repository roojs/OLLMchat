PullManager

Object Hierarchy:

Description:

public class PullManager : Object

Manages background pull operations for models.

Handles status tracking, persistence, rate-limited UI updates, and retry scheduling. Delegates actual pull execution to PullManagerThread.

Since:

1.3.4
