Options

Description:

public class Options : Object, Serializable

Options object for Ollama API calls.

This is technically accessed through call.options (e.g., chatCall.options, embedCall.options). Contains all runtime parameters that can be passed to Ollama API. Default values are -1 for numbers (indicating no value set) or empty string for strings.
