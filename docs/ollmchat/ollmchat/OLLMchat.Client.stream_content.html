stream_content

Description:

public signal void stream_content (string new_text, Chat response)

Emitted when streaming content (not thinking) is received from the chat API.

This signal is emitted only for regular content chunks, not thinking content. Tools can connect to this signal to capture streaming messages and build strings, including extracting code blocks as they arrive.

Parameters:

new_text

The new content text chunk received (not thinking)

response

The Response.Chat object containing the streaming state
