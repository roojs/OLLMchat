Description:

public static void setup_embed_usage (Config2 config)

Sets up the embed ModelUsage entry in Config2.

Creates a ModelUsage entry for "ocvector.embed" in the config's usage map if it doesn't already exist. Uses the default connection and "bge-m3" model. This should be called when setting up the codebase search tool.

Parameters:

config

The Config2 instance to update
