index_folder

Description:

public async int index_folder (Folder folder, bool recurse = false, bool force = false, int64 scan_time = 0) throws Error

Index a folder and optionally recurse into subfolders.

Uses ProjectFiles to get the list of files to index. ProjectFiles already filters out ignored files and non-text files, and handles recursion.

Parameters:

folder

The folder to index (must exist in database)

recurse

If true, recursively process subfolders (used by ProjectFiles.update_from)

force

If true, skip incremental check and force re-indexing

scan_time

Fixed scan time for this scan session (0 = create new)

Returns:

Number of files indexed
