protected override string generate_user_prompt (string user_query) throws Error

Generates the user prompt with additional context data.

Based on Cursor's implementation, this includes: - <additional_data> section with <current_file>, <attached_files>, <manually_added_selection> - <user_query> tag with the actual user query

user_query

The actual user query/message

User prompt string with additional context
