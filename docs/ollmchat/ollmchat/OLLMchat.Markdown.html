Classes:

  • DummyRenderer - Dummy renderer for testing the Parser. Extends RenderBase and overrides methods to print callbacks instead of rendering.
  • PangoRender - Simple renderer that converts markdown and HTML tags to Pango markup strings.
  • Parser - Parser for markdown text that calls specific callbacks on Render.
  • RenderBase - Base abstract class for renderers that use the Parser.
