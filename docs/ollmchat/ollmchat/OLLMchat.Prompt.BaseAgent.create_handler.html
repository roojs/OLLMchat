create_handler

Description:

public virtual Object create_handler (Client client, SessionBase session)

Creates a handler for a specific request.

This method must be overridden in subclasses or by code that has access to AgentHandler. BaseAgent cannot reference AgentHandler directly due to build order constraints.

Parameters:

client

The client instance for this request

session

The session instance (for accessing Manager and tools)

Returns:

A new handler instance (AgentHandler)
