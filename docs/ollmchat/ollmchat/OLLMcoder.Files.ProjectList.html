Description:

public class ProjectList : Object, ListModel

Manages project list with deduplication.

Implements ListModel interface using Gee.ArrayList as backing store. Provides ListStore-compatible methods that update the backing store and emit items_changed signals. Projects are Folders with is_project = true.
