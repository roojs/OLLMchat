ModelPullManager

Object Hierarchy:

Description:

public class ModelPullManager : Object

Manages background pull operations for models.

Handles status tracking, persistence, rate-limited UI updates, and retry scheduling. Delegates actual pull execution to ModelPullManagerThread.

Since:

1.3.4
