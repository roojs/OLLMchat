OLLMchat.Ollama

Description:

Content:

Interfaces:

  • MessageInterface - Interface for objects that provide chat content.

Classes:

  • BaseCall - Abstract base class for all Ollama API calls.
  • BaseResponse - Abstract base class for Ollama API responses.
  • CallFunction - Represents a function call with name and arguments. Used within ToolCall to represent the function being called.
  • ChatCall - Chat API call implementation for sending messages and receiving responses.
  • ChatResponse - Response from a chat API call.
  • Client - Main client class for interacting with Ollama API and OpenAI-compatible REST interfaces.
  • EmbedCall - API call to generate embeddings for input text.
  • EmbedResponse - Represents the response from the embeddings API.
  • Function - Concrete class representing a function within a Tool.
  • Message - Simple message class for chat conversations. Implements Json.Serializable for use in chat calls.
  • Model - Represents model information from the Ollama server.
  • ModelsCall - API call to list available models on the Ollama server.
  • OllamaBase - Base class for Ollama API objects that need JSON serialization.
  • Options - Options object for Ollama API calls.
  • Param - Abstract base class for parameter definitions.
  • ParamArray - Represents an array parameter with items definition.
  • ParamObject - Represents an object parameter with nested properties.
  • ParamSimple - Represents a simple parameter type (string, integer, boolean).
  • PsCall - API call to list currently running models on the Ollama server.
  • ShowModelCall - API call to get detailed information about a specific model.
  • Tool - Abstract base class for tools that can be used with Ollama function calling.
  • ToolCall - Represents a tool call from the assistant. Used in assistant messages with tool_calls array.

Error domains:

  • OllamaError
