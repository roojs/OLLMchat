public static void setup_analysis_usage (Config2 config)

Sets up the analysis ModelUsage entry in Config2.

Creates a ModelUsage entry for "ocvector.analysis" in the config's usage map if it doesn't already exist. Uses the default connection and "qwen3-coder:30b" model. This should be called when setting up the codebase search tool.
