Description:

protected PermissionResult check (string perm, Operation operation)

Checks if a permission string allows the requested operation.

Parameters:

perm

Permission string (e.g., "rwx", "r--", "---", "???")

operation

Operation type (READ, WRITE, or EXECUTE)

Returns:

PermissionResult.YES if allowed, PermissionResult.NO if denied, PermissionResult.ASK if unknown
