Config

Object Hierarchy:

Description:

public class Config : Object, Serializable

Configuration management for OLLMchat client settings.

Handles loading and saving client configuration to JSON file. Supports single client configuration (Phase 1) with structure designed to support multiple clients in the future.

1.0
