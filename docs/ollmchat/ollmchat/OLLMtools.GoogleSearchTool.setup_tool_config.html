setup_tool_config

Description:

public static bool setup_tool_config (Config2 config)

Sets up the Google search tool configuration with default values.

Creates a GoogleSearchToolConfig in `Config2.tools["google_search"]` if it doesn't exist. The config will have empty api_key and engine_id, which the user must configure.

If a new config is created, it will be saved automatically. If saving fails, a warning will be logged but the method will still return true (config was created successfully).

Parameters:

config

The Config2 instance to update

Returns:

true if the tool config was created, false if it already existed
